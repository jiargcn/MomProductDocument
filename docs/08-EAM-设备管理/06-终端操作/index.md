# 终端操作

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施、运维（主）；现场维修/巡检人员、实施顾问（顺带）。操作见[终端操作-维护与查询参考](终端操作-维护与查询参考.md)。

## 业务目的与适用范围

现场师傅拿着 PDA 却不知道能做什么、提交失败又不知道该查 Web 哪个分组——本页就是为此写的。它汇总 **EAM PDA** 入口：设备/工装的保养、巡检、点检、开拉、报修、维修与备件执行。业务规则与状态机以 Web 对应分组为准，此处只写终端菜单与操作差异。

不含 WMS PDA、MES 线边客户端，也不含 ANDON 呼叫终端（若现场另有安灯终端）。读完本页，应能对照 PDA 每个入口找到对应 Web 分组排障。

## PDA 能力一览

| 分组 | 典型入口 |
| --- | --- |
| 保养 | 设备/工装：派发、执行、确认 |
| 巡检 | 设备/工装：执行、确认、工单 |
| 点检 | 设备/工装：执行、确认 |
| 开拉 | 设备/工装：执行、确认（若启用） |
| 报修 | 新建、我的报修、审批、详情、记录 |
| 维修 | 设备/工装：派发、执行、确认、工单 |
| 备件 | 申领、出库、归还、入库 |
| 工艺维修 | 部分菜单存在；是否启用以环境为准 |

```mermaid
flowchart LR
    A["领取/派发工单"] --> B["扫码确认对象"]
    B --> C["填报/拍照"]
    C --> D["提交/确认"]
    D --> E["异常可转报修"]
```

!!! example "📝 示例数据占位"
    PDA 领取巡检工单 → 扫码 EQ-001 → 按项填报拍照 → 提交；Web 侧状态同步为待验证/已完成。

!!! example "写实示例：给定配置 → 期望行为"
    **给定：** Web 已生成待接单巡检工单；用户有 EAM PDA 巡检菜单授权；工单对象设备 EQ-001；一项要求拍照。
    **期望：**

    1. PDA 可见该工单入口；无菜单授权则看不到（不是“系统没生成工单”）。
    2. 扫码须与工单对象编码一致，扫错码不能提交。
    3. 提交后 Web 侧状态推进；终端只是入口，前置状态不满足时提交失败应回 Web 查工单状态。
    4. 备件出库同步失败时勿在 PDA 重复盲提；按[备件管理](../03-备件管理/index.md)查同步状态。

### 建议验证点

- 授权用户可见目标 EAM PDA 菜单；撤权后入口消失。
- 巡检或保养工单 PDA 执行后 Web 状态一致。
- 报修可在 PDA 新建并在 Web 可见。
- 扫错设备/工装码被拦截。
- 备件出库同步结果可查（若启用）；失败不重复提交。
- 工艺维修菜单：有入口不等于业务已启用（`GAP-016`）。

## 与 Web / 其它终端边界

现场常把 EAM PDA 和 WMS/MES 的终端混为一谈，下表划清彼此边界：

| 对比 | 说明 |
| --- | --- |
| Web EAM | 计划审批、主数据、复杂查询多在 Web |
| WMS PDA | 仓储作业，不在此列 |
| MES 线边 | 报工领活，不在此列 |

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 菜单看不到 | PDA 菜单授权 | 无入口 |
| 提交失败 | Web 侧状态前置 | 终端只是入口 |
| 备件同步失败 | 同备件 WMS 边界 | 勿重复提交 |

### 选择器范围（骨架）

通例见[通用选择器过滤惯例](../../02-业务模型/12-通用选择器过滤惯例.md)。终端选择器与 Web **共用**后端对象；本表只写 PDA 差异。精确状态集与权限投影见 `FSEM-006` / `GAP-014`。

| 选择字段 | 选择对象 | 可选范围（当前可写） | 范围依赖 | 选不到时通常原因 |
| --- | --- | --- | --- | --- |
| 工单 / 报修单 | EAM 工单·报修 | 待接单/待执行等可执行态（精确集 ❓） | Web 状态前置 | 状态不对、无授权菜单 |
| 设备 / 工装扫码 | DBC 台账 | 须与工单对象编码一致 | 台账、工单对象 | 扫错码、台账停用 |
| 派发 / 执行人 | 用户·岗位 | ❓ PDA 权限投影未逐页实测 | 角色、菜单 | 菜单无入口、非班组 |
| 备件申领对象 | 备件 | 同 Web 备件边界；库存权威在 WMS | 备件/WMS | 同步失败勿重复提交 |
| 工艺维修入口 | （可选菜单） | ❓ 是否启用以环境为准（`GAP-016`） | 菜单配置 | 有菜单无业务 |
