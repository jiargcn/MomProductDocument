# 终端操作

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：现场维修/巡检人员、实施顾问；操作见[终端操作-维护与查询参考](终端操作-维护与查询参考.md)。

## 业务目的与适用范围

本页汇总 **EAM PDA** 入口：设备/工装的保养、巡检、点检、开拉、报修、维修与备件执行。业务规则与状态机以 Web 对应分组为准；此处只写终端菜单与操作差异。

不含 WMS PDA、MES 线边客户端。

## PDA 能力一览

| 分组 | 典型入口 |
| --- | --- |
| 保养 | 设备/工装：派发、执行、确认 |
| 巡检 | 设备/工装：执行、确认、工单 |
| 点检 | 设备/工装：执行、确认 |
| 开拉 | 设备/工装：执行、确认（若启用） |
| 报修 | 新建、我的报修、审批、详情、记录 |
| 维修 | 设备/工装：派发、执行、确认、工单 |
| 备件 | 申领、出库、归还、入库 |
| 工艺维修 | 部分菜单存在；是否启用以环境为准 |

| 对比 | 说明 |
| --- | --- |
| Web EAM | 计划审批、主数据、复杂查询多在 Web |

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 菜单看不到 | PDA 菜单授权 | 无入口 |
| 提交失败 | Web 侧状态前置 | 终端只是入口 |
| 备件同步失败 | 同备件 WMS 边界 | 勿重复提交 |

### 选择器范围（骨架）

通例见[通用选择器过滤惯例](../../02-业务模型/12-通用选择器过滤惯例.md)。终端选择器与 Web **共用**后端对象；本表只写 PDA 差异。精确可选状态与权限范围见通例；未闭合项见辅证 `FSEM-006` / `GAP-014`。

| 选择字段 | 选择对象 | 可选范围（当前可写） | 范围依赖 | 选不到时通常原因 |
| --- | --- | --- | --- | --- |
| 工单 / 报修单 | EAM 工单·报修 | 待接单/待执行等可执行态（精确集 ❓） | Web 状态前置 | 状态不对、无授权菜单 |
| 设备 / 工装扫码 | DBC 台账 | 须与工单对象编码一致 | 台账、工单对象 | 扫错码、台账停用 |
| 派发 / 执行人 | 用户·岗位 | ❓ PDA 权限投影未逐页实测 | 角色、菜单 | 菜单无入口、非班组 |
| 备件申领对象 | 备件 | 同 Web 备件边界；库存权威在 WMS | 备件/WMS | 同步失败勿重复提交 |
| 工艺维修入口 | （可选菜单） | ❓ 是否启用以环境为准（`GAP-016`） | 菜单配置 | 有菜单无业务 |

## 限制与待确认

- `GAP-016`：PDA 离线/断网续传、附件大小、工艺维修菜单有效性与终端规则待环境核验。
- 终端工单可选的精确可选状态与权限投影范围尚未闭合；验收勿按「全状态可见」假定。（辅证：`FSEM-006`）
