# 终端操作

> 状态：页面结构已建立，待在 EAM 模块轮次中结合 EAM PDA 端源码取证。

本页承接归属于 EAM 的终端操作。设备、工装、巡检、保养和维修的业务规则仍在 EAM 业务页面中统一维护；本页仅补充设备扫码、点检采集、拍照附件、离线处理、终端权限和异常反馈等差异。

## BATCH-03 占位任务

| 主题 | 需要确认的内容 |
| --- | --- |
| 终端菜单 | EAM PDA 或移动端中属于设备、工装、巡检、保养、维修的入口。 |
| 执行对象 | 设备、工装、巡检任务、保养任务、维修任务、备件领用和附件。 |
| 扫码规则 | 扫设备、工装、点检点、备件、库位时的校验和异常提示。 |
| 状态前置 | 哪些任务状态允许开始、暂停、提交、关闭、转维修或重新打开。 |
| 数据影响 | 终端操作如何形成巡检记录、保养记录、维修记录、备件消耗和设备履历。 |
| 权限与审计 | 终端用户、岗位、区域、设备范围、拍照附件和现场操作日志。 |
