# 终端操作

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：现场维修/巡检人员、实施顾问；操作见[终端操作-维护与查询参考](终端操作-维护与查询参考.md)。

## 业务目的与适用范围

本页汇总 **EAM PDA** 入口：设备/工装的保养、巡检、点检、开拉、报修、维修与备件执行。业务规则与状态机以 Web 对应分组为准；此处只写终端菜单与操作差异。

不含 WMS PDA、MES 线边客户端。

## 如何使用本组文档

| 你的目的 | 建议阅读 |
| --- | --- |
| 想知道 PDA 能做什么 | 本页。 |

| 状态卡住 | 对应 Web 分组维护参考。 |

## PDA 能力一览

| 分组 | 典型入口 |
| --- | --- |
| 保养 | 设备/工装：派发、执行、确认 |
| 巡检 | 设备/工装：执行、确认、工单 |
| 点检 | 设备/工装：执行、确认 |
| 开拉 | 设备/工装：执行、确认（若启用） |
| 报修 | 新建、我的报修、审批、详情、记录 |
| 维修 | 设备/工装：派发、执行、确认、工单 |
| 备件 | 申领、出库、归还、入库 |
| 工艺维修 | 部分菜单存在；是否启用以环境为准 |

    D --> E["异常可转报修"]

## 与 Web / 其它终端边界

| 对比 | 说明 |
| --- | --- |
| Web EAM | 计划审批、主数据、复杂查询多在 Web |
| WMS PDA | 仓储作业，不在此列 |
| MES 线边 | 报工领活，不在此列 |

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 菜单看不到 | PDA 菜单授权 | 无入口 |
| 提交失败 | Web 侧状态前置 | 终端只是入口 |
| 备件同步失败 | 同备件 WMS 边界 | 勿重复提交 |

## 限制与待确认

- 离线与附件大小限制。
- 工艺维修菜单有效性。
