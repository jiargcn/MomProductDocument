# 终端操作-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[终端操作](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 巡检/保养/点检现场执行 | “维护类工单” |
| 报修/维修 | “报修维修” |
| 备件 | “备件” |

## 维护类工单

| 步骤 | 提醒 |
| --- | --- |
| 派发（若有） | 与 Web 待接单一致。 |
| 执行 | 核对设备/工装编码；按项填报；按要求拍照。 |
| 确认/验证 | 未确认勿当完成。 |

## 报修维修

| 步骤 | 提醒 |
| --- | --- |
| 新建报修 | 故障类型、描述、停机、图片。 |
| 审批 | 权限账号。 |
| 维修派发/执行/确认 | 状态与 Web 维修枚举一致。 |

## 备件

| 步骤 | 提醒 |
| --- | --- |
| 申领→出库→归还/入库 | 关注 WMS 同步结果。 |
