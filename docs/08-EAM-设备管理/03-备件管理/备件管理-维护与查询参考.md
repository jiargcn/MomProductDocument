# 备件管理-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[备件管理](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 查结存 | “台账” |
| 申领出库 | “申领与出入库” |
| 工单领料 | “工单备件” |
| 盘点 | “盘点” |
| 同步失败 | “WMS 同步” |

## 页面与业务对象

| 菜单入口 | 用途 |
| --- | --- |
| 备件台账 | EAM 结存视图。 |
| 领用申请、备件出库/记录、入库申请/记录 | 独立领用链。 |
| 盘点计划/工单/差异/调整 | 盘点。 |
| 工单备件申领/出库/归还/入库 | 工单绑定路径。 |
| PDA 备件* | 现场执行。 |

## 台账

| 应核对 | 提醒 |
| --- | --- |
| 物料号、库区库位、数量、安全库存 | 与 WMS 对账时以 WMS 为准排查差异。 |

## 申领与出入库

| 应维护/确认 | 提醒 |
| --- | --- |
| 申请数量、去向设备/工单 | 可追溯。 |
| 出库执行后同步状态 | 已同步 WMS / 失败需重试或纠错。 |

## 工单备件

| 应核对 | 提醒 |
| --- | --- |
| 关联维修/保养工单 | 勿漏绑。 |
| 出库/归还/入库回写单号 | 失败不要重复盲开 WMS 单。 |

## WMS 同步

| 步骤 | 关注点 |
| --- | --- |
| 出库 | 非计划发料接口；需返回出库单号。 |
| 入库 | 非计划收货接口；需返回入库单号。 |
| 失败 | 查错误信息、账期、库位映射。 |

## 最小验收清单

- [ ] 台账可查到目标备件。
- [ ] 一笔出库后 WMS 侧出现对应非计划发料（启用同步时）。
- [ ] 同步失败可复现并看到失败状态。
- [ ] 工单备件路径与独立领用路径都能走通（按组织启用情况）。
