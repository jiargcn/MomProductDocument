# 备件管理

> 状态：BATCH-03 首轮占位页。当前页面用于承接备件主数据、备件库存、领用、更换和维修成本等内容；具体是否由 WMS 库存对象承载待取证。

## 1. 页面定位

备件管理连接 EAM 维修业务与 WMS 库存管理。文档需要明确备件是普通物料、EAM 专用物料，还是有独立备件模型；在未取证前不直接写死库存粒度或扣减规则。

## 2. 后续取证范围

| 主题 | 需要确认的内容 |
| --- | --- |
| 备件主数据 | 备件编码、名称、规格、适用设备、供应商、替代件、最小库存等字段来源。 |
| 库存关系 | 是否使用 WMS 库存余额、库存事务、库位、批次、包装等对象。 |
| 领用与退库 | 维修领用、保养领用、退库、报废是否生成 WMS 单据或库存事务。 |
| 成本归集 | 备件消耗是否进入维修成本、设备履历或财务结算。 |
| 导入与盘点 | 备件导入模板、库存盘点和差异处理是否与 WMS 共用。 |
