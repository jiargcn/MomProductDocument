# 工装管理

> 状态：BATCH-03 首轮占位页。当前页面用于承接工装台账、领用、保养、寿命、维修和生产引用关系；具体实体与 DBC / EAM 归属待取证。

## 1. 页面定位

工装管理关注工装在生产过程中的可用性、位置、寿命和履历。它需要与 MES 工单/工序、DBC 工艺路线、QMS 检验、EAM 保养维修形成关系。

## 2. 后续取证范围

| 主题 | 需要确认的内容 |
| --- | --- |
| 工装台账 | 工装编码、类型、状态、寿命、适用工序/设备/产线、责任人。 |
| 使用记录 | 工装领用、归还、绑定工单、绑定工序、使用次数或寿命消耗。 |
| 保养维修 | 工装保养计划、维修记录、报废规则是否与设备保养维修共用模型。 |
| 质量影响 | 工装异常是否触发 QMS 检验、质量评审或 ANDON 异常。 |
| 终端操作 | 工装扫码、绑定、归还、点检、拍照等终端入口。 |
