## 履历与改制

| 应维护/确认 | 提醒 |
| --- | --- |
| 验收结果与工装编码 | 通过后可供计划引用。 |
| 出库/入库时间与经办 | 位置可追溯。 |
| 变更/改制类型与部门 | 改制保留关系记录。 |

### 常用动作

验收、登记出入库、变更、改制；PDA 工装巡检/保养/维修见终端页。

## 最小验收清单

- [ ] EAM 菜单打开工装台账落地为 DBC 页。
- [ ] 完成验收或出入库各一笔。
- [ ] 工装可加入点检/巡检计划（若业务需要）。
