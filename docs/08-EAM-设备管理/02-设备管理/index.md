# 设备管理

> 状态：BATCH-03 首轮占位页。当前页面用于说明设备台账、设备状态、维修、停机、履历等 EAM 核心业务的文档结构；具体页面、字段、动作和状态待源码与测试环境取证。

## 1. 页面定位

设备管理回答“设备是什么、现在在哪里、处于什么状态、发生过什么事件、下一步需要谁处理”。它需要与 MES 生产执行、QMS 检验、ANDON 异常和数采数据形成关联。

## 2. 业务模型占位

| 主题 | 需要说明的内容 |
| --- | --- |
| 设备台账 | 设备编码、名称、分类、位置、产线/工位、状态、责任人、供应商/生产商。 |
| 设备状态 | 启用、停用、维修中、保养中、故障、报废等状态是否存在及来源。 |
| 维修记录 | 维修申请、维修任务、维修记录是否复用申请/任务/记录模型。 |
| 设备履历 | 巡检、保养、维修、故障、备件更换、数采异常等历史事件如何聚合。 |
| 跨模块引用 | MES 工单、QMS 检验、ANDON 故障、数采报警如何关联设备。 |
