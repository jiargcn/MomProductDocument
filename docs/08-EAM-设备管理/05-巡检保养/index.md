# 巡检保养

> 状态：BATCH-03 首轮占位页。当前页面用于承接巡检计划、巡检任务、巡检记录、保养计划、保养任务、保养记录等内容；是否复用申请/任务/记录模型待取证。

## 1. 页面定位

巡检保养用于预防设备和工装异常。文档需要说明计划如何生成任务、现场如何执行、异常如何转维修或 ANDON、记录如何进入设备/工装履历。

## 2. 业务模型占位

| 对象 | 需要说明的内容 |
| --- | --- |
| 巡检计划 | 巡检对象、周期、点检项、责任岗位、生成规则。 |
| 巡检任务 | 任务生成、派工、承接、执行、超期、关闭。 |
| 巡检记录 | 检查结果、异常项、照片附件、处理意见和设备履历。 |

| 保养任务/记录 | 执行过程、确认、异常转维修、备件消耗和审计。 |
