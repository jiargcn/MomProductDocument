# 基础数据

> 状态：BATCH-03 首轮占位页。当前页面用于承接 EAM 设备管理所需的基础数据入口；具体实体、字段、唯一性、导入模板和引用关系待 EAM / DBC 源码与 DDL 取证。

## 1. 页面定位

EAM 基础数据用于支撑设备、备件、工装、巡检、保养和维修业务。它与 DBC 主数据存在交叉：设备台账、工装台账、生产商等当前可能位于 DBC 或被 DBC 引用；本文档先按 EAM 使用视角说明，真实归属后续取证。

## 2. 后续取证范围

| 主题 | 需要确认的内容 |
| --- | --- |
| 基础对象 | 设备分类、设备台账、备件分类、工装分类、生产商、供应商、位置、产线/工位等对象来源。 |
| 新增约束 | 编码唯一性、状态默认值、组织/工厂归属、是否允许跨工厂共享。 |
| 编辑限制 | 投入使用后哪些字段禁止修改，哪些字段修改需要同步设备、工装、巡检或维修记录。 |
| 导入规则 | 哪些字段可人工导入，哪些字段由系统生成或从 DBC / System 引用。 |
| 业务引用 | 被巡检计划、保养计划、维修记录、ANDON 异常、MES 生产执行和 QMS 检验引用的方式。 |
