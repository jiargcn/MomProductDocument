# 基础数据-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[基础数据](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 配模块开关/参数 | “EAM 基础配置” |
| 配报修分类 | “故障类型与原因” |
| 配巡检/保养/点检内容 | “方案与检查项” |
| 配派工班组 | “厂区班组角色” |
| 改设备身份/位置 | 转 DBC 设备台账（勿在本组硬改） |

## 页面与业务对象

| 菜单入口 | 用途 |
| --- | --- |
| EAM基础配置 | 键值参数。 |
| 故障类型 / 故障原因 | 报修分类。 |
| 保养/巡检/点检方案、项、选择集 | 计划方案底座。 |
| 开拉方案/项（若启用） | 开拉底座。 |
| 厂区班组角色维护 | 派工角色。 |
| 主要部件 / 部件关联 | 设备结构。 |
| 生产商管理 / 供应商管理（EAM） | EAM 侧厂商。 |
| 备件基础、库位等 | 备件主数据与定位。 |
| 文档类型、故障率参数、生产时长等 | 辅助。 |
| EAM工作日历 | 排程日历。 |

## EAM 基础配置

| 应维护/确认 | 提醒 |
| --- | --- |
| 编码、标题、配置值 | 改前确认被哪些计划/自动策略引用。 |

## 故障类型与原因

| 应维护/确认 | 提醒 |
| --- | --- |
| 编码、名称、类型、标准工时 | 报修必选；影响维修统计。 |
| 可选关联设备 | 限定适用设备时再填。 |

### 常用动作

新增/修改/启停；避免删除已被历史报修引用的类型。

## 方案与检查项

| 应维护/确认 | 提醒 |
| --- | --- |
| 方案编码/名称、适用类型（设备/工装等） | 计划上的方案编码必须存在。 |
| 检查/保养项内容、顺序、选择集 | 项要可判定；选择集供定性结果。 |
| 点检与巡检方案分开维护 | 对应不同计划菜单。 |

### 常用动作

维护方案 → 维护项 →（可选）选择集 → 在[巡检保养](../05-巡检保养/巡检保养-维护与查询参考.md)建计划引用。

## 厂区班组角色

| 应维护/确认 | 提醒 |
| --- | --- |
| 角色与班组/车间关系 | 派工、接单依赖。 |
| 与系统用户/岗位映射 | 以现场权限配置为准。 |

## 最小验收清单

- [ ] 至少一类故障类型可用于报修。
- [ ] 至少一套保养/巡检/点检方案可被计划选中。
- [ ] 班组角色可支撑一条工单派发。
- [ ] 设备身份变更走 DBC 台账，而非本组字段。
