# 工装台账管理

## 概述

工装台账记录工装模具的全生命周期信息，用于EAM工装管理和生产工序关联。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| toolCode | 工装编码 | VARCHAR(50) | 必填 | 工装唯一标识 | 工装的唯一标识 |
| toolName | 工装名称 | VARCHAR(200) | 必填 | 工装显示 | 工装的显示名称 |
| toolType | 工装类型 | VARCHAR(50) | 必填 | 工装分类 | 工装类型 |
| spec | 规格 | VARCHAR(100) | 非必填 | 工装识别 | 规格型号 |
| manufacturer | 制造商 | VARCHAR(100) | 非必填 | 维修联络 | 生产厂家 |
| purchaseDate | 购置日期 | DATE | 非必填 | 寿命统计 | 購置时间 |
| lifecycleCount | 设计寿命 | INT | 非必填 | 寿命预警 | 设计使用次数 |
| currentCount | 当前使用次数 | INT | 系统自动 | 寿命跟踪 | 实际使用次数 |
| status | 状态 | ENUM | 字典项 | 工装可用性 | 工装状态 |
| location | 当前库位 | VARCHAR(50) | 非必填 | 库位查询 | 存放库位 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 工装分配 | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 工装启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 工装停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
工装类型（1）←→（N）工装
工装 → 库位（当前存放位置）
工装 → 工序配置（工装-工序关系）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | toolCode 唯一 |
| 自增长 | currentCount 由系统自动累计 |
