# 模具类型管理

## 概述

模具类型管理定义模具的分类体系，用于模具管理和保养计划制定。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| moldTypeCode | 类型编码 | VARCHAR(50) | 必填 | [模具类型](03-模具类型管理.md)唯一标识 | 模具类型的唯一标识 |
| moldTypeName | 类型名称 | VARCHAR(200) | 必填 | [模具类型](03-模具类型管理.md)显示 | 模具类型的显示名称 |
| status | 状态 | ENUM | 字典项：启用/禁用 | [模具类型](03-模具类型管理.md)可用性 | 状态 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
模具类型（1）←→（N）模具信息管理
```
