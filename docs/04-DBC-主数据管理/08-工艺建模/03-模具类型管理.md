# 模具类型管理

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。

## 这项分类原本应解决什么问题

模具类型应为模具、工装、工艺条件和现场使用建立统一的分类口径，使维护人员能够按类型识别适用范围、责任分工和查询维度。它不应替代工装台账、模具实物台账或 EAM 的维护执行记录。

## 当前可确认状态

当前可查的 DBC 与 MES 资料中，尚未找到与本页面一一对应、可确认的独立维护对象和操作入口。因此，本页保留为业务字典与建设占位：可以说明期望的业务作用，但不对新增字段、编辑限制、导入规则、选择器或实际引用范围作未经证实的断言。

工装台账中存在类型信息，工艺建模与 EAM 中也可能使用模具/工装相关口径；是否共用同一分类源、由哪个模块负责维护，需要在后续资料核对后确定。
