## 概述

工序是生产制造过程中的最小作业单元，定义产品制造过程中的具体加工作为及其作业参数。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| processCode | 工序编码 | VARCHAR(50) | 必填、唯一 | MES工序流转（工序的唯一标识）、报工归属（报工记录按工序归集） | 工序的唯一身份标识，用于标识具体工序，系统主键 |
| processName | 工序名称 | VARCHAR(200) | 必填 | MES显示（工序名称展示）、报工显示 | 工序的显示名称，如"焊接"、"组装"、"检测" |
| processType | 工序类型 | ENUM | 字典项：加工/装配/检测/包装/流转 | 工序作业分类（按类型汇总工时） | 工序的主要作业类型 |
| workContent | 作业内容 | VARCHAR(500) | 非必填 | 作业指导书（显示工序的具体作业要求和标准） | 工序的具体作业内容和操作要求，关联SOP文件内容，供工人参考执行 |
| standardTime | 标准工时 | DECIMAL(18,4) | 非必填 | （待补充） | 完成一个作业循环的标准时间，单位小时/件 |
| qualifiedRate | 合格率 | DECIMAL(5,2) | 非必填，默认100% | （待补充） | 工序的标准合格率 |
| needDevice | 需设备 | BOOLEAN | 默认否 | （待补充） | 标识此工序是否需要设备才能作业 |
| needMaterial | 需物料 | BOOLEAN | 默认是 | （待补充） | 标识此工序是否需要物料投料 |
| qualityControlPoint | 质量控制点 | BOOLEAN | 默认否 | （待补充） | 标识此工序是否为质量控制点 |
| nextProcessCode | 下一工序 | VARCHAR(50) | 非必填，来源表：工序 | MES工序流转方向（报工完成后自动流转到下一工序） | 工序完成后的下一个工序编码，报工完成后系统自动将物料流转到下一工序，串联工序顺序 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 工序的补充说明，如特殊作业要求、设备型号要求等 |

## 关联关系

```
工序 → 工艺路线（顺序关系）
工序 → MES报工记录
工序 → 设备（需设备时绑定）
工序 ← BOM子项（工位级用料）
工序 → QMS检验记录（质量控制点触发）
```
