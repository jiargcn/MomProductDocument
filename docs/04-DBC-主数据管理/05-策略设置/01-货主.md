# 货主

## 概述

货主定义库存物料的所有权归属，用于多主体经营场景下的库存隔离和财务核算。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| ownerCode | 货主编码 | VARCHAR(50) | 必填、唯一 | WMS库存隔离 | 系统主键 |
| ownerName | 货主名称 | VARCHAR(200) | 必填 | WMS显示 | |
| contactPerson | 联系人 | VARCHAR(100) | 非必填 | 业务联络 | |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 业务联络 | |
| status | 状态 | ENUM | 字典项：启用/禁用 | WMS业务 | 禁用后不可新建业务 |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 库存隔离 | 库存按货主独立计算，互不混用 |
| WMS | 入库/出库 | 业务单据必须指定货主 |
| 财务 | 成本核算 | 货主维度独立核算库存成本 |

## 关联关系

```
货主（1）←→（N）库存余额
货主 → 入库单/出库单
```
