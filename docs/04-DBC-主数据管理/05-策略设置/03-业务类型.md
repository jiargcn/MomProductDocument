# 业务类型

## 概述

业务类型定义库存事务的分类，用于 WMS 库存事务的分类统计和业务流程的区分。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| bizTypeCode | 业务类型编码 | VARCHAR(50) | 必填、唯一 | WMS事务分类 | 系统主键 |
| bizTypeName | 业务类型名称 | VARCHAR(200) | 必填 | WMS显示 | 如：采购入库/成品入库/调拨出库 |
| bizCategory | 业务类别 | ENUM | 字典项：入库/出库/移库/库存调整 | WMS分类统计 | |
| isGenerateTrans | 产生事务 | BOOLEAN | 默认是 | WMS事务记录 | 某些内部调拨不产生事务 |
| isRequireApprove | 需要审批 | BOOLEAN | 默认否 | WMS审批流 | 开启后需审批才执行 |
| allowNegativeStock | 允许负库存 | BOOLEAN | 默认否 | WMS库存校验 | 影响此业务类型的库存控制 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 库存事务 | 事务记录归属业务类型 |
| WMS | 库存报表 | 按业务类型分类统计 |
| WMS | 审批流 | 开启审批后触发流程 |

## 关联关系

```
业务类型 → 库存事务记录
业务类型 → 单据设置（业务单据的配置）
```
