# 业务类型

## 概述

业务类型定义库存事务的分类，用于 WMS 库存事务的分类统计和业务流程的区分。每个业务类型可关联特定的入库/出库库存事务类型，用于事务追溯和自动化处理。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| bizTypeCode | 代码 | VARCHAR(50) | 必填、唯一 | WMS事务分类（事务记录按业务类型归类）、单据关联（单据保存时关联业务类型） | 业务类型的唯一标识，用于标识具体的库存事务类型，如 UnplannedIssueSpare、ProductReceiptHold 等 |
| bizTypeName | 名称 | VARCHAR(200) | 必填 | WMS显示（业务类型名称展示）、报表（按类型统计库存事务） | 业务类型的显示名称，如"备件归还-计划外出库"、"隔离转线边申请"等 |
| bizCategory | 分组 | VARCHAR(50) | 非必填 | 业务分类（页面Tab筛选）、统计（按分组汇总） | 业务分组，如"计划外出入库"、"制品收货"、"下架"、"包装"、"库移"等 |
| inTransTypeCode | 入库库存事务 | VARCHAR(50) | 非必填 | 入库事务（入库单执行时关联的事务类型） | 入库时产生的事务类型编码，如 RCT-UNP |
| outTransTypeCode | 出库库存事务 | VARCHAR(50) | 非必填 | 出库事务（出库单执行时关联的事务类型） | 出库时产生的事务类型编码，待确认 |
| isGenerateTrans | 产生事务 | BOOLEAN | 默认是 | 事务记录（是否产生库存事务） | 标识此业务类型是否产生库存事务记录 |
| isRequireApprove | 需要审批 | BOOLEAN | 默认否 | WMS审批流（开启后单据需审批才可执行） | 标识此业务类型的单据是否需要审批才能执行 |
| allowNegativeStock | 允许负库存 | BOOLEAN | 默认否 | 出库校验（出库时是否允许负库存） | 出库时是否允许负库存 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 下拉可选（不可用时不出现在业务类型选择列表） | 业务类型的可用状态 |
| effectiveDate | 生效时间 | DATETIME | 非必填 | 业务生效（定时启用业务类型） | 业务类型的生效时间 |
| invalidDate | 失效时间 | DATETIME | 非必填 | 业务失效（定时停用业务类型） | 业务类型的失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 业务类型的补充说明 |

## 页面结构

业务类型页面包含：
- **列表页**：展示代码、名称、分组、入库库存事务、出库库存事务、操作列
- **每行设置按钮**：点击展开快捷操作菜单（列表设置/搜索设置/全部）
- **详情抽屉**：点击代码链接打开详情抽屉，包含明细/附件/备注/变更记录等标签页

## 分组（业务类别）说明

| 分组 | 说明 | 典型业务类型代码 |
|------|------|----------------|
| 计划外出入库 | 非计划性出入库业务 | UnplannedIssueSpare, UnplannedReceipt, UnplannedDeliver |
| 制品收货 | 成品/半成品入库 | ProductReceiptHold |
| 下架 | 物料下架出库 | MaterialPrepare, OutissueRequest, Repleinment, ProductionReceipt |
| 包装 | 包装相关业务 | ContainerAdjust, PalletAdjust, OverPackage, MergePackage, SplitPackage |
| 库移 | 库存移库业务 | TransferIssue, Move |
| 库存修改 | 库存调整类业务 | Change |

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | bizTypeCode 唯一 |
| 联动影响 | isEnabled=否→业务类型下拉列表不显示，但历史单据仍可引用 |
