# 班次管理-维护与查询参考

> 对应业务说明：[班次管理](01-班次管理.md)。

## 新增与编辑

| 维护项 | 要求与限制 |
| --- | --- |
| 编码、开始/结束时间 | 均为必填；时间口径应与现场作息一致。 |
| 跨天与休息安排 | 跨天标记必填；休息开始/结束时间在导入中为必填，需先验证跨天班次的实际处理。 |
| 状态与有效期 | 可用状态必填；停用前检查班组和在途业务。 |

## 导入、查询与风险

模板包含编码、名称、开始/结束时间、有效期、跨天、可用状态、休息事项及休息时间。不应导入人员、考勤、工单或实际记录。列表优先显示编码、名称、起止时间、跨天、状态和有效期；详情建议分为“班次时间、休息安排、状态、审计”。

【待补充：跨天、休息重叠、有效期冲突和已引用班次变更的测试结果。】
