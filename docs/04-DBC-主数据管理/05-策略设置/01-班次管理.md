# 班次管理

## 概述

班次管理定义生产班次的时间安排，用于MES排班和考勤统计。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| shiftCode | 班次代码 | VARCHAR(50) | 必填 | MES排班（排班时选择班次） | 班次的唯一标识 |
| shiftName | 班次名称 | VARCHAR(200) | 非必填 | MES显示（显示班次名称） | 班次的显示名称 |
| startTime | 开始时间 | TIME | 必填 | 考勤统计（上班刷卡时间比对） | 班次上班时间 |
| endTime | 结束时间 | TIME | 必填 | 考勤统计（下班刷卡时间比对） | 班次下班时间 |
| isCrossDay | 是否跨天 | BOOLEAN | 默认否 | 跨天判断（22:00-06:00型跨天班次） | 是否跨越午夜0点 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 排班可用性 | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 班次启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 班次停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
班次 → 班组（班次分配）
班次 → 考勤记录
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | shiftCode 唯一 |
| 联动影响 | isCrossDay=true 时，跨天00:00后归属次日班次 |
