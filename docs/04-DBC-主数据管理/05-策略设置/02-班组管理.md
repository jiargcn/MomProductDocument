# 班组管理

## 概述

班组管理定义生产班组的人员组织，用于MES派工和报工归属。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| teamCode | 班组编码 | VARCHAR(50) | 必填 | MES派工（派工时选择班组） | 班组的唯一标识 |
| teamName | 班组名称 | VARCHAR(200) | 必填 | MES显示（显示班组名称） | 班组的显示名称 |
| teamType | 班组类型 | VARCHAR(50) | 必填 | 班组分类 | 班组类型 |
| leader | 负责人 | VARCHAR(50) | 必填 | 职责归属（班组异常问题找班组长） | 班组长用户 |
| workshopCode | 车间代码 | VARCHAR(50) | 必填 | 组织归属（班组归属哪个车间） | 所属车间 |
| productionLineCode | 产线代码 | VARCHAR(50) | 必填 | 产线归属 | 所属产线 |
| members | 班组成员 | VARCHAR(500) | 非必填 | 成员管理 | 班组人员列表 |
| effectiveDate | 生效时间 | DATE | 非必填 | 班组启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 班组停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
车间（1）←→（N）班组
产线（1）←→（N）班组
班组 → MES报工记录
班组 → MES派工记录
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | teamCode 唯一 |
| 来源表 | workshopCode → 车间表，productionLineCode → 产线表 |
