# 班组管理-维护与查询参考

> 对应业务说明：[班组管理](02-班组管理.md)。

## 新增与编辑

班组新增需要编码、名称、车间、生产线、负责人、班组类别和可用状态；上层车间、生产线必须先维护。成员信息当前不在导入模板中，人员维护和排班应通过对应组织/人事能力完成。

## 导入、查询与风险

模板包含编码、名称、车间、生产线、负责人编码/名称、班组类别、有效期、备注和可用状态。不应导入成员清单、考勤、工单或实际产出。列表优先显示班组、现场归属、负责人、类别、状态和有效期；详情建议分为“组织归属、负责人、状态、审计”。

【待补充：负责人有效性、班组改线、停用和对生产记录的影响测试。】
