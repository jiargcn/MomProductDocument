# 单据开关

## 概述

单据开关用于控制各类业务单据的开关状态，设置某项业务功能是否启用。每个开关对应一个特定业务场景的校验或处理逻辑。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| switchCode | 代码 | VARCHAR(50) | 必填、唯一 | 系统判断（业务逻辑校验时读取开关状态） | 单据开关的唯一标识，如 DeliverJobValidateCustomerLabel 等 |
| description | 描述 | VARCHAR(500) | 非必填 | 开关说明 | 单据开关的功能描述 |
| switchValue | 有效设置值 | VARCHAR(50) | 非必填 | 功能启用（开关值决定功能是否启用） | 开关的有效值，如"是" |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 下拉可选（禁用时不出现在开关选择列表） | 单据开关的可用状态 |
| createTime | 创建时间 | DATETIME | 系统生成 | 记录审计 | 自动记录创建时间 |
| createBy | 创建者 | VARCHAR(100) | 系统生成 | 记录审计 | 自动记录创建者 |
| updateTime | 最后更新时间 | DATETIME | 系统生成 | 记录审计 | 自动记录最后更新时间 |
| updateBy | 最后更新者 | VARCHAR(100) | 系统生成 | 记录审计 | 自动记录最后更新者 |

## 页面结构

单据开关页面包含：
- **列表页**：展示代码、描述、有效设置值、创建时间、创建者、最后更新时间、最后更新者、操作列
- **编辑按钮**：每行有编辑按钮，点击打开编辑抽屉
- **详情抽屉**：包含明细/附件/备注/变更记录等标签页

## 开关类型示例

| 代码 | 描述 | 有效设置值 |
|------|------|----------|
| DeliverJobValidateCustomerLabel | 发货任务校验客户标签 | 是 |
| ReserveReplenishMaterialsAccordingHighReservoirArea | 根据高层库区预留补料 | 是/否 |
| productreceiptScrapConfirmationSwitch | 入库报废确认开关 | 是/否 |

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | switchCode 唯一 |
| 系统字段 | createTime/createBy/updateTime/updateBy 由系统自动维护 |
