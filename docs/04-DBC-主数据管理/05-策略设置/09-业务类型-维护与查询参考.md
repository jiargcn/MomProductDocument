# 业务类型-维护与查询参考

> 对应业务说明：[业务类型](03-业务类型.md)。

## 配置前检查清单

1. 明确业务场景、入出库方向和库存事务口径。
2. 明确可用物料、库存状态、库区和在途地点范围。
3. 明确自动提交、自动通过、自动执行、直接生成记录等是否允许。
4. 明确可修改库位、数量、批次、包装及部分/超量完成的业务边界。
5. 在测试环境完成创建、执行、撤销、库存、打印和异常回归。

## 查询与变更风险

列表应优先显示代码、名称、分组、可用状态及关键自动处理标记；详情建议分为“适用范围、库存与事务、执行限制、编号与打印、状态与审计”。当前未确认每一项设置在所有业务模块都已生效，生产环境变更应采用审批、备份、灰度验证和可回退方案。
