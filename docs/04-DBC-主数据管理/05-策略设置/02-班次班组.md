# 班次班组

## 概述

班次班组定义生产现场的人员组织结构，用于 MES 报工归属和 EAM 设备排班。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| shiftCode | 班次编码 | VARCHAR(50) | 必填、唯一 | MES排班 | 系统主键 |
| shiftName | 班次名称 | VARCHAR(200) | 必填 | MES显示 | 如：早班/中班/晚班 |
| startTime | 开始时间 | TIME | 必填 | MES考勤统计 | |
| endTime | 结束时间 | TIME | 必填 | MES考勤统计 | |
| teamCode | 班组编码 | VARCHAR(50) | 必填、唯一 | MES派工 | 系统主键 |
| teamName | 班组名称 | VARCHAR(200) | 必填 | MES显示 | 如：装配A组/检测B组 |
| workshopCode | 所属车间 | VARCHAR(50) | 必填，来源表：车间 | 组织归属 | |
| teamLeader | 班组长 | VARCHAR(50) | 非必填，来源表：用户 | 职责归属 | |
| teamSize | 人数 | INT | 非必填 | MES派工 | |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES排班 | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 报工 | 报工记录归属班组 |
| MES | 考勤 | 上下工记录归属班组 |
| EAM | 设备排班 | 设备运行归属班组责任 |
| 考勤 | 工时统计 | 班组维度工时汇总 |

## 关联关系

```
车间（1）←→（N）班组
班组 → MES报工记录
班组 → MES上下工记录
班组 → EAM设备点检记录
```
