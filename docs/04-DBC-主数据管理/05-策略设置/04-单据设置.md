# 单据设置

## 概述

单据设置定义各类业务单据的编号规则和基本属性。每个单据类型对应一种业务单据，如采购入库、成品入库、隔离转线边等，系统根据单据类型匹配编号规则。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| docTypeCode | 代码 | VARCHAR(50) | 必填、唯一 | 单据引用（新建单据时关联单据类型） | 单据类型的唯一标识，如 HoldToWipRequest、ProductionScrapRequest 等 |
| docTypeName | 名称 | VARCHAR(200) | 必填 | 单据显示（单据标题显示） | 单据类型的显示名称，如"隔离转线边申请"、"工序报废申请"等 |
| description | 描述 | VARCHAR(500) | 非必填 | 单据说明 | 单据类型的补充说明 |
| docType | 单据类型 | VARCHAR(50) | 非必填 | 单据分类（页面Tab分类） | 单据类型分类，如"申请"、"记录"等 |
| bizTypeCode | 业务类型 | VARCHAR(50) | 非必填 | 业务关联（关联业务类型用于事务归类） | 关联的业务类型编码 |
| numberPrefix | 号码前缀 | VARCHAR(50) | 非必填 | 编号生成（单据编号前缀） | 单据编号的前缀，如 REQ56、WI- 等 |
| numberTimeFormat | 号码时间格式 | VARCHAR(50) | 非必填 | 编号生成（时间部分格式） | 编号中的时间格式，如 yyyyMMdd、yyyyMM 等 |
| numberSeqLength | 号码流水长度 | INT | 非必填 | 编号生成（流水号位数） | 流水号位数，如 4 表示 0001 |
| numberSeparator | 号码分隔符 | VARCHAR(10) | 非必填 | 编号生成（分隔符） | 编号各部分之间的分隔符 |
| seqResetCycle | 流水重置周期 | VARCHAR(50) | 非必填 | 编号生成（流水号重置周期） | 流水号重置周期，如"天"、"月"、"年" |
| effectiveDate | 生效时间 | DATETIME | 非必填 | 单据启用（定时启用单据类型） | 单据类型的生效时间 |
| invalidDate | 失效时间 | DATETIME | 非必填 | 单据停用（定时停用单据类型） | 单据类型的失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 单据类型的补充说明 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 下拉可选（禁用时不出现在单据类型选择列表） | 单据类型的可用状态 |
| createTime | 创建时间 | DATETIME | 系统生成 | 记录审计 | 自动记录创建时间 |
| createBy | 创建者 | VARCHAR(100) | 系统生成 | 记录审计 | 自动记录创建者 |
| updateTime | 最后更新时间 | DATETIME | 系统生成 | 记录审计 | 自动记录最后更新时间 |
| updateBy | 最后更新者 | VARCHAR(100) | 系统生成 | 记录审计 | 自动记录最后更新者 |

## 页面结构

单据设置页面包含：
- **列表页**：展示代码、名称、描述、单据类型、业务类型、号码前缀、号码时间格式、号码流水长度、号码分隔符、流水重置周期、生效时间、失效时间、备注、是否可用、操作列
- **新增按钮**：点击打开新增抽屉（页面无独立新增按钮，需在列表页操作）
- **编辑按钮**：每行有编辑按钮，点击打开编辑抽屉
- **详情抽屉**：包含明细/附件/备注/变更记录等标签页

## 编号规则说明

| 字段组合 | 示例 | 生成效果 |
|----------|------|----------|
| 前缀+时间格式+流水长度 | REQ56 + yyyyMMdd + 4 | REQ56202505200001 |
| 前缀+时间格式 | WI- + yyyyMMdd | WI-20250520 |
| 纯前缀+流水号 | PO- + SN(10) | PO-0000000001 |

## 单据类型示例

| 代码 | 名称 | 单据类型 | 业务类型 | 号码前缀 |
|------|------|----------|----------|----------|
| HoldToWipRequest | 隔离转线边申请 | 申请 | HoldToWip | REQ56 |
| HoldToWipRecord | 隔离转线边记录 | 记录 | - | - |
| ProductionScrapRequest | 工序报废申请 | 申请 | - | - |
| ProductionScrapRecord | 工序报废记录 | 记录 | - | - |

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | docTypeCode 唯一 |
| 联动影响 | isEnabled=否→单据类型下拉列表不显示，但历史单据仍可引用 |
| 系统字段 | createTime/createBy/updateTime/updateBy 由系统自动维护 |
