# 单据设置

## 概述

单据设置定义业务单据（采购订单、销售订单、入库单等）的编号规则和流程配置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| docTypeCode | 单据类型编码 | VARCHAR(50) | 必填、唯一 | 所有业务单据 | 系统主键 |
| docTypeName | 单据类型名称 | VARCHAR(200) | 必填 | 单据显示 | 如：采购订单/入库单 |
| prefixRule | 编号前缀规则 | VARCHAR(50) | 必填 | 单据编号生成 | 如：PO-、WIN- |
| sequenceRule | 流水号规则 | VARCHAR(50) | 必填 | 单据编号生成 | 如：YYYYMMDD+4位序号 |
| defaultStatus | 默认状态 | ENUM | 字典项：草稿/已提交/已审批 | 单据新建状态 | |
| needApprove | 需要审批 | BOOLEAN | 默认否 | 审批流触发 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| 采购 | 采购订单编号 | 编号规则应用于所有采购单 |
| WMS | 入库单编号 | 按规则自动生成单据号 |
| WMS | 出库单编号 | 按规则自动生成单据号 |
| 财务 | 收付款单据 | 编号关联财务凭证 |

## 关联关系

```
单据设置 → 业务单据（采购订单/入库单/出库单/调拨单）
单据设置 → 审批流配置（需要审批时）
```
