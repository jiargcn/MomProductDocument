## 概述

标准成本价格单记录物料的标准成本，用于WMS出库成本计算和MES成本核算。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| materialCode | 物料代码 | VARCHAR(50) | 必填 | 成本归属 | 关联物料 |
| standardCost | 标准成本 | DECIMAL(18,4) | 必填 | WMS出库成本（出库计价基准）、MES成本核算（工单成本计算基准） | 标准单位成本 |
| currency | 币种 | VARCHAR(10) | 非必填 | 成本计价币种 | 币种 |
| effectiveDate | 生效时间 | DATE | 必填 | 成本启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 成本停用时间 | 失效时间 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 成本可用性 | 是否可用 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
物料（1）←→（N）标准成本价格单
标准成本 → WMS出库单（成本计算）
标准成本 → MES工单（成本核算）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | materialCode + effectiveDate 唯一 |
| 联动影响 | 新成本生效后，老成本自动失效；历史单据成本不变 |
