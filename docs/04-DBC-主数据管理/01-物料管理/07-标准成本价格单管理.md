# 标准成本价格单管理

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：成本/财务协同人员、主数据维护人员、采购与生产管理人员。

## 业务目的与适用范围

标准成本价格单用于维护物料在一定适用范围内的标准成本或参考价格，为成本核算、计划分析、采购/生产协同提供统一口径。它表达的是管理或核算基准，不等同于每一笔采购、收货、销售或库存事务的实际成交金额。

## 何时需要维护

物料新建、成本周期切换、原材料价格或工艺成本发生重大变化，或需要对标准与实际进行分析时，应新增、调整或停用价格单。

## 价格单如何被业务使用

```mermaid
flowchart LR
    A["物料与成本口径"] --> B["维护标准成本价格单"]
    B --> C["成本/计划分析"]
    C --> D["采购、生产或报表引用"]
    D --> E["标准与实际差异分析"]
```

维护前应先明确价格适用对象、币种/单位、有效期间和维护责任。若同一物料存在多个成本版本或不同组织口径，应避免在没有适用范围说明的情况下覆盖原价格。

## 关键维护与变更

| 维护点 | 业务判断 | 使用建议 |
| --- | --- | --- |
| 适用物料 | 价格对应哪项物料或物料范围。 | 先核对物料身份和单位。 |
| 标准价格与币种 | 是否为当前管理口径认可的标准。 | 不把单次采购价直接写成标准成本。 |
| 生效期间/版本 | 新旧价格何时切换。 | 成本结转或在制业务期间避免无控制覆盖。 |
| 启停与更正 | 历史价格是否仍需用于追溯。 | 优先新增新版本，保留历史依据。 |

## 查询、详情与联查

| 查询目标 | 建议联查 |
| --- | --- |
| 某物料当前标准成本 | 物料、有效期、价格单状态。 |
| 某期间为何采用该价格 | 价格版本、生效范围、维护记录。 |
| 标准与实际为何不同 | 采购/生产实际记录、成本分析或报表。 |

详情页建议按“物料与单位、价格与币种、适用期间/版本、维护与审批、引用与分析、系统信息”分组。

## 常见问题与处理

| 情况 | 建议处理 |
| --- | --- |
| 同一物料存在多条有效价格 | 先核对期间、版本和适用组织，不要随意覆盖。 |
| 价格与采购价不一致 | 区分标准成本与实际成交价，分别查对应业务。 |
| 需要更正历史价格 | 评估已结转或已分析期间，按成本管理规则处理。 |

## 当前限制与待确认事项

- 价格单实际字段、唯一性、审批/启停和导入规则待继续核验；
- 与财务结算、库存计价、采购价和生产成本的自动引用边界待确认；
- 价格的组织、仓库、币种和单位适用范围需补测试环境示例。
