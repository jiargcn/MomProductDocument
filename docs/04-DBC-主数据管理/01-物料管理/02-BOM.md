# BOM（物料清单）

## 概述

BOM（Bill of Materials）描述产品结构中父项物料与子项物料的层级用量关系，是 MES 工单展开、MES 齐套检查、WMS 发料的核心依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| bomCode | BOM编码 | VARCHAR(50) | 必填、唯一 | MES工单展开 | 系统主键 |
| bomName | BOM名称 | VARCHAR(200) | 必填 | MES工单展开 | 通常与产品名称一致 |
| parentMaterial | 父项物料 | VARCHAR(50) | 必填，来源表：物料 | MES工单展开、工序跳阶 | 产品/半成品 |
| version | 版本 | VARCHAR(20) | 必填 | MES工单展开 | 变更管理依据 |
| effectiveDate | 生效日期 | DATE | 必填 | MES工单展开 | 控制BOM版本切换 |
| invalidDate | 失效日期 | DATE | 非必填 | MES工单展开 | 空=永久有效 |
| status | 状态 | ENUM | 字典项：草稿/生效/禁用 | MES工单展开 | |
| bomLevel | BOM层级 | INT | 必填，默认0 | MES齐套展开 | 0=顶层BOM |
| standardWorkTime | 标准工时 | DECIMAL(18,4) | 非必填 | MES报工、成本核算 | 单位：小时 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 子项物料字段（BOMLine）

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| sequence | 序号 | INT | 必填 | 发料顺序 | 影响发料优先级 |
| childMaterial | 子项物料 | VARCHAR(50) | 必填，来源表：物料 | WMS发料、MES工序用料 | |
| quantity | 用量 | DECIMAL(18,4) | 必填 | WMS发料数量计算 | 父项1对应的子项数量 |
| scrapRate | 报废率 | DECIMAL(5,2) | 非必填，默认0 | WMS发料数量 | 实际发料=用量×(1+报废率) |
| stationNo | 工位 | VARCHAR(50) | 非必填 | MES工序发料 | 指定发料到具体工位 |
| isKeyItem | 关键件 | BOOLEAN | 非必填 | MES齐套检查 | 关键件未齐套不允许开工 |
| isOptional | 选配件 | BOOLEAN | 非必填 | MES工单选配 | 选配件可由工单选择是否参与 |
| backFlush | 倒冲料 | BOOLEAN | 非必填 | MES报工倒冲 | 报工后自动触发WMS出库 |
| substituteMaterial | 替代物料 | VARCHAR(50) | 非必填，来源表：物料 | MES齐套替代 | 物料不足时自动替代 |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 工单展开/齐套检查 | BOM结构和用量决定工单需料量 |
| MES | 工序流转 | 多层级BOM决定工序之间的物料流转 |
| WMS | 发料计算 | 用量×工单数量+报废率=实际发料量 |
| WMS | 核料 | 发料时校验库存是否满足BOM展开量 |
| 成本 | 标准成本计算 | BOM+工序工时=标准成本 |

## 关联关系

```
父项物料（成品/半成品）← BOM → 子项物料（原材料/辅料）
BOM → BOMLine（多层嵌套，可支持6层以上）
物料 → BOM（产品配置用料）
替代物料 → 物料（BOM Line级替代关系）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | 状态（草稿/生效/禁用），层级（顶层BOM/子BOM） |
| 联动影响 | 替代物料配置→齐套不足时自动替换；报废率→影响发料数量；倒冲→报工自动触发WMS出库 |
| 版本管理 | BOM变更应记录变更原因和生效日期，工单引用时按生效日期使用对应版本 |
