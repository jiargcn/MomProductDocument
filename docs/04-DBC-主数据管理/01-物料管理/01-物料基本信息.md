# 物料基本信息

## 概述

物料基本信息是 MES 制造运营系统中最核心的主数据之一，作为所有业务单据（采购、生产、销售、库存）的计价和用料依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| materialCode | 物料编码 | VARCHAR(50) | 必填、唯一 | 所有业务单据 | 系统主键 |
| materialName | 物料名称 | VARCHAR(200) | 必填 | 所有业务单据 | 显示名称 |
| materialType | 物料类型 | ENUM | 字典项：成品/原料/辅料/包装物/工装 | 生产计划、库存分类 | 影响BOM展开范围 |
| unit | 单位 | VARCHAR(10) | 必填 | 入库/出库/盘点计量 | 建议与计量单位统一 |
| spec | 规格型号 | VARCHAR(200) | 非必填 | 采购询价、客户对账 | |
| unitCost | 标准成本 | DECIMAL(18,4) | 非必填 | WMS计价、MES成本统计 | 成本核算基准 |
| materialGroup | 物料组 | VARCHAR(50) | 字典项 | MRP运算、库存分类 | 影响采购策略 |
| sourceType | 来货方式 | ENUM | 字典项：采购/自制/委外 | 采购收货、检验入口 | |
| inspectionType | 检验类型 | ENUM | 字典项：免检/全检/抽检 | QMS来料检验 | 影响检验是否触发 |
| safetyStock | 安全库存 | DECIMAL(18,4) | 非必填 | 库存预警、补料建议 | |
| minStock | 最小库存 | DECIMAL(18,4) | 非必填 | 补料触发阈值 | |
| maxStock | 最大库存 | DECIMAL(18,4) | 非必填 | 采购计划上限 | |
| leadTime | 采购提前期 | INT | 非必填（天） | MRP运算、工单齐套 | |
| defaultWarehouse | 默认仓库 | VARCHAR(50) | 来源表：仓库 | 入库默认仓位 | |
| defaultStorageArea | 默认库区 | VARCHAR(50) | 来源表：库区 | 入库默认库区 | |
| batchManage | 批次管理 | BOOLEAN | 必填 | 采购入库、生产领料 | 开启后强制批次追溯 |
| shelfLife | 有效期 | INT | 非必填（天） | 库存预警、先进先出 | |
| weight | 重量 | DECIMAL(18,4) | 非必填 | 发货计费、AGV搬运 | 单位：kg |
| volume | 体积 | DECIMAL(18,4) | 非必填 | 库位容量校验 | 单位：m³ |
| status | 状态 | ENUM | 字典项：启用/禁用 | 所有业务单据 | 禁用后不可入出库 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 入库/出库/盘点/调拨 | 物料出入的计价和计量依据 |
| MES | 生产工单/BOM展开 | BOM展开用料依据，影响工单齐套 |
| QMS | 来料检验/生产检验 | 检验批次的划分依据 |
| 采购 | 供应商报价/采购订单 | 采购价格和交货依据 |

## 关联关系

```
物料 ←→ BOM（父物料/子物料多对多）
物料 ←→ 供应商物料（供应商报价，一对多）
物料 ←→ 客户物料（客户规格匹配，多对多）
物料 ←→ 物料库区配置（默认库区/库位分配）
物料 ←→ BOM（替代物料）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | 预定义选项：物料类型（成品/原料/辅料/包装物）、来货方式（采购/自制/委外）、检验类型（免检/全检/抽检）、状态（启用/禁用） |
| 来源表 | 默认仓库→仓库表，默认库区→库区表 |
| 联动影响 | 物料单位变更→历史单据计价重算；物料类型变更→BOM展开范围变化 |
| 版本生效 | 物料信息变更后可配置是否立即生效或下生效日期 |
