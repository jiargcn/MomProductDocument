# 包装规格

## 概述

包装规格定义包装容器/器具的标准规格，用于库存计算和发货计量。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| packCode | 包装代码 | VARCHAR(50) | 必填 | TODO | 包装的唯一标识 |
| packName | 包装名称 | VARCHAR(200) | 必填 | TODO | 包装显示名称 |
| packDesc | 包装描述 | VARCHAR(500) | 非必填 | TODO | 包装详细描述 |
| packType | 包装类型 | VARCHAR(50) | 非必填 | TODO | 包装类型 |
| unit | 单位 | VARCHAR(10) | 非必填 | TODO | 单位 |
| isEquipment | 是否器具 | BOOLEAN | 默认否 | TODO | 是否为可回收器具 |
| parentPack | 上级包装 | VARCHAR(50) | 非必填 | TODO | 上级包装 |
| length | 长 | DECIMAL(10,2) | 非必填 | TODO | 长度（cm） |
| width | 宽 | DECIMAL(10,2) | 非必填 | TODO | 宽度（cm） |
| height | 高 | DECIMAL(10,2) | 非必填 | TODO | 高度（cm） |
| weight | 重量 | DECIMAL(18,4) | 非必填 | TODO | 重量（kg） |
| volume | 标准体积 | DECIMAL(18,4) | 非必填 | TODO | 标准体积（m³） |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | TODO | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | TODO | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | TODO | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
包装规格（1）←→（N）物料包装信息
包装规格 → 上级包装（自关联）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | packCode 唯一 |
| 联动影响 | isEquipment=true 时参与器具管理 |
