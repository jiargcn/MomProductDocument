# 生产线物料关系管理

## 概述

生产线物料关系定义物料与生产线的关联，用于MES工单展开和产线库存管理。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| materialCode | 物料代码 | VARCHAR(50) | 必填 | 物料归属 | 关联物料 |
| productionLineCode | 生产线代码 | VARCHAR(50) | 必填 | 产线归属 | 关联生产线 |
| isPrimary | 是否主产线 | BOOLEAN | 默认否 | 工单展开（主产线优先） | 是否主产线 |
| defaultQty | 默认用量 | DECIMAL(18,4) | 非必填 | 工单展开（默认带出用量） | 默认用量 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 关系可用性 | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 关系启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 关系停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
物料（1）←→（N）生产线物料关系 → 生产线
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | materialCode + productionLineCode 唯一 |
| 联动影响 | isPrimary=true 时，同物料其他主产线自动取消 |
