## 当前页面事实卡（第二轮源码、DDL 已证实）

> 适用基线：测试环境对应的 `dev` 分支，2026-07-15。`basic_productionlineitem` 的一条记录是“生产线 + 物料”的关系，同时可配置完工收货库位、原料库位和物料类型；它不是 BOM 用量、主产线或工位用料配置。

### 数据对象、业务键与分类视图

| 对象/字段 | 当前实现 | 文档结论 |
| --- | --- | --- |
| 数据表 | `basic_productionlineitem`，主键 `id`；`item_code` 为 `varchar(64) NOT NULL`，`production_line_code` 为可空 `varchar(64)`，`available` 非空默认 `true`。 | 历史草稿的 `materialCode`、`productionLineCode` 可分别映射为 `itemCode`、`productionLineCode`；“主产线、默认用量”不是当前表字段。 |
| 库位与类型 | `fg_location_code`、`raw_location_code`、`item_type` 均可空；`item_type` 为 `varchar(32)`。 | 完工收货/原料库位是可选字段；当前主 Web 页面已注释其表单配置，不能把它们写成必填配置。物料类型由服务从物料主数据带入。 |
| 业务唯一性 | 新增、更新服务按 `productionLineCode + itemCode` 查询重复并排除自身；DDL 未见该组合复合唯一约束。 | 当前组合是服务层业务键；数据库并发/旁路写入保护不足。 |
| 原材料/产成品视图 | 后端提供原材料页（排除 `CCP`、`BCP`）和产成品页（仅 `CCP`、`BCP`）；PC、YCL 有专用导出 Controller。 | 同一关系表按物料类型分视图，不是两张独立关系表。具体 `CCP`/`BCP` 的业务定义仍应以物料字典为准。 |

### 新增约束、选择器与编辑限制

| 字段/动作 | 当前证据 | 当前结论 |
| --- | --- | --- |
| 产线代码 `productionLineCode` | Web 从一次性加载的产线列表中下拉选择；服务查询生产线代码，不存在则报错。 | 产线必须存在。前端未见“只可用产线”筛选，服务也未见可用性校验，不能写成“必须选择可用产线”。 |
| 物料代码 `itemCode` | 当前主 Web 页面选择器筛选物料 `type in (CCP, BCP)` 且 `available = true`；服务只校验物料存在，并在保存时由物料主数据回填 `itemType`。 | 当前该页面日常新增只选择可用 CCP/BCP 物料；服务接口允许的物料类型边界比页面宽，原材料视图的维护入口需另行核验。 |
| 是否可用、有效期 | 前端新增 API 强制 `available = true`；页面提交校验失效时间晚于生效时间；服务校验 `available` 非空。 | Web 新建默认可用；有效期可空，时间先后是前端规则。 |
| 库位字段 | DDL/VO 有完工收货库位、原料库位，但其 Web schema 和必填规则已注释。服务的库位非空校验方法也未被当前创建/更新校验调用。 | 当前不能将库位作为 Web 必填项或已生效的收货/投料指引；需要从 PC/YCL 页面与测试环境进一步核验。 |
| 编辑 | Web 编辑锁定产线和物料，服务更新仍执行业务键校验、物料/产线存在性校验并整体更新。 | 培训按“产线、物料不可编辑”执行；接口绕过页面的修改边界与数据迁移影响待测试验证。 |
| 启用/禁用/删除 | Web 有编辑、启用、禁用、删除动作；启用/禁用后端接口使用 `dbc:productionlineitem:enable/disable` 权限并调用通用工具。删除由通用继承服务执行，页面未见引用检查。 | 启停的接口权限已有源码证据；删除是否检查 MES/WMS 引用尚不能确认。 |

| 主题 | 当前实现 | 文档结论 |

## 概述

生产线物料关系维护某物料可在哪条生产线上执行，并为 MES/BOM 查询提供产线和物料类型维度。当前源码没有证实它会直接创建工单、生成发料或维护产线库存。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| `productionLineCode` | 产线代码 | `varchar(64)` | DDL 可空，服务要求存在 | MES/BOM 查询 | Web 表单必填。 |
| `itemCode` | 物料代码 | `varchar(64)` | 非空 | MES/BOM 查询 | Web 选择 CCP/BCP 可用物料。 |
| `itemType` | 物料类型 | `varchar(32)` | 可空 | PC/YCL 分类视图 | 由服务从物料主数据回填。 |
| `fgLocationCode` | 完工收货库位 | `varchar(64)` | 可空 | 待核验 | 主页面字段已注释。 |
| `rawLocationCode` | 原料库位 | `varchar(64)` | 可空 | 待核验 | 主页面字段已注释。 |
| `available` | 是否可用 | `boolean` | 非空，默认 true | 可用性过滤 | Web 新建默认 true。 |
| `activeTime` / `expireTime` | 生效/失效时间 | `timestamp(6)` | 可空 | 有效期 | 前端校验时间先后。 |

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | 服务层以 `productionLineCode + itemCode` 查重；DDL 未见复合唯一约束。 |
| 联动影响 | 服务保存时按物料主数据回填 `itemType`，关系会进入 PC/YCL 分类和按产线查 BOM 版本的查询范围。 |
