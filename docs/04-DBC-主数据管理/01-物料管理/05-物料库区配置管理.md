## 概述

物料库区配置定义物料与库区/库位的默认分配关系，用于入库时的库区推荐。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| materialCode | 物料代码 | VARCHAR(50) | 必填 | 物料归属 | 关联物料 |
| warehouseCode | 仓库代码 | VARCHAR(50) | 必填 | 仓库归属 | 关联仓库 |
| areaCode | 库区代码 | VARCHAR(50) | 非必填 | 库区归属 | 关联库区 |
| locationCode | 库位代码 | VARCHAR(50) | 非必填 | 库位归属 | 关联库位 |
| isDefault | 是否默认 | BOOLEAN | 默认否 | 入库推荐（默认库区优先推荐） | 是否默认配置 |
| priority | 优先级 | INT | 非必填 | 库区推荐顺序 | 优先级，数字越大越优先 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 配置可用性 | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 配置启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 配置停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
物料（1）←→（N）物料库区配置
物料库区配置 → 仓库
物料库区配置 → 库区
物料库区配置 → 库位
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | materialCode + warehouseCode + areaCode + locationCode 唯一 |
| 联动影响 | isDefault=true 时，同物料同仓库其他默认配置自动取消 |
