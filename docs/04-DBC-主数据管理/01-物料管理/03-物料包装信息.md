## 概述

物料包装信息记录物料的包装规格数量关系，用于入库和发料时的包装单位换算。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| materialCode | 物料代码 | VARCHAR(50) | 必填 | （待补充） | 关联物料编码 |
| unit | 计量单位 | VARCHAR(10) | 必填 | （待补充） | 计量单位 |
| packSpec | 包装规格 | VARCHAR(50) | 必填 | （待补充） | 包装规格描述 |
| packType | 包装类型 | VARCHAR(50) | 非必填 | （待补充） | 包装类型 |
| packQty | 包装数量 | DECIMAL(18,4) | 必填 | （待补充） | 每包装含有的数量 |
| isDefault | 默认包装规格 | BOOLEAN | 默认否 | （待补充） | 是否默认包装 |
| effectiveDate | 生效时间 | DATE | 非必填 | （待补充） | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | （待补充） | 失效时间 |

## 关联关系

```
物料（1）←→（N）物料包装信息
物料包装信息 → 包装规格
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | materialCode + packSpec + unit 唯一 |
| 联动影响 | isDefault=true 时，同物料其他默认规格自动取消 |
