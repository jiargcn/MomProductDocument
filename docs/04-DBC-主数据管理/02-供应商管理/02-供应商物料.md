# 供应商物料

## 概述

供应商物料记录某供应商对特定物料的报价信息，是采购价格执行、供应商比价的依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| id | 主键 | BIGINT | 必填、自增 | | 系统主键 |
| supplierCode | 供应商编码 | VARCHAR(50) | 必填，来源表：供应商 | 采购订单价格引用 | 联合主键之一 |
| materialCode | 物料编码 | VARCHAR(50) | 必填，来源表：物料 | 采购订单价格引用 | 联合主键之一 |
| supplierMaterialCode | 供应商物料编码 | VARCHAR(50) | 非必填 | 供应商对账 | 供应商侧物料编号 |
| supplierMaterialName | 供应商物料名称 | VARCHAR(200) | 非必填 | 供应商对账 | |
| unit | 计量单位 | VARCHAR(10) | 必填 | 采购订单计价 | |
| price | 单价 | DECIMAL(18,4) | 必填 | 采购订单计价 | 含税单价 |
| currency | 币种 | VARCHAR(10) | 字典项：CNY/USD/EUR | 采购订单计价 | 默认CNY |
| minOrderQty | 最小订货量 | DECIMAL(18,4) | 非必填 | 采购订单校验 | 低于此量不允许下单 |
| leadTime | 交期 | INT | 非必填（天） | 采购订单交期 | |
| effectiveDate | 生效日期 | DATE | 必填 | 采购订单计价 | 价格启用时间 |
| invalidDate | 失效日期 | DATE | 非必填 | 采购订单计价 | 空=长期有效 |
| isDefault | 是否默认 | BOOLEAN | 默认否 | 采购下单默认带出 | 同供应商同物料只允许一个默认 |
| status | 状态 | ENUM | 字典项：启用/禁用 | 采购下单 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| 采购 | 采购订单价格 | 下单时自动带出默认报价，支持手动选择 |
| 采购 | 供应商比价 | 同一物料多供应商价格对比 |
| 采购 | 预算成本 | 标准采购成本统计 |

## 关联关系

```
供应商（1）←→（N）供应商物料 → 物料（N）
供应商物料.price → 采购订单.unitPrice
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | 币种（CNY/USD/EUR）、状态（启用/禁用） |
| 来源表 | 供应商编码→供应商表，物料编码→物料表 |
| 联动影响 | 失效日期到达→价格自动失效；修改默认报价→影响新采购单价格 |
| 唯一约束 | 同供应商+同物料+同单位+生效日期唯一 |
