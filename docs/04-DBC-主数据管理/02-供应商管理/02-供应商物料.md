# 供应商物料

## 概述

供应商物料记录某供应商对特定物料的报价信息，是采购价格执行、供应商比价的依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| supplierCode | 供应商编码 | VARCHAR(50) | 必填，来源表：供应商 | 采购订单价格引用（确定报价的供应商） | 供应商的唯一标识，关联到供应商主数据，采购下单时系统根据供应商编码带出报价信息 |
| materialCode | 物料编码 | VARCHAR(50) | 必填，来源表：物料 | 采购订单价格引用（确定报价的物料） | 物料的唯一标识，关联到物料主数据，系统根据物料编码匹配对应供应商的报价 |
| supplierMaterialCode | 供应商物料编码 | VARCHAR(50) | 非必填 | 供应商对账（双方对账时的物料对照） | 供应商侧对应该物料的编码或料号，用于供应商对账时双方物料信息对照 |
| supplierMaterialName | 供应商物料名称 | VARCHAR(200) | 非必填 | 供应商对账（供应商侧物料名称确认） | 供应商侧对该物料的叫法，用于对账时名称核对，防止因物料名称不一致产生争议 |
| unit | 计量单位 | VARCHAR(10) | 必填 | 采购订单计价（数量的计量单位） | 供应商对此物料报价的计量单位，如：个、箱、千克等，采购单的数量以此为单位 |
| price | 单价 | DECIMAL(18,4) | 必填 | 采购订单计价（订单金额=单价×数量） | 供应商对此物料的含税单价，采购下单时自动带出此价格，订单金额=单价×采购数量 |
| currency | 币种 | VARCHAR(10) | 字典项：CNY/USD/EUR | 采购订单计价（币种影响汇率和结算） | 报价的币种，默认CNY，跨境外币采购时按汇率换算，汇率按付款日汇率计算 |
| minOrderQty | 最小订货量 | DECIMAL(18,4) | 非必填 | 采购订单校验（低于此量不允许下单） | 供应商要求的最少下单数量，低于此量供应商可能不接单或单价上浮，系统拦截低于最小量的订单 |
| leadTime | 交期 | INT | 非必填（天） | 采购订单交期（预计到货时间计算） | 供应商承诺的交货周期，从下单到到货的天数，用于采购经理评估来料时间和工单齐套判断 |
| effectiveDate | 生效日期 | DATE | 必填 | 采购订单计价（生效日期前不可使用此价格） | 报价的启用时间，在此日期之后采购订单才可使用此价格下定，之前的订单使用旧价格 |
| invalidDate | 失效日期 | DATE | 非必填 | 采购订单计价（失效后此价格不可使用） | 报价的失效时间，填写后此价格在此日期之后不可用于新建采购单，空值表示长期有效 |
| isDefault | 是否默认 | BOOLEAN | 默认否 | 采购下单默认带出（新建采购单时自动填充） | 标识此报价是否为默认报价，同供应商同物料有多条报价时，默认报价在新建采购单时自动带出 |
| status | 状态 | ENUM | 字典项：启用/禁用 | 采购下单（禁用后不可被采购单引用） | 报价的启用/禁用状态，禁用后不可被新建采购单引用，但历史订单价格不变 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 报价的补充说明，如价格条款（是否含税）、特殊规格要求等 |

## 字段级业务影响

### isDefault（是否默认）

| 场景 | 对采购的影响 |
|------|-------------|
| 设为默认 | 新建采购单时自动带出此供应商此物料的报价 |
| 非默认 | 采购单需手动从报价列表中选择 |
| 多条默认 | 同供应商同物料只允许一个默认，系统自动取消旧的默认 |

### effectiveDate / invalidDate（价格有效期）

| 场景 | 对采购的影响 |
|------|-------------|
| 有效期内 | 新建采购单可以使用此价格 |
| 未到生效日 | 新建采购单不显示此价格，需等生效日期到达 |
| 已过失效日 | 新建采购单不可使用此价格，系统提示价格已过期 |
| 失效后新建报价 | 原有报价失效，新报价生效，历史订单价格不变 |

## 关联关系

```
供应商（1）←→（N）供应商物料 → 物料（N）
供应商物料.price → 采购订单.unitPrice
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | currency（CNY/USD/EUR）、status（启用/禁用） |
| 来源表 | supplierCode→供应商表，materialCode→物料表 |
| 联动影响 | 失效日期到达→价格自动失效；设为默认→原默认报价自动取消；修改价格→不影响历史订单 |
| 唯一约束 | 同供应商+同物料+同单位+生效日期唯一，不可重复建价 |
