# 供应商物料

## 概述

供应商物料记录某供应商对特定物料的报价信息，是采购价格执行、供应商比价的依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| supplierCode | 供应商代码 | VARCHAR(50) | 必填 | 采购订单价格引用（确定报价的供应商） | 供应商的唯一标识，关联到供应商主数据 |
| supplierName | 供应商名称 | VARCHAR(200) | 必填 | 采购订单价格引用（显示供应商名称） | 供应商的名称 |
| materialCode | 物料代码 | VARCHAR(50) | 必填 | 采购订单价格引用（确定报价的物料） | 物料的唯一标识，关联到物料主数据 |
| materialName | 物料名称 | VARCHAR(200) | 必填 | 采购订单价格引用（显示物料名称） | 物料的名称 |
| platformCode | 月台 | VARCHAR(50) | 非必填 | TODO | 供应商月台 |
| stockUnit | 库存计量单位 | VARCHAR(10) | 非必填 | TODO | 库存计量单位 |
| supplierMaterialCode | 供应商物料代码 | VARCHAR(50) | 非必填 | 供应商对账（双方对账时的物料对照） | 供应商侧对应该物料的编码或料号 |
| purchaseUnit | 采购计量单位 | VARCHAR(10) | 非必填 | TODO | 采购计量单位 |
| allowOverIssue | 是否允许超发 | BOOLEAN | 默认否 | TODO | 是否允许超发 |
| conversionRate | 转换率 | DECIMAL(18,4) | 非必填 | TODO | 单位转换率 |
| effectiveDate | 生效时间 | DATE | 非必填 | TODO | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | TODO | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 报价的补充说明 |

## 关联关系

```
供应商（1）←→（N）供应商物料 → 物料（N）
供应商物料.price → 采购订单.unitPrice
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | supplierCode + materialCode 唯一 |
| 联动影响 | 失效日期到达→价格自动失效 |
