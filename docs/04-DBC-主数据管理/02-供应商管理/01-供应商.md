# 供应商

## 概述

供应商主数据记录与企业有采购业务关系的供应方信息，是采购订单、来料检验、付款结算的基础。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| supplierCode | 供应商编码 | VARCHAR(50) | 必填、唯一 | 采购订单、付款 | 系统主键 |
| supplierName | 供应商名称 | VARCHAR(200) | 必填 | 采购订单、报表 | |
| supplierType | 供应商类型 | ENUM | 字典项：生产性/贸易性/个人 | 资质审核、分类统计 | |
| contactPerson | 联系人 | VARCHAR(100) | 非必填 | 采购询价、异常联络 | |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 采购询价、到货通知 | |
| contactEmail | 邮箱 | VARCHAR(100) | 非必填 | 邮件通知、结算对账 | |
| address | 地址 | VARCHAR(500) | 非必填 | 采购到货、物流跟踪 | |
| bankName | 开户银行 | VARCHAR(100) | 非必填 | 付款结算 | |
| bankAccount | 银行账号 | VARCHAR(50) | 非必填 | 付款结算 | |
| taxNo | 税号 | VARCHAR(50) | 非必填 | 开票、税务申报 | |
| paymentTerm | 付款账期 | VARCHAR(50) | 字典项：月结30/预付/到付 | 付款计划生成 | |
| creditRating | 信用等级 | ENUM | 字典项：A/B/C/D | 采购分配策略 | |
| status | 状态 | ENUM | 字典项：启用/禁用 | 采购下单 | 禁用后不可新建采购单 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| 采购 | 采购订单 | 供应商编码/名称/联系方式显示在订单上 |
| QMS | 来料检验 | 供应商检验历史记录、供应商质量评分 |
| WMS | 到货接收 | 按供应商分类显示待检物料 |
| 财务 | 付款结算 | 银行信息、付款账期用于付款计划 |
| 基础数据 | 供应商物料 | 物料与供应商的报价关系 |

## 关联关系

```
供应商 ←→ 供应商物料（物料报价关系，一对多）
供应商 ←→ 采购订单（采购业务流）
供应商 ←→ 来料检验记录（质量追溯）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | 供应商类型（生产性/贸易性/个人）、付款账期（月结30/预付/到付）、信用等级（A/B/C/D）、状态（启用/禁用） |
| 联动影响 | 供应商禁用→采购订单中该供应商不可选；付款账期→影响付款计划生成 |
