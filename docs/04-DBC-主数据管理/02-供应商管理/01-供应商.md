# 供应商

## 概述

供应商主数据记录与企业有采购业务关系的供应方信息，是采购订单、来料检验、付款结算的基础。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| supplierCode | 供应商编码 | VARCHAR(50) | 必填、唯一 | 采购订单（显示供应商信息）、付款（结算对象） | 供应商在系统中的唯一身份标识，所有采购单、付款单通过此编码引用对应供应商，系统主键 |
| supplierName | 供应商名称 | VARCHAR(200) | 必填 | 采购订单（显示供应商名称）、报表 | 供应商的全称，用于单据显示和业务人员识别，应与营业执照一致 |
| supplierType | 供应商类型 | ENUM | 字典项：生产性/贸易性/个人 | 资质审核（生产性需现场审核）、分类统计（按类型分析供应商结构） | 区分供应商的性质，生产性供应商需现场审核厂房设备，贸易性供应商仅需代理资质，个人供应商有特殊税务处理 |
| contactPerson | 联系人 | VARCHAR(100) | 非必填 | 采购询价（联络供应商）、异常处理（到货问题联络） | 日常采购业务的主要联络人姓名 |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 采购询价（电话联络）、到货通知（告知送货时间） | 联系人电话，用于采购过程中的电话沟通 |
| contactEmail | 邮箱 | VARCHAR(100) | 非必填 | 邮件通知（订单发送、对账函发送）、结算对账（电子对账单据接收） | 供应商接收邮件的电子邮箱，用于订单和对账单的电子传递 |
| address | 地址 | VARCHAR(500) | 非必填 | 采购到货（物流跟踪依据）、送货地址（入库单显示供应商发货地址） | 供应商的发货地址，用于采购到货的物流跟踪和入库单的地址显示 |
| bankName | 开户银行 | VARCHAR(100) | 非必填 | 付款结算（收款方银行信息）、开票依据 | 供应商收款的开户银行名称，用于付款时填写电汇信息 |
| bankAccount | 银行账号 | VARCHAR(50) | 非必填 | 付款结算（收款方账号）、财务对账 | 供应商的银行账号，用于付款时的电汇账号填写 |
| taxNo | 税号 | VARCHAR(50) | 非必填 | 开票（增值税发票抬头）、税务申报 | 供应商的纳税人识别号，用于采购发票开具和税务申报 |
| paymentTerm | 付款账期 | VARCHAR(50) | 字典项：月结30/预付/到付等 | 付款计划生成（按账期生成付款计划）、财务预算 | 约定货款的结算方式，月结30表示收货后30天付款，预付表示发货前付款，到付表示货到付款 |
| creditRating | 信用等级 | ENUM | 字典项：A/B/C/D | 采购分配策略（高信用供应商优先分配订单）、风险控制 | 供应商的信用评级结果，A级优先分配采购份额，D级供应商下单需额外审批 |
| status | 状态 | ENUM | 字典项：启用/禁用 | 采购下单（禁用后不可新建采购单）、来料检验（禁用后不出现） | 供应商的启用/禁用状态，禁用后不可在新建采购订单中被选择，但历史数据保留 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 供应商的补充说明，如特殊资质、合作历史、注意事项等 |

## 字段级业务影响

### status（状态）

| 状态 | 对采购的影响 | 对QMS的影响 | 注意事项 |
|------|-------------|-------------|----------|
| **启用** | 可正常新建采购订单，可分配采购份额 | 供应商出现在来料检验的供应商列表中 | |
| **禁用** | 不可新建采购订单，历史采购单可查 | 供应商从来料检验列表中移除 | 禁用前应确认无进行中的采购单和未结清付款 |

### creditRating（信用等级）

| 等级 | 对采购的影响 | 典型场景 |
|------|-------------|----------|
| **A** | 优先分配采购份额，可享受更长的付款账期 | 长期稳定合作、质量合格率>99% |
| **B** | 正常分配采购份额 | 质量合格、偶尔交期延迟 |
| **C** | 减少采购份额，下单需额外说明 | 多次质量异常或交期延误 |
| **D** | 限制下单，仅限紧急采购，管理层审批 | 严重质量或交期问题 |

## 关联关系

```
供应商（1）←→（N）供应商物料 → 物料
供应商 ←→ 采购订单（采购业务流）
供应商 ←→ 来料检验记录（质量追溯）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | supplierType（生产性/贸易性/个人）、paymentTerm（月结30/预付/到付）、creditRating（A/B/C/D）、status（启用/禁用） |
| 联动影响 | 禁用→采购订单中该供应商不可选；付款账期→影响付款计划生成；信用等级→影响采购分配和审批流程 |
