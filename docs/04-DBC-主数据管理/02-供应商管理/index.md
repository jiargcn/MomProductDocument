# 供应商管理

> 阅读对象：测试、实施、运维（主）；采购与主数据维护人员（顺带）。

## 这一组业务解决什么问题

供应商管理维护采购协同中的供应方身份，以及供应方物料编码与系统物料的对应口径。读完本组文档，应能判断某次采购单据、收货或追溯查询中，供应商或供应商物料选不到、匹配不上，是因为供应商资料本身的问题，还是物料侧配置的问题。

## 如何使用本组文档

| 你的目的 | 建议阅读 |
| --- | --- |
| 想理解供应商如何与采购、收货建立对应关系 | 本页「这一组业务解决什么问题」与「关键业务对象与关系」。 |
| 要新建或维护供应商、供应商物料 | 按下方「建议学习与操作顺序」逐项进入对应叶页。 |
| 采购或收货中供应商/物料选不到 | 本页「常见问题与相关分组」。 |

## 常见问题与相关分组

供应商无法在采购或收货业务中选择时，先确认供应商状态、物料匹配和来源单据条件；物料自身的识别、单位和包装问题回到[物料管理](../01-物料管理/index.md)，本组不重复维护物料本身的属性。
