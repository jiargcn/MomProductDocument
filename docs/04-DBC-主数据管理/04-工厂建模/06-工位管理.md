# 工位管理

## 概述

工位是生产线上的具体作业点，定义工人或设备执行工序的具体位置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| stationCode | 工位编码 | VARCHAR(50) | 必填、唯一 | MES报工（报工时选择具体工位）、工序流转（物料在工位间转移） | 工位的唯一身份标识，用于标识具体工位，系统主键 |
| stationName | 工位名称 | VARCHAR(200) | 必填 | MES显示（工位名称展示） | 工位的显示名称，如"装配工位A1-01" |
| productionLineCode | 所属生产线 | VARCHAR(50) | 必填，来源表：生产线 | 组织归属（明确工位属于哪条产线）、报工归属（报工记录归属产线） | 工位所属生产线的编码 |
| stationType | 工位类型 | ENUM | 字典项：加工/装配/检测/包装 | 工序分配（不同类型工位分配对应的工序）、报工方式（检测工位报工可能需要关联检验结果） | 工位的主要作业类型，影响可分配的工序类型和报工时的必填字段 |
| deviceCode | 绑定设备 | VARCHAR(50) | 非必填，来源表：设备台账 | 设备点检联动（工位作业前需先完成设备点检）、设备运行状态采集（工位设备状态影响工位可用性） | 工位绑定的设备编码，绑定后此工位开工前需先确认设备点检完成状态 |
| workerCapacity | 人员配置数 | INT | 默认1 | MES派工（同一工位可同时容纳的作业人员数量）、工位利用率统计 | 同一工位可同时容纳的作业人员数量，用于派工时判断工位是否有空位 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES派工（禁用工位不可派工）、报工（禁用工位的报工记录仍可查） | 工位的启用/禁用状态，禁用后不可派工 |

## 字段级业务影响

### deviceCode（绑定设备）

| 场景 | 对MES的影响 |
|------|-------------|
| 绑定设备且设备已完成点检 | 工位可正常开工 |
| 绑定设备但设备点检未完成 | 工位开工时提示点检未完成，需先完成点检 |
| 未绑定设备 | 工位可直接开工，不触发设备点检校验 |

## 关联关系

```
生产线（1）←→（N）工位
工位 → 设备（绑定关系）
工位 → MES报工记录
工位 → 工序配置（工位-工序关系）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | stationType（加工/装配/检测/包装）、status（启用/禁用） |
| 来源表 | productionLineCode→生产线表，deviceCode→设备台账表 |
| 联动影响 | 绑定设备→开工前触发点检确认；工位类型→影响可分配的工序类型 |
