# 工位管理

## 概述

工位是生产线上的具体作业点，定义工人或设备执行工序的具体位置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| stationCode | 工位编码 | VARCHAR(50) | 必填、唯一 | MES报工点 | 系统主键 |
| stationName | 工位名称 | VARCHAR(200) | 必填 | MES显示 | |
| productionLineCode | 所属生产线 | VARCHAR(50) | 必填，来源表：生产线 | 组织归属 | |
| stationType | 工位类型 | ENUM | 字典项：加工/装配/检测/包装 | 工序分配 | |
| deviceCode | 绑定设备 | VARCHAR(50) | 非必填，来源表：设备台账 | 设备点检联动 | |
| workerCapacity | 人员配置数 | INT | 默认1 | MES派工 | 同工位可容纳人数 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES派工 | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 报工 | 报工记录归属具体工位 |
| MES | 工序流转 | 工位间的物料移转 |
| MES | 设备点检 | 绑定设备的工位需执行点检 |
| EAM | 设备运行数据 | 工位设备运行状态采集 |

## 关联关系

```
生产线（1）←→（N）工位
工位 → 设备（绑定关系）
工位 → MES报工记录
工位 → 工序配置（工位-工序关系）
```
