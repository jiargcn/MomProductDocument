## 概述

生产线是车间内的生产流水线定义，连接工序和工位，是 MES 工单展开和工序流转的载体。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| productionLineCode | 生产线编码 | VARCHAR(50) | 必填、唯一 | MES工单展开（工单需分配到生产线）、报工归属（报工记录归属生产线） | 生产线的唯一身份标识，用于标识具体产线，系统主键 |
| productionLineName | 生产线名称 | VARCHAR(200) | 必填 | MES显示（产线名称展示）、报表（按产线统计） | 生产线的显示名称，如"装配线A1" |
| workshopCode | 所属车间 | VARCHAR(50) | 必填，来源表：车间 | 组织归属（明确产线属于哪个车间）、报表层级（按车间-产线层级汇总） | 生产线所属车间的编码，定义产线的上级组织 |
| productionLineType | 生产线类型 | ENUM | 字典项：连续型/离散型/批量型 | （待补充） | 生产线的产品流动方式，连续型适用于流水线生产，离散型适用于单件生产，批量型适用于分批生产 |
| cycleTime | 标准产能节拍 | INT | 非必填（秒/件） | （待补充） | 生产一件产品需要的标准时间，单位秒/件 |
| shiftCount | 班次 | INT | 默认1 | （待补充） | 生产线每天工作的班次数，如每天3班（早中晚）则shiftCount=3 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES工单分配（禁用产线不可分配工单）、报工（禁用产线的报工记录仍可查） | 生产线的启用/禁用状态，禁用后不可分配新工单 |

## 关联关系

```
车间（1）←→（N）生产线
生产线（1）←→（N）工位
生产线（1）←→（N）工序配置
生产线 → MES工单
```
