# 模具管理

## 概述

模具主数据记录工装模具的全生命周期信息，从购置入库到维修保养的全过程管理。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| moldCode | 模具编码 | VARCHAR(50) | 必填、唯一 | EAM全流程 | 系统主键 |
| moldName | 模具名称 | VARCHAR(200) | 必填 | EAM显示 | |
| moldType | 模具类型 | VARCHAR(50) | 必填，来源表：模具类型 | 分类统计 | |
| specification | 规格 | VARCHAR(100) | 非必填 | 采购参考 | |
| manufacturer | 制造商 | VARCHAR(100) | 非必填 | 维修联络 | |
| purchaseDate | 购置日期 | DATE | 非必填 | 折旧计算 | |
| lifecycleCount | 设计寿命 | INT | 非必填（次数） | EAM寿命预警 | 达到后提醒报废 |
| currentCount | 当前使用次数 | INT | 系统自动累计 | EAM寿命跟踪 | |
| status | 状态 | ENUM | 字典项：闲置/在用/维修/报废 | EAM工装管理 | |
| location | 当前库位 | VARCHAR(50) | 非必填，来源表：库位 | EAM库区管理 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| EAM | 工装台账 | 模具资产登记 |
| EAM | 出入库 | 工装借出/归还记录 |
| EAM | 维修保养 | 使用次数累积触发保养 |
| MES | 工序执行 | 模具安装到工位使用 |

## 关联关系

```
模具类型（1）←→（N）模具
模具 → 库位（当前存放位置）
模具 → 工装台账明细（使用寿命记录）
```
