## 概述

库位是库区内可存放物料的最小单元，定义物料在仓库中的具体存放位置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| locationCode | 代码 | VARCHAR(50) | 必填、唯一 | WMS所有库存业务（入库/出库/盘点时指定具体位置） | 库位的唯一身份标识，系统主键 |
| locationName | 名称 | VARCHAR(200) | 必填 | 库位显示（便于人员识别） | 库位的显示名称 |
| warehouseCode | 仓库代码 | VARCHAR(50) | 必填 | 库位归属（明确库位属于哪个仓库） | 库位所属的仓库编码 |
| areaCode | 库区代码 | VARCHAR(50) | 必填 | 库位归属（明确库位属于哪个库区） | 库位所属的库区编码 |
| locationGroupCode | 库位组代码 | VARCHAR(50) | 非必填 | 库位分组（按组统计） | 库位所属的库位组 |
| erpLocationType | ERP库位类型 | VARCHAR(50) | 非必填 | ERP集成（ERP系统库位类型对照） | ERP系统的库位类型 |
| erpLocationCode | ERP库位代码 | VARCHAR(50) | 非必填 | ERP集成（ERP系统库位代码对照） | ERP系统的库位代码 |
| lane | 巷道 | VARCHAR(50) | 非必填 | 库位定位（巷道导航） | 库位所在巷道 |
| shelf | 货架 | VARCHAR(50) | 非必填 | 库位定位（货架编号） | 货架编号 |
| row | 行 | INT | 非必填 | 库位定位（行号） | 库位行号 |
| col | 列 | INT | 非必填 | 库位定位（列号） | 库位列号 |
| priority | 备货优先级 | INT | 非必填 | 备货推荐（优先级高的先备货） | 备货优先级，数字越大优先级越高 |
| maxWeight | 最大承重 | DECIMAL(18,4) | 非必填 | 库位承重校验（入库时校验物料重量是否超限） | 库位的最大承重限制（kg） |
| maxArea | 最大面积 | DECIMAL(18,4) | 非必填 | 库位容量校验（面积校验） | 最大面积（m²） |
| stdVolume | 标准体积 | DECIMAL(18,4) | 非必填 | 库位空间计算（标准容量） | 标准体积（m³） |
| maxVolume | 最大体积 | DECIMAL(18,4) | 非必填 | 库位容量校验（最大容量限制） | 最大体积（m³） |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 入库库位分配（不可用时不出现在推荐列表） | 库位的可用状态 |
| effectiveDate | 生效时间 | DATE | 非必填 | 库位启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 库位停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 库位的补充说明 |

## 关联关系

```
库区（1）←→（N）库位
库位 → 库存余额（按库位维度库存）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | locationCode 唯一 |
| 来源表 | warehouseCode → 仓库表，areaCode → 库区表 |
| 联动影响 | isEnabled设为否→入库时不推荐，但不影响出库和盘点 |
