# 库位管理

## 概述

库位是库区内可存放物料的最小单元，定义物料在仓库中的具体存放位置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| locationCode | 库位编码 | VARCHAR(50) | 必填、唯一 | WMS所有库存业务 | 系统主键 |
| locationName | 库位名称 | VARCHAR(200) | 必填 | 库位显示 | |
| areaCode | 所属库区 | VARCHAR(50) | 必填，来源表：库区 | 库位归属组织 | |
| locationType | 库位类型 | ENUM | 字典项：货架/地堆/桶槽/容器 | 存放方式和取料策略 | |
| maxCapacity | 最大容量 | DECIMAL(18,4) | 非必填 | 库位容量校验 | 单位：立方米或托 |
| maxWeight | 最大承重 | DECIMAL(18,4) | 非必填 | 库位承重校验 | 单位：kg |
| length | 长度 | DECIMAL(10,2) | 非必填 | 库位空间计算 | 单位：m |
| width | 宽度 | DECIMAL(10,2) | 非必填 | 库位空间计算 | 单位：m |
| height | 高度 | DECIMAL(10,2) | 非必填 | 库位空间计算 | 单位：m |
| isOccupied | 是否占用 | BOOLEAN | 系统自动更新 | 库位状态显示 | |
| isAvailable | 是否可用 | BOOLEAN | 默认是 | 入库库位分配 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 入库分配 | 按库位容量和可用状态分配 |
| WMS | 发料取料 | 指定库位取料路径 |
| WMS | 盘点 | 按库位维度盘点 |
| WMS | 库位容量校验 | 存放时校验库位是否超限 |

## 关联关系

```
库区（1）←→（N）库位
库位 → 库存余额（按库位维度库存）
库位 → 物料库区配置（物料与库位偏好关系）
```
