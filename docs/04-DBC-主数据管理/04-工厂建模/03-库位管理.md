# 库位管理

## 概述

库位是库区内可存放物料的最小单元，定义物料在仓库中的具体存放位置。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| locationCode | 库位编码 | VARCHAR(50) | 必填、唯一 | WMS所有库存业务（入库/出库/盘点时指定具体位置） | 库位的唯一身份标识，格式通常包含仓库-库区-库位编号，如"A01-01-001"代表仓库A01库区01的001号库位，系统主键 |
| locationName | 库位名称 | VARCHAR(200) | 必填 | 库位显示（便于人员识别） | 库位的显示名称，通常与编码对应，如"A区-1号线-001" |
| areaCode | 所属库区 | VARCHAR(50) | 必填，来源表：库区 | 库位归属（明确库位属于哪个库区）、层级汇总（按库区统计库存） | 库位所属的库区编码，定义库位的上级组织关系 |
| locationType | 库位类型 | ENUM | 字典项：货架/地堆/桶槽/容器 | 存放方式（决定物料如何放置和取料）、库位容量计算方式 | 库位的物理形式，货架需考虑层高和承重，地堆直接码放，桶槽用于液体物料，容器用于周转箱等可移动存放单元 |
| maxCapacity | 最大容量 | DECIMAL(18,4) | 非必填 | 库位容量校验（入库时校验是否超过容量）、库位推荐（容量不足时推荐其他库位） | 库位的最大体积容量，单位通常为立方米或卡板数，入库时系统校验物料总体积是否超过此值 |
| maxWeight | 最大承重 | DECIMAL(18,4) | 非必填 | 库位承重校验（入库时校验物料重量是否超限） | 库位的最大承重限制，单位kg，入库时系统校验物料总重量是否超过此值，超重时不允许存放 |
| length | 长度 | DECIMAL(10,2) | 非必填 | 库位空间计算（计算可用体积）、库位推荐（匹配物料尺寸） | 库位内部空间的长度，单位米，用于计算可用库存体积和匹配物料存放 |
| width | 宽度 | DECIMAL(10,2) | 非必填 | 库位空间计算 | 库位内部空间的宽度，单位米 |
| height | 高度 | DECIMAL(10,2) | 非必填 | 库位空间计算（与长度宽度共同计算可用体积） | 库位内部空间的高度，单位米，用于计算可用体积和对高层货架库位的存放校验 |
| isOccupied | 是否占用 | BOOLEAN | 系统自动更新 | 库位状态显示（显示当前是否已有物料存放） | 系统根据实际库存自动更新的状态标识，有库存时为true，无库存时为false，用于库位状态看板 |
| isAvailable | 是否可用 | BOOLEAN | 默认是 | 入库库位分配（不可用时不出现在推荐列表） | 人工可设置的可用性标识，设为不可用后入库时不推荐此库位，但不影响历史库存数据的查看 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 库位的补充说明，如特殊存放要求、编号规则说明等 |

## 字段级业务影响

### locationType（库位类型）

| 类型 | 对入库的影响 | 对发料的影响 | 适用场景 |
|------|-------------|-------------|----------|
| **货架** | 物料上架到指定货架位，考虑层高和承重 | 拣货时按货架位顺序行走取料 | 标准化物料，小件货物 |
| **地堆** | 物料直接码放在地面，按区域划分 | 叉车直接叉取，拣货路径不同 | 大件货物，无需上架 |
| **桶槽** | 液体/散料存入专用容器，需配合液位管理 | 抽取或倾倒，按量出库 | 化工原料、液体物料 |
| **容器** | 物料放入标准容器，容器有独立编码 | 整容器发料，扫描容器编码快速出入库 | 周转箱、托盘 |

### isAvailable（是否可用）

| 值 | 对入库的影响 |
|----|-------------|
| **是** | 库位正常出现在入库推荐列表中 |
| **否** | 库位不出现在入库推荐列表中，但已有库存正常可用 |

## 关联关系

```
库区（1）←→（N）库位
库位 → 库存余额（按库位维度库存）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | locationType（货架/地堆/桶槽/容器） |
| 来源表 | areaCode→库区表 |
| 联动影响 | isAvailable设为否→入库时不推荐，但不影响出库和盘点；库位被占用时（isOccupied=true）不可删除 |
