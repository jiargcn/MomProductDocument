# 月台管理

## 概述

月台管理定义仓库的收货/发货月台信息，用于物流调度和到货预排。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| platformCode | 月台代码 | VARCHAR(50) | 必填 | 物流调度（确定月台地址） | 月台的唯一标识 |
| platformName | 月台名称 | VARCHAR(200) | 必填 | 月台显示 | 月台的显示名称 |
| warehouseCode | 仓库代码 | VARCHAR(50) | 必填 | 月台归属（确定属于哪个仓库） | 所属仓库 |
| address | 地址 | VARCHAR(500) | 非必填 | 发货单据打印（打印发货地址） | 月台地址 |
| contactPerson | 联系人 | VARCHAR(100) | 非必填 | 物流跟踪联络 | 联系人姓名 |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 物流跟踪联络 | 联系电话 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 月台分配（不可用时不可选） | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 月台启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 月台停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
仓库（1）←→（N）月台
月台 → 发货计划（到货地址）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 唯一约束 | platformCode 唯一 |
