# 仓库管理

## 概述

仓库主数据定义企业仓储实体的物理位置，是所有库存业务（入库/出库/盘点/调拨）的基础计量单位。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| warehouseCode | 仓库编码 | VARCHAR(50) | 必填、唯一 | 所有库存业务 | 系统主键 |
| warehouseName | 仓库名称 | VARCHAR(200) | 必填 | 库存报表、库区分配 | |
| warehouseType | 仓库类型 | ENUM | 字典项：原材料库/成品库/半成品库/包材库/工装库 | 库区分类、权限隔离 | |
| warehouseAddress | 地址 | VARCHAR(500) | 非必填 | 库管巡检 | |
| manager | 仓库管理员 | VARCHAR(50) | 非必填，来源表：用户 | 职责归属 | |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 异常联络 | |
| status | 状态 | ENUM | 字典项：启用/禁用 | 库区分配、入库选择 | 禁用后不可作为入库目标 |
| allowNegativeStock | 允许负库存 | BOOLEAN | 默认否 | WMS库存校验 | 开启后出库可超库存 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 入库/出库/调拨 | 仓库作为业务单据的归属组织 |
