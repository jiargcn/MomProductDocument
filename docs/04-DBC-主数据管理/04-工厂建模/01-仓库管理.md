# 仓库管理

## 概述

仓库主数据定义企业仓储实体的物理位置，是所有库存业务（入库/出库/盘点/调拨）的基础计量单位。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| warehouseCode | 仓库编码 | VARCHAR(50) | 必填、唯一 | 所有库存业务（入库单/出库单/调拨单的组织归属） | 仓库的唯一身份标识，所有库存业务单据通过此编码关联仓库，系统主键 |
| warehouseName | 仓库名称 | VARCHAR(200) | 必填 | 库存报表（按仓库维度统计）、库区分配（显示仓库名） | 仓库的显示名称，用于各单据和报表中的展示 |
| warehouseType | 仓库类型 | ENUM | 字典项：原材料库/成品库/半成品库/包材库/工装库 | 库区分类（按类型划分库区）、库存报表（按类型统计）、权限隔离（不同类型仓库由不同角色管理） | 仓库的类型分类，影响库存统计口径和库区分配的默认类型 |
| warehouseAddress | 地址 | VARCHAR(500) | 非必填 | 库管巡检（物理位置指引） | 仓库的物理地址，用于库管员巡检和到货时的送货地址确认 |
| manager | 仓库管理员 | VARCHAR(50) | 非必填，来源表：用户 | 职责归属（异常问题找对应管理员） | 仓库的负责人账号，用于异常问题时通知和职责追溯 |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 异常联络（仓库异常时联络） | 仓库管理员的联系电话，用于紧急情况时联络 |
| status | 状态 | ENUM | 字典项：启用/禁用 | 入库选择（禁用仓库不可选作入库目标）、库区分配（禁用后下属库区一并不可用） | 仓库的启用/禁用状态，禁用后不可作为新建入库单的目标仓库 |
| allowNegativeStock | 允许负库存 | BOOLEAN | 默认否 | WMS库存校验（开启后出库可超出库存数量） | 是否允许库存出现负数（出库数量超过可用库存），开启后即使库存为0也可以做出库，关闭时出库数量不得大于可用库存 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 仓库的补充说明，如特殊存放条件、管理要求等 |

## 字段级业务影响

### status（状态）

| 状态 | 对WMS的影响 | 注意事项 |
|------|-------------|----------|
| **启用** | 可正常选择作为入库/出库目标 | |
| **禁用** | 不可选作入库目标，出库单不可选此仓库 | 禁用前应确认无在此仓库的进行中单据 |

### allowNegativeStock（允许负库存）

| 值 | 对WMS出库的影响 | 典型场景 |
|----|---------------|----------|
| **false** | 出库数量不得大于当前可用库存，不足时提示库存不足 | 一般制造业严格管控库存，关闭此选项 |
| **true** | 出库可超出当前库存，显示负数库存，便于紧急借料后补回 | 内部调拨频繁的场景，允许临时负数后快速平账 |

## 关联关系

```
仓库（1）←→（N）库区
仓库 → 库存余额（按仓库维度统计）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | warehouseType（原材料库/成品库/半成品库/包材库/工装库）、status（启用/禁用） |
| 来源表 | manager→用户表 |
| 联动影响 | 仓库禁用→下属库区自动不可用；allowNegativeStock开启→库存可出现负数，需配合盘点修正 |
