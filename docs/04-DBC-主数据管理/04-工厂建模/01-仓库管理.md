## 概述

仓库主数据定义企业仓储实体的物理位置，是所有库存业务（入库/出库/盘点/调拨）的基础计量单位。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| warehouseCode | 代码 | VARCHAR(50) | 必填、唯一 | 所有库存业务（入库单/出库单/调拨单的组织归属） | 仓库的唯一身份标识，系统主键 |
| warehouseName | 名称 | VARCHAR(200) | 必填 | 库存报表（按仓库维度统计）、库区分配（显示仓库名） | 仓库的显示名称 |
| warehouseType | 类型 | VARCHAR(50) | 非必填 | 库区分类（按类型划分库区）、库存报表（按类型统计） | 仓库的类型分类 |
| description | 描述 | VARCHAR(500) | 非必填 | 仓库说明 | 仓库的描述信息 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 入库选择（禁用仓库不可选作入库目标） | 仓库的可用状态 |
| effectiveDate | 生效时间 | DATE | 非必填 | 仓库启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 仓库停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 仓库的补充说明 |

## 关联关系

```
仓库（1）←→（N）库区
仓库 → 库存余额（按仓库维度统计）
```
