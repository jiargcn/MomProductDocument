# 车间管理

> 当前基线：`dev`（测试环境目标基线），源码与 DDL 取证日期：2026-07-16。
> 页面定位：维护生产空间的车间主数据，供生产线、工位及 MES 相关业务引用。当前模型不包含工厂/组织、负责人或车间上下级层次字段；历史草稿中的这些字段不代表当前实现。

## 当前实现摘要

持久化表为 `basic_workshop`。页面支持新增、编辑、启停、删除、导入和导出；服务使用专用保存逻辑，按代码查重。类型使用 `WORKSHOP_TYPE` 字典，但当前 DDL、普通保存和导入均未将其设为必填。

代码唯一性仅在服务层保证；导入与普通新增的可用状态、类型规则不一致，删除未见下游引用保护，详见[产品差距总账](../../15-版本路线图/03-问题与待确认/产品差距总账.md) GAP-053。

## 字段与新增约束

| 技术字段 | 中文名 | DDL 类型/约束 | 当前页面规则 | 当前服务规则 |
| --- | --- | --- | --- | --- |
| `code` | 代码 | `varchar(64)`，非空 | 必填、最多 50 字符、`validateYS` 校验；编辑锁定。 | 非空、按代码排重；更新排除自身 ID。 |
| `name` | 名称 | `varchar(64)`，可空 | 必填、最多 50 字符。 | 普通保存未见服务必填；导入显式要求非空。 |
| `type` | 类型 | `varchar(64)`，可空 | `WORKSHOP_TYPE` 字典；当前页面无必填规则。 | 未见普通保存或导入的必填/字典存在性校验。 |
| `available` | 是否可用 | `boolean`，非空，默认 `true` | 开关控件，默认 `true`，必填。 | 普通保存校验非空。 |
| `activeTime` / `expireTime` | 生效/失效时间 | `timestamp(6)`，可空 | 日期时间选择器。 | 未见时间区间校验。 |
| `remark` | 备注 | `text`，可空 | 最多 50 字符。 | 无额外校验。 |

`description`、审计、删除、扩展属性、并发戳、站点和租户为系统/持久化字段；`manager`、`parentWorkshop` 等历史字段不在当前 DDL、VO、导入类和页面配置中。

## 编辑、启停与删除

| 动作 | 当前可证实行为 | 使用口径 |
| --- | --- | --- |
| 编辑 | Web 锁定代码，服务仍接收代码并按更新 ID 排重后整体更新。 | 正常页面不修改代码；接口是否可改码需以实际请求与测试验证为准。 |
| 启用/停用 | 列表提供启停；服务的可选查询按 `available=true` 过滤。 | 停用前应核查生产线、工位、MES 任务、质量与异常引用；当前未见统一拦截。 |
| 删除 | 服务仅校验车间存在后删除。 | 已被生产线/工位/执行数据引用的车间不得按“可直接删除”培训。 |

前端按钮权限标识为 `dbc:workshop:create/import/export/update/enable/disable/delete`。后端权限策略需纳入 RBAC 实测，按钮标识不构成服务端授权结论。

## 导入规则

`WorkshopImportExcelVo` 字段顺序为：代码、名称、类型、是否可用、生效时间、失效时间、备注。

| 校验/模式 | 当前规则 |
| --- | --- |
| 必填 | 代码、名称；Excel 注解还标记是否可用必填，但服务校验代码未再校验。 |
| 类型 | 使用 `WORKSHOP_TYPE` 字典转换；未标记必填。 |
| 既有记录 | 按代码查找；不存在且非覆盖模式时插入，存在且非仅新增模式时更新。 |
| 状态 | 新增导入记录被强制写为 `available=true`；更新记录按导入数据写入。 |
| 错误处理 | 当前专用导入方法返回错误列表，但未见“有错误且非部分更新即整批回滚”的实现。 |

模板不包含历史的负责人、父车间、工厂/组织字段，也不包含审计、删除、扩展、并发、站点和租户字段。

## 列表、详情与快速跳转

当前默认列顺序为：代码、名称、类型、是否可用、生效时间、失效时间、备注、创建时间、创建者、最后更新时间、最后更新者、操作。默认筛选为代码、名称、类型；`available` 在页面中未设为搜索字段，但其搜索默认值已配置，需测试环境确认是否生效。

建议培训主列表优先显示：代码、名称、类型、是否可用、有效期；备注与审计字段作为补充。

当前使用通用详情组件，未见车间专用关联 Tab。后续建议：

| 分组 | 内容 | 建议跳转 |
| --- | --- | --- |
| 基本信息 | 代码、名称、类型、可用状态 | 生产线、工位 |
| 有效期与审计 | 生效/失效时间、备注、审计字段 | 操作日志（待公共能力取证） |
| 下游引用 | 生产线、工位、MES 执行、QMS、ANDON | 各关联查询预填车间条件（待实现/验证） |
