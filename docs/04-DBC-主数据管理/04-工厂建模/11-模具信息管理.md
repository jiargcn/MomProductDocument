# 模具管理

## 概述

模具主数据记录工装模具的全生命周期信息，从购置入库到维修保养的全过程管理。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| moldCode | 模具编码 | VARCHAR(50) | 必填、唯一 | EAM全流程（工装借出/归还/维修/保养的关联对象） | 模具的唯一身份标识，用于标识具体模具，系统主键 |
| moldName | 模具名称 | VARCHAR(200) | 必填 | EAM显示（模具名称展示） | 模具的显示名称，如"注塑模具-A001" |
| moldType | 模具类型 | VARCHAR(50) | 必填，来源表：模具类型 | 分类统计（按类型汇总模具数量和寿命）、保养计划（不同类型保养周期不同） | 模具所属的类型分类，关联模具类型主数据，用于分类管理和保养计划制定 |
| specification | 规格 | VARCHAR(100) | 非必填 | 采购参考（采购新模具时的规格依据） | 模具的规格型号描述，如"500T-型腔2" |
| manufacturer | 制造商 | VARCHAR(100) | 非必填 | 维修联络（模具故障时联络原厂） | 模具的生产厂家，用于维修时的技术联络 |
| purchaseDate | 购置日期 | DATE | 非必填 | 折旧计算（按购置日期计算折旧）、寿命统计 | 模具的购置时间，用于计算折旧和设备役龄 |
| lifecycleCount | 设计寿命 | INT | 非必填（次数） | EAM寿命预警（使用次数达到设计寿命时提醒报废） | 模具设计的使用总次数，达到此值后系统预警建议报废或大修 |
| currentCount | 当前使用次数 | INT | 系统自动累计 | EAM寿命跟踪（实时显示已使用次数）、寿命预警 | 系统自动累计的模具实际使用次数，每完成一次生产循环累加1，用于判断寿命消耗 |
| status | 状态 | ENUM | 字典项：闲置/在用/维修/报废 | EAM工装管理（不同状态决定工装是否可借出）、MES工序执行（维修中工装不可用） | 模具的当前使用状态，闲置时可借出，在用中不可借出，维修中不可使用，报废后不可借出 |
| location | 当前库位 | VARCHAR(50) | 非必填，来源表：库位 | EAM库区管理（查看工装当前存放在哪个库位） | 模具当前存放的库位编码，模具不在产线使用时归还到此库位 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 模具的补充说明，如特殊管理要求、历史维修记录等 |

## 字段级业务影响

### status（状态）

| 状态 | 对EAM的影响 | 对MES的影响 | 注意事项 |
|------|------------|-------------|----------|
| **闲置** | 可正常借出，库存可查 | 模具在库区，可分配到工位使用 | |
| **在用** | 不可借出，显示借用人 | 工位正在使用此模具，工序可正常执行 | |
| **维修** | 不可借出，显示维修中 | 模具不可用，工位开工受阻，需更换模具 | 需尽快完成维修或联系备用模具 |
| **报废** | 不可借出，不可维修登记 | 模具从可用列表移除，工位需更换其他模具 | 报废前应确认已完成最后一批生产 |

## 关联关系

```
模具类型（1）←→（N）模具
模具 → 库位（当前存放位置）
模具 → 工装台账明细（使用寿命记录）
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 来源表 | moldType→模具类型表，location→库位表 |
| 联动影响 | 状态变更→借出归还记录自动更新；currentCount系统自动维护；寿命预警按当前次数/设计寿命计算 |
| 自增长 | currentCount由系统自动累计，不可手动修改 |
