# 生产线管理

## 概述

生产线是车间内的生产流水线定义，连接工序和工位，是 MES 工单展开和工序流转的载体。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| productionLineCode | 生产线编码 | VARCHAR(50) | 必填、唯一 | MES工单展开 | 系统主键 |
| productionLineName | 生产线名称 | VARCHAR(200) | 必填 | MES显示 | |
| workshopCode | 所属车间 | VARCHAR(50) | 必填，来源表：车间 | 组织归属 | |
| productionLineType | 生产线类型 | ENUM | 字典项：连续型/离散型/批量型 | MES工序流转模式 | |
| cycleTime | 标准产能节拍 | INT | 非必填（秒/件） | MES产能计算 | 单件产品标准工时 |
| shiftCount | 班次 | INT | 默认1 | MES排班 | 每日班次数 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES工单分配 | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 工单分配 | 工单按生产线展开工序 |
| MES | 工序流转 | 工序在生产线上按顺序流转 |
| MES | 产能计算 | 节拍时间×生产时间=产量 |
| MES | 报工 | 报工记录归属生产线 |

## 关联关系

```
车间（1）←→（N）生产线
生产线（1）←→（N）工位
生产线（1）←→（N）工序
生产线 → MES工单
```
