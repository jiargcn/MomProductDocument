# 生产线管理

## 概述

生产线是车间内的生产流水线定义，连接工序和工位，是 MES 工单展开和工序流转的载体。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| productionLineCode | 生产线编码 | VARCHAR(50) | 必填、唯一 | MES工单展开（工单需分配到生产线）、报工归属（报工记录归属生产线） | 生产线的唯一身份标识，用于标识具体产线，系统主键 |
| productionLineName | 生产线名称 | VARCHAR(200) | 必填 | MES显示（产线名称展示）、报表（按产线统计） | 生产线的显示名称，如"装配线A1" |
| workshopCode | 所属车间 | VARCHAR(50) | 必填，来源表：车间 | 组织归属（明确产线属于哪个车间）、报表层级（按车间-产线层级汇总） | 生产线所属车间的编码，定义产线的上级组织 |
| productionLineType | 生产线类型 | ENUM | 字典项：连续型/离散型/批量型 | MES工序流转模式（不同类型产线的工序流转逻辑不同）、报工方式（连续型按时间报工，离散型按工序报工） | 生产线的产品流动方式，连续型适用于流水线生产，离散型适用于单件生产，批量型适用于分批生产 |
| cycleTime | 标准产能节拍 | INT | 非必填（秒/件） | MES产能计算（节拍时间决定单位时间产出）、工单交期（工单完工时间=节拍×数量） | 生产一件产品需要的标准时间，单位秒/件，用于评估产能和计算工单完工时间 |
| shiftCount | 班次 | INT | 默认1 | MES排班（每日班次数决定排班循环）、产能计算（日产能=节拍×工作时长×班次） | 生产线每天工作的班次数，如每天3班（早中晚）则shiftCount=3 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES工单分配（禁用产线不可分配工单）、报工（禁用产线的报工记录仍可查） | 生产线的启用/禁用状态，禁用后不可分配新工单 |

## 字段级业务影响

### productionLineType（生产线类型）

| 类型 | 对MES工序流转的影响 | 对报工的影响 | 适用场景 |
|------|-------------------|-------------|----------|
| **连续型** | 工序间连续流转，无明显工单转移 | 按时间间隔自动报工，如每小时产出数量 | 食品、化工等流程型生产 |
| **离散型** | 工序间有明确跳转，按工序顺序执行 | 按工序完成后手动报工，才可流转到下一工序 | 机械装配、电子组装 |
| **批量型** | 按批次流转，一批物料在同一工序加工后才流转到下一工序 | 按批次报工，批次内全部完成后一起流转 | 热处理、喷涂等批量加工 |

### cycleTime（标准节拍）

| 场景 | 对MES的影响 |
|------|-------------|
| 工单排产 | 系统根据节拍时间自动计算工单完工时间 |
| 产能评估 | 当订单量超过产线产能时，系统提示交期不足 |
| 报工校验 | 报工数量超过节拍产能时系统警告（报工数量/节拍时间异常） |

## 关联关系

```
车间（1）←→（N）生产线
生产线（1）←→（N）工位
生产线（1）←→（N）工序配置
生产线 → MES工单
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | productionLineType（连续型/离散型/批量型）、status（启用/禁用） |
| 来源表 | workshopCode→车间表 |
| 联动影响 | cycleTime与shiftCount共同决定日产能，日产能不足时MES提示交期预警 |
