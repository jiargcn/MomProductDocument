# 工艺路线

## 概述

工艺路线定义产品制造的工序顺序和作业参数，是 MES 工单展开和工序流转的核心依据。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| routeCode | 工艺编码 | VARCHAR(50) | 必填、唯一 | MES工单展开 | 系统主键 |
| routeName | 工艺名称 | VARCHAR(200) | 必填 | MES显示 | |
| materialCode | 适用物料 | VARCHAR(50) | 必填，来源表：物料 | MES工单匹配 | |
| version | 版本 | VARCHAR(20) | 必填 | 工艺变更管理 | |
| effectiveDate | 生效日期 | DATE | 必填 | MES工单展开 | |
| invalidDate | 失效日期 | DATE | 非必填 | MES工单展开 | 空=永久有效 |
| status | 状态 | ENUM | 字典项：草稿/生效/禁用 | MES工单展开 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 工艺路线明细（RouteLine）

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| sequence | 工序序号 | INT | 必填 | 工序执行顺序 | |
| processCode | 工序编码 | VARCHAR(50) | 必填，来源表：工序 | MES工序执行 | |
| productionLineCode | 生产线 | VARCHAR(50) | 非必填，来源表：生产线 | MES派工 | |
| stationCode | 工位 | VARCHAR(50) | 非必填，来源表：工位 | MES报工点 | |
| standardTime | 标准工时 | DECIMAL(18,4) | 非必填 | 产能计算 | 覆盖工序默认工时 |
| isKeyProcess | 关键工序 | BOOLEAN | 默认否 | MES齐套、工序追溯 | 关键工序未完工不可流转 |
| isInspectProcess | 检验工序 | BOOLEAN | 默认否 | QMS触发 | 自动触发质检 |
| isTransferProcess | 流转工序 | BOOLEAN | 默认否 | WMS触发 | 自动触发物料移转 |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 工单展开 | 按工艺路线顺序生成工序任务 |
| MES | 报工 | 按工序逐级报工 |
| MES | 齐套检查 | 工序级物料需求计算 |
| QMS | 检验触发 | 检验工序自动触发质量检验 |
| WMS | 物料移转 | 流转工序自动触发库位间移转 |

## 关联关系

```
物料 → 工艺路线（1:N）
工艺路线 → 工艺路线明细（1:N）→ 工序
工艺路线明细.productionLine → 生产线
工艺路线明细.station → 工位
```
