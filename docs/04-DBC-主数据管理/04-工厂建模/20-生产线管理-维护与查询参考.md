# 生产线管理-维护与查询参考

> 对应业务说明：[生产线管理](07-生产线管理.md)。

## 新增与编辑

| 维护项 | 新增要求 | 编辑与变更建议 |
| --- | --- | --- |
| 编码、名称、类型 | 编码、名称必填；类型按产线实际用途选择。 | 改码或换类型前评估工单、工艺和物料关系。 |
| 所属车间 | 必填，车间必须预先维护。 | 不建议直接跨车间改挂；应按现场迁移计划处理。 |
| 原料/成品地点 | 作为关联资料处理。 | 当前主页面与作业的实际挂接待验证，不把它当作已生效的自动收发规则。 |
| 是否可用与有效期 | 可用状态必填。 | 停线前检查工位、工单、线边库存与设备/终端引用。 |

## 批量导入规则

模板应包含：编码、名称、类型、车间、是否可用、生效时间、失效时间、备注。

- 不应导入工位、物料关系、BOM、工单、实际产能或线边库存。
- 编码、名称、车间和可用状态为必填；车间须先存在。
- 导入后联查工位、生产线物料关系和工艺路线；重复、更新模式和错误回执需测试确认。

## 查询与详情建议

列表首屏展示编码、名称、类型、车间、状态、有效期；按车间、类型、状态筛选。详情分为“基本关系、地点与线边资料、关联工位/物料/工艺、状态与有效期、审计”，并联查工位、BOM、工单和线边库存。

## 常见问题

| 现象 | 处理建议 |
| --- | --- |
| 工单或物料无法匹配产线 | 同时核对产线状态、物料关系、工艺路线和实际任务条件。 |
| 需要停线或迁移 | 先处理下属工位、在途工单和线边库存，再停用或调整。 |
