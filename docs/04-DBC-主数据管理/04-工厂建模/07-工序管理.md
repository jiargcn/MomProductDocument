# 工序管理

## 概述

工序是生产制造过程中的最小作业单元，定义产品制造过程中的具体加工作为及其作业参数。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| processCode | 工序编码 | VARCHAR(50) | 必填、唯一 | MES工序流转 | 系统主键 |
| processName | 工序名称 | VARCHAR(200) | 必填 | MES显示 | |
| processType | 工序类型 | ENUM | 字典项：加工/装配/检测/包装/流转 | 作业类型分类 | |
| workContent | 作业内容 | VARCHAR(500) | 非必填 | 作业指导书内容 | |
| standardTime | 标准工时 | DECIMAL(18,4) | 非必填 | MES产能计算、成本核算 | 单位：小时/件 |
| qualifiedRate | 合格率 | DECIMAL(5,2) | 非必填，默认100% | MES质量预测 | |
| needDevice | 需设备 | BOOLEAN | 默认否 | MES派工、设备联动 | |
| needMaterial | 需物料 | BOOLEAN | 默认是 | MES齐套、BOM展开 | |
| qualityControlPoint | 质量控制点 | BOOLEAN | 默认否 | QMS生产检验触发 | 设为控制点则自动触发检验 |
| nextProcessCode | 下一工序 | VARCHAR(50) | 非必填，来源表：工序 | MES工序流转方向 | |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 工单展开 | 按工艺路线展开工序顺序 |
| MES | 报工 | 报工记录归属工序 |
| MES | 齐套检查 | 工序级物料齐套判断 |
| QMS | 生产检验 | 质量控制点触发检验申请 |
| 成本 | 标准成本 | 标准工时×人工费率=工序成本 |

## 关联关系

```
工序 → 工艺路线（顺序关系）
工序 → MES报工记录
工序 → 设备（需设备时绑定）
工序 ← BOM子项（工位级用料）
工序 → QMS检验记录
```
