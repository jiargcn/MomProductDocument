# 库区管理

## 概述

库区是仓库内的逻辑分区，同一仓库内按功能或物料类型划分管理区域。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| areaCode | 库区编码 | VARCHAR(50) | 必填、唯一 | 入库分配、库位查询 | 系统主键 |
| areaName | 库区名称 | VARCHAR(200) | 必填 | 库区显示 | |
| warehouseCode | 所属仓库 | VARCHAR(50) | 必填，来源表：仓库 | 库位归属组织 | |
| areaType | 库区类型 | ENUM | 字典项：存储区/暂存区/质检区/隔离区/发运区 | 库内作业调度 | 不同类型支持不同作业 |
| materialType | 物料类型 | VARCHAR(50) | 非必填，字典项 | 入库分配 | 指定此库区存放的物料类型 |
| temperatureZone | 温区 | ENUM | 字典项：常温/冷藏/冷冻/恒温 | 物料存放条件 | 医药/食品行业必填 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 入库分配 | 物料入库时按物料类型和库区配置自动分配 |
| WMS | 质检流程 | 质检区库位用于待检品暂存 |
| WMS | 隔离管理 | 隔离区用于不合格品隔离 |
| WMS | 发运备料 | 发运区用于发货前集货 |

## 关联关系

```
仓库（1）←→（N）库区
库区（1）←→（N）库位
库区 → 入库分配规则
```
