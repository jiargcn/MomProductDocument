# 库区管理

## 概述

库区是仓库内的逻辑分区，同一仓库内按功能或物料类型划分管理区域。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| areaCode | 库区编码 | VARCHAR(50) | 必填、唯一 | 入库分配（系统根据库区配置自动推荐）、库位查询（按库区筛选） | 库区的唯一身份标识，用于标识和引用具体库区，系统主键 |
| areaName | 库区名称 | VARCHAR(200) | 必填 | 库区显示（各单据和列表中展示） | 库区的显示名称，如"原材料A区-待检区" |
| warehouseCode | 所属仓库 | VARCHAR(50) | 必填，来源表：仓库 | 库位归属（库位归属于哪个仓库）、报表统计（按仓库维度汇总库区数据） | 库区所属的仓库编码，定义库区的上级组织关系 |
| areaType | 库区类型 | ENUM | 字典项：存储区/暂存区/质检区/隔离区/发运区 | 入库分配（物料进入哪种类型的库区）、库内作业调度（不同类型支持不同作业） | 库区的功能类型，存储区用于正常存放，暂存区用于临时堆放，质检区用于来料待检，隔离区用于不合格品隔离，发运区用于发货前集货 |
| materialType | 物料类型 | VARCHAR(50) | 非必填，字典项 | 入库分配（按物料类型推荐库区） | 指定此库区优先存放的物料类型，入库时系统可根据物料类型自动推荐到对应库区，如填写"原料"则原材料入库时优先推荐此库区 |
| temperatureZone | 温区 | ENUM | 字典项：常温/冷藏/冷冻/恒温 | 物料存放条件校验（物料的温区要求与库区匹配） | 库区的温度环境要求，部分物料（食品、医药）有温区要求，不匹配时系统警告但允许强制入库 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 库区的补充说明，如特殊管理要求、存放注意事项等 |

## 字段级业务影响

### areaType（库区类型）

| 类型 | 对入库的影响 | 对出库的影响 | 典型应用 |
|------|-------------|-------------|----------|
| **存储区** | 物料检验后进入此区，正常存放 | 出库时从此区拣货 | 常规物料存放 |
| **暂存区** | 物料到货后临时存放，等待分配到正式库区 | 出库时直接从暂存区拣货，适用于紧急用料 | 卸货后待分配 |
| **质检区** | 来料检验时物料进入此区，检验完成前不可移出 | 检验完成前不可出库，合格品转到存储区，不合格品留隔离区 | 来料待检品 |
| **隔离区** | 不合格品或待处理品进入此区，隔离存放 | 隔离区物料需处理后才能出库 | 不合格品隔离 |
| **发运区** | 成品发货前集货到此区，等待装车 | 发货时从发运区装车出库 | 发货集货 |

### temperatureZone（温区）

| 温区 | 适用场景 |
|------|---------|
| **常温** | 一般制造业物料，15-30℃ |
| **冷藏** | 食品、药品，2-8℃ |
| **冷冻** | 速冻食品，-18℃以下 |
| **恒温** | 精密零部件，20-25℃ |

## 关联关系

```
仓库（1）←→（N）库区
库区（1）←→（N）库位
库区 → 入库分配规则
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | areaType（存储区/暂存区/质检区/隔离区/发运区）、temperatureZone（常温/冷藏/冷冻/恒温） |
| 来源表 | warehouseCode→仓库表 |
| 联动影响 | 库区类型决定物料在检验流程中的状态变化；温区不匹配时系统警告但不阻止入库 |
