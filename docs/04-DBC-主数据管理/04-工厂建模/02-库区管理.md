## 概述

库区是仓库内的逻辑分区，同一仓库内按功能或物料类型划分管理区域。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| areaCode | 代码 | VARCHAR(50) | 必填、唯一 | 入库分配（系统根据库区配置自动推荐）、库位查询（按库区筛选） | 库区的唯一身份标识，系统主键 |
| areaName | 名称 | VARCHAR(200) | 必填 | 库区显示（各单据和列表中展示） | 库区的显示名称，如"原材料A区-待检区" |
| warehouseCode | 仓库代码 | VARCHAR(50) | 必填 | 库位归属（库位归属于哪个仓库）、[报表统计](../../06-MES-生产管理/05-报表统计/index.md)（按仓库维度汇总库区数据） | 库区所属的仓库编码 |
| areaType | 类型 | VARCHAR(50) | 非必填 | 入库分配（按类型推荐库区） | 库区的功能类型 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 库位分配（不可用时不出现在推荐列表） | 库区的可用状态 |
| deptCode | 所属部门代码 | VARCHAR(50) | 非必填 | 部门归属（按部门统计库区） | 所属部门编码 |
| deptName | 所属部门名称 | VARCHAR(100) | 非必填 | 部门归属（显示部门名称） | 所属部门名称 |
| effectiveDate | 生效时间 | DATE | 非必填 | 库区启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 库区停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 库区的补充说明 |

## 关联关系

```
仓库（1）←→（N）库区
库区（1）←→（N）库位
库区 → 入库分配规则
```
