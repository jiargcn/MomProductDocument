# 车间管理

## 概述

车间是生产制造的物理组织单元，定义生产工单执行的具体场所。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| workshopCode | 车间编码 | VARCHAR(50) | 必填、唯一 | MES工单派工（工单需指定执行车间）、报工归属（报工记录归属车间） | 车间的唯一身份标识，用于标识和引用具体车间，系统主键 |
| workshopName | 车间名称 | VARCHAR(200) | 必填 | MES报工显示（展示车间名称）、报表（按车间统计产量） | 车间的显示名称，如"沈阳工厂-装配车间A" |
| workshopType | 车间类型 | ENUM | 字典项：装配/加工/检测/包装 | 生产类型分类（按类型汇总不同产线）、MES工序分配（不同类型车间分配不同工序） | 车间的主要生产类型，决定车间内可执行的工序类型和配置的报工方式 |
| manager | 车间主任 | VARCHAR(50) | 非必填，来源表：用户 | 职责归属（车间异常问题找对应负责人）、审批流（车间级审批找车间主任） | 车间负责人账号，用于车间级异常问题联络和审批路由 |
| parentWorkshop | 上级车间 | VARCHAR(50) | 非必填，来源表：车间 | 车间层级（支持多级组织结构） | 上级车间的编码，用于构建车间层级关系，如"装配车间A"的上级可以是"沈阳工厂" |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES工单派工（禁用车间不可派工）、报工（禁用车间的报工记录仍可查） | 车间启用/禁用状态，禁用后不可新建派工到此车间，但历史报工数据保留 |

## 关联关系

```
车间（1）←→（N）生产线
车间（1）←→（N）工位
车间（1）←→（N）设备
车间 → MES工单
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | workshopType（装配/加工/检测/包装）、status（启用/禁用） |
| 来源表 | manager→用户表，parentWorkshop→车间表（自关联） |
