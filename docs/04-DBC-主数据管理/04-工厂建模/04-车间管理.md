# 车间管理

## 概述

车间是生产制造的物理组织单元，定义生产工单执行的具体场所。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| workshopCode | 车间编码 | VARCHAR(50) | 必填、唯一 | MES工单派工 | 系统主键 |
| workshopName | 车间名称 | VARCHAR(200) | 必填 | MES报工显示 | |
| workshopType | 车间类型 | ENUM | 字典项：装配/加工/检测/包装 | 生产类型分类 | |
| manager | 车间主任 | VARCHAR(50) | 非必填，来源表：用户 | 职责归属 | |
| parentWorkshop | 上级车间 | VARCHAR(50) | 非必填，来源表：车间 | 车间层级 | 支持多级组织 |
| status | 状态 | ENUM | 字典项：启用/禁用 | MES工单派工 | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| MES | 工单派工 | 工单需指定执行车间 |
| MES | 报工 | 报工记录归属车间 |
| MES | 工序流转 | 工序按车间组织流转 |
| EAM | 设备归属 | 设备台账归属到车间 |

## 关联关系

```
车间（1）←→（N）生产线
车间（1）←→（N）工位
车间（1）←→（N）设备
生产线 → MES工单
```
