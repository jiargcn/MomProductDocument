# 库位组管理

## 概述

库位组管理定义库位的分组信息，用于库位分类管理。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| groupCode | 库位组代码 | VARCHAR(50) | 必填 | 库位归属（库位归属于哪个组） | 库位组的唯一标识 |
| groupName | 库位组名称 | VARCHAR(200) | 必填 | 库位组显示 | 库位组的显示名称 |
| warehouseCode | 仓库代码 | VARCHAR(50) | 非必填 | 库位组归属 | 所属仓库 |
| isEnabled | 是否可用 | BOOLEAN | 默认是 | 库位分配（不可用时不出现在推荐列表） | 是否可用 |
| effectiveDate | 生效时间 | DATE | 非必填 | 库位组启用时间 | 生效时间 |
| invalidDate | 失效时间 | DATE | 非必填 | 库位组停用时间 | 失效时间 |
| remark | 备注 | VARCHAR(500) | 非必填 | | 补充说明 |

## 关联关系

```
仓库（1）←→（N）库位组
库位组（1）←→（N）库位
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
