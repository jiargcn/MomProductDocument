# 库位组管理

> 当前基线：`dev`（测试环境目标基线），源码与 DDL 取证日期：2026-07-16。
> 页面定位：维护同一库区内库位的逻辑分组。库位组是库位新增和导入的必填归属，供上架、库存查询和终端选择器形成空间层级；不适用申请、任务与记录模型。

## 当前实现摘要

持久化表为 `basic_locationgroup`。该页面使用专用 Controller 与 `LocationgroupServiceImpl`，新增、编辑、删除和导入均进入库位组专用服务。业务重复检查以“代码 + 库区代码”为组合，仓库与库区关系会被服务验证；但 DDL 未见相应复合唯一约束，详见产品差距总账 GAP-051。

## 字段与新增约束

| 技术字段 | 中文名 | DDL 类型/约束 | Web 规则 | 服务规则 |
| --- | --- | --- | --- | --- |
| `code` | 代码 | `varchar(64)`，非空 | 必填、最多 50 字符、`validateYS` 校验。 | 新增按代码+库区检查重复。 |
| `name` | 名称 | `varchar(64)`，可空 | 最多 50 字符。 | 导入要求；普通新增未见同等必填注解。 |
| `warehouseCode` | 仓库代码 | `varchar(64)`，非空 | 必填；仓库分页选择器，筛选 `available=true`。 | 校验仓库存在。 |
| `areaCode` | 库区代码 | `varchar(64)`，非空 | 必填；库区选择器依赖仓库代码，筛选同仓库且 `available=true`。 | 校验库区存在且属于所选仓库。 |
| `available` | 是否可用 | `boolean`，非空，默认 `true` | 开关控件，默认 `true`，必填。 | 校验非空。 |
| `activeTime` / `expireTime` | 生效/失效时间 | `timestamp(6)`，可空 | 日期时间选择器。 | 未见时间区间校验。 |
| `remark` | 备注 | `text`，可空 | 最多 50 字符。 | 未见额外校验。 |

`description`、审计、逻辑删除、扩展属性、并发戳、站点和租户等为 DDL/系统字段，不作为人工新增或导入字段。

## 编辑、启停与删除

| 动作 | 当前可证实行为 | 使用口径 |
| --- | --- | --- |
| 编辑 | Web 锁定 `code`、`warehouseCode`；`areaCode` 仍可选择。服务先按代码和原/新库区校验冲突，再整体更新。 | 不修改代码和仓库；若改库区，必须确认新库区不存在同代码库位组，并评估已有库位引用。 |
| 启用/停用 | 列表提供启停动作；库位选择器会按 `available=true` 过滤库位组。 | 停用前须核查其下库位及 WMS 在途业务；未见统一引用拦截。 |
| 删除 | 服务仅校验库位组存在后删除。 | 已有关联库位的组不应视为可安全删除，需先做引用核查。 |

前端按钮权限标识为 `dbc:locationgroup:create/import/export/update/enable/disable/delete`。当前专用 Controller 的导入 `@PreAuthorize` 注解被注释；按钮可见性不等于后端授权结论。

## 导入规则

导入类 `LocationGroupImportExcelVO` 的字段顺序为：代码、名称、仓库代码、区域代码、是否可用、生效时间、失效时间、备注。

| 校验/模式 | 当前规则 |
| --- | --- |
| 必填 | 代码、名称、仓库代码、区域代码、是否可用。 |
| 关联 | 校验仓库存在；校验库区存在且属于该仓库。 |
| 业务键 | 导入按 `code + areaCode` 查重；同一库区同代码为重复数据。 |
| 新增/覆盖 | 不存在且非覆盖模式时插入；存在且非仅新增模式时更新；仅新增遇已存在记录生成错误。 |
| 错误处理 | 有错误且未选择部分更新时整批回滚。 |

模板不包含 `description`、审计字段、删除信息、扩展属性、并发戳、站点和租户等系统字段。

## 列表、详情与快速跳转

当前默认列顺序为：代码、名称、仓库代码、库区代码、是否可用、生效时间、失效时间、备注、创建时间、创建者、最后更新时间、最后更新者、操作。默认筛选为代码、名称、库区代码、是否可用（默认“可用”）。

建议主列表优先显示：代码、名称、仓库、库区、是否可用；有效期、备注及审计列作为补充。

当前使用通用详情组件，未见专用关联 Tab。后续建议：

| 分组 | 内容 | 建议跳转 |
| --- | --- | --- |
| 基本与空间归属 | 代码、名称、仓库、库区、是否可用 | 仓库、库区 |
| 有效期与审计 | 生效/失效时间、备注、审计字段 | 操作日志（待公共能力取证） |
| 下游引用 | 组内库位、库存余额、在途任务 | 库位列表、WMS 库存查询并预填条件（待实现/验证） |

## 关联关系

```
仓库 → 库区 → 库位组（basic_locationgroup）→ 库位
                                             └→ WMS 空间选择与库存查询
```
