# 客户物料

> 当前基线：`dev`（测试环境目标基线），源码与 DDL 取证日期：2026-07-16。
> 页面定位：维护客户与企业物料之间的对照、客户计量与客户包装配置，为销售、出库和供应链协同提供可用的客户物料关系。本页不维护客户报价、价格或币种；这些不在当前 `basic_customeritem` 模型中。

## 当前实现摘要

持久化表为 `basic_customeritem`。真实字段为客户代码、物料代码、客户物料代码、客户计量单位、转换率、客户包装规格/标包数量、替代包装规格/数量、可用状态、有效期和备注。当前 Web 主表单开放客户、物料、客户物料代码、包装规格、标包数量、客户计量单位、有效期和备注；转换率和替代包装字段已注释，状态字段不在表单中。

服务按客户+物料查询重复，但 DDL 未提供复合唯一约束；Web 编辑锁定客户和物料，服务更新仍接收完整对象。普通保存对客户和物料做存在性查询，包装规格只在普通保存中检查非空；导入反而校验包装规格主数据类型。详见产品差距总账 GAP-059。

## 字段与新增约束

| 技术字段 | 中文名 | DDL 类型/约束 | 当前页面规则 | 当前服务证据 |
| --- | --- | --- | --- | --- |
| `customerCode` | 客户代码 | `varchar(64)`，非空 | 必填；客户选择器，数据源为客户分页接口，筛选 `available=true`；编辑锁定。 | 查询客户是否存在。 |
| `customerName` | 客户名称 | 非表字段/显示字段 | 当前页面未维护。 | 需由客户主数据补全，映射一致性待验证。 |
| `itemCode` | 物料代码 | `varchar(64)`，非空 | 必填；物料选择器，筛选 `available=true`；编辑锁定。 | 查询物料是否存在。 |
| `customerItemCode` | 客户物料代码 | `varchar(64)`，可空 | 最多 50 字符、编码格式校验。 | 未见客户物料代码唯一性校验。 |
| `customerUom` | 客户计量单位 | `varchar(64)`，可空 | UOM 字典字段；当前未见前端必填规则。 | VO 标记非空；导入必填并做字典转换。 |
| `convertRate` | 转换率 | `numeric(18,6)`，可空 | 当前表单已注释。 | VO/DDL 存在，未见普通保存数值范围校验。 |
| `packageUnit` | 客户包装规格 | `varchar(64)`，可空 | 必填；包装规格选择器，筛选 `available=true`。 | 普通保存仅检查非空；未见当前包装规格是否存在/类型校验。 |
| `packageStdQty` | 客户标包数量 | `numeric(18,6)`，可空 | 必填，数字输入最小 1、精度 6。 | 仅检查非空。 |
| `altPackageUnit` / `altPackageStdQty` | 客户替代包装规格/数量 | `varchar(64)` / `numeric(18,6)`，可空 | 当前表单已注释。 | 未见当前校验。 |
| `available` | 是否可用 | `boolean`，非空，默认 `true` | 当前表单未显示；列表默认查可用记录并提供启停。 | 新建/导入均强制写为 `true`。 |
| `activeTime` / `expireTime` | 生效/失效时间 | `timestamp(6)`，可空 | 日期时间选择器；前端要求失效时间晚于生效时间。 | 未见同等服务端区间校验。 |
| `remark` | 备注 | `text`，可空 | 最多 50 字符。 | 无额外校验。 |

## 编辑、启停与删除

| 动作 | 当前可证实行为 | 使用口径 |
| --- | --- | --- |
| 编辑 | Web 锁定客户和物料；服务更新接收完整对象，未见同等锁定或业务键重校验。 | 不应把 UI 锁定等同于接口不可改；改动在用客户/物料关系前应核查销售、出库及 SCP 引用。 |
| 启用/停用 | 前端提供启停，业务查询 `selectCustomeritemExist` 仅返回可用关系。 | 停用将影响后续客户物料选择；存量单据影响待测试。 |
| 删除 | 删除引用保护尚未完成取证。 | 已被订单、出库、对账或接口引用的关系不应视为可安全删除。 |

前端权限标识为 `dbc:customeritem:*`；新增按钮却使用拼写不同的 `dbc:customeitem:create`。接口层鉴权需 RBAC 实测，不以按钮配置下结论。

## 导入规则

`CustomeritemImportExcelVO` 当前模板字段包括：客户代码、物料代码、客户物料代码、客户包装规格、客户标包数量、客户计量单位、生效时间、失效时间、备注；替代包装和可用状态字段已注释。

| 校验/模式 | 当前规则 |
| --- | --- |
| 必填 | 客户代码、物料代码、导入类中的 `packUnit`、`packQty`、客户计量单位。 |
| 关联字段 | 校验客户、物料存在；导入独有地校验包装规格存在且类型为 `CUSTOMER_PACK`。 |
| 既有记录 | 按客户代码+物料代码查找。 |
| 状态 | 新增与导入均强制可用；模板不允许录入状态。 |
| 错误处理 | 由通用导入框架处理；整批回滚与模式 1/2/3 的实际行为待测试。 |

导入 VO 使用 `packUnit`、`packQty`，而持久化/普通页面使用 `packageUnit`、`packageStdQty`；必须验证转换映射是否完整，不能假定导入模板已能写入当前表单配置。

## 列表、详情与快速跳转

当前默认列顺序为：客户代码、物料代码、客户物料代码、客户包装规格、客户标包数量、客户计量单位、是否可用、生效时间、失效时间、备注、审计列、操作。默认筛选为客户代码、物料代码、客户物料代码，列表初始化为可用关系。

建议主列表优先显示：客户、物料、客户物料代码、客户包装规格、标包数量、计量单位、可用状态；有效期与审计列作为补充。

当前采用通用详情组件，未见专用关联页签。后续建议：

| 分组 | 内容 | 建议跳转 |
| --- | --- | --- |
| 客户与物料身份 | 客户代码/名称、物料代码/名称、客户物料代码、可用状态 | 客户、物料详情 |
