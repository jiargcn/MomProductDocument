# 客户

## 概述

客户主数据记录与企业有销售业务往来的采购方信息，是销售订单、出货管理、客户对账的基础。

## 字段说明

| 字段名 | 中文名 | 类型 | 约束 | 影响业务 | 备注 |
|--------|--------|------|------|----------|------|
| customerCode | 客户编码 | VARCHAR(50) | 必填、唯一 | 销售订单、出货 | 系统主键 |
| customerName | 客户名称 | VARCHAR(200) | 必填 | 销售订单、报表 | |
| customerType | 客户类型 | ENUM | 字典项：终端客户/贸易商/配套商 | 销售分类统计 | |
| contactPerson | 联系人 | VARCHAR(100) | 非必填 | 出货通知、异常联络 | |
| contactPhone | 联系电话 | VARCHAR(50) | 非必填 | 出货通知 | |
| contactEmail | 邮箱 | VARCHAR(100) | 非必填 | 邮件通知、对账 | |
| address | 地址 | VARCHAR(500) | 非必填 | 出货地址、质检寄样 | |
| bankName | 开户银行 | VARCHAR(100) | 非必填 | 收款结算 | |
| bankAccount | 银行账号 | VARCHAR(50) | 非必填 | 收款结算 | |
| taxNo | 税号 | VARCHAR(50) | 非必填 | 开票、税务申报 | |
| paymentTerm | 付款账期 | VARCHAR(50) | 字典项：月结30/预付/到付 | 应收账款管理 | |
| creditLimit | 信用额度 | DECIMAL(18,2) | 非必填 | 订单审批超额度拦截 | |
| status | 状态 | ENUM | 字典项：启用/禁用 | 销售下单 | 禁用后不可新建订单 |
| remark | 备注 | VARCHAR(500) | 非必填 | | |

## 业务影响

| 业务模块 | 影响场景 | 说明 |
|----------|----------|------|
| WMS | 成品发货 | 发货地址、客户联系人显示 |
| WMS | 客户退货 | 退货接收依据 |
| QMS | 客户退回检验 | 退货检验批次的来源 |
| 销售 | 销售订单 | 客户编码/名称/账期用于订单 |
| 财务 | 应收账款 | 信用额度、付款账期用于账款管理 |

## 关联关系

```
客户（1）←→（N）客户物料 → 物料
客户（1）←→（N）客户月台 → 月台
客户 ←→ 销售订单 ←→ 成品发货
客户 ←→ 客户退货 ←→ 客户退回检验
```

## 字段约束说明

| 约束类型 | 说明 |
|----------|------|
| 字典项 | 客户类型（终端客户/贸易商/配套商）、付款账期、状态 |
| 联动影响 | 信用额度→销售订单金额超限时触发审批；禁用状态→销售订单中该客户不可选 |
