# MOM 制造运营管理系统

## 产品简介

MOM（Manufacturing Operations Management）制造运营管理系统，是一套面向离散制造业的全流程数字化管理平台，涵盖从主数据、库房、生产、质量、设备、异常、供应链到排程的核心业务闭环。

## 目标受众

本文档面向以下角色的内部同事：

- **开发人员**：理解模块设计、接口规范与扩展点
- **测试人员**：掌握功能边界、业务流程与验证场景
- **实施人员**：了解部署架构、配置项与上线流程
- **运维人员**：熟悉监控指标、日志体系与故障排查
