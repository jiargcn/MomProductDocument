# 质量评审

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施、运维（主）；质量评审岗、MRB、仓储协同等现场角色（顺带）；操作见[质量评审-维护与查询参考](质量评审-维护与查询参考.md)。

## 业务目的与适用范围

质量评审处理检验不合格后的处置决策：让步、报废、返修，以及客诉/索赔类质量通知。读完本页，应能立刻判断三件事：

1. **走评审还是只出通知单**：检验评审 ATR 给出处置结论；Q1/Q2/Q3 是通知/索赔单据，**不替代**评审结论。
2. **看错菜单**：QMS「检验评审」≠ WMS「质量评审 / 待评审物料」（后者是仓储待处置库存视角）。
3. **结论后库存差哪一步**：评审可回写检验记录使用决策；隔离/放行/报废出库等**事务仍以 WMS 为准**——让步 ≠ 已自动放行。

上游来源见[来料检验](../02-来料检验/index.md)、[生产检验](../03-生产检验/index.md)、[客户检验](../04-客户检验/index.md)。方案配置见[检验配置](../01-检验配置/index.md)。

菜单包含：

1. **检验评审**申请 / 任务 / 记录：对检验记录给出让步、报废或返修等结论，并可回写检验记录使用决策。
2. **质量通知**：q1 / q2 / q3 通知单（客诉/供应商索赔/问题跟踪等业务单据，与评审 ATR 并存）。

## 一笔质量评审如何完成

```mermaid
flowchart LR
    A["检验记录不合格\n或其他质量入口"] --> B["检验评审申请"]
    B --> C["检验评审任务"]
    C --> D["评审结论\n让步 / 报废 / 返修"]
    D --> E["检验评审记录"]
    D --> F["回写检验记录使用决策"]
    F --> G["WMS 库存处置\n（隔离/报废/放行等）"]
    A --> H["可选 Q1/Q2/Q3 通知单"]
```

| 对象 | 业务含义 |
| --- | --- |
| 评审申请 | 提出对某物料/包装/检验记录的评审请求。 |
| 评审任务 | 评审执行；可含结论、评审人、时间、流程节点。 |
| 评审记录 | 固化评审结果。 |
| 明细 | 包装/批次/数量/库存状态/库位/仓库等定位信息。 |
| Q1/Q2/Q3 | 通知与索赔类单据；Q2 可含供应商、采购收货号、索赔金额等。 |

评审结论：让步、报废、返修。任务完成可将库存状态映射为检验记录使用决策（合格 / 报废 / 隔离）。

!!! example "写实示例：给定 → 期望"

    **路径 A · 来料不合格进评审**

    - **给定：** 来料检验记录已发布，评估码拒绝或部分不合格；组织要求 MRB。
    - **期望：** 可建检验评审申请并关联原记录；任务结论为让步/报废/返修之一；完成后检验记录使用决策与映射预期一致。

    **路径 B · 让步后库存**

    - **给定：** 评审结论=让步；使用决策已回写为合格或组织约定值。
    - **期望：** QMS 侧结论已落；**库存放行仍须在 WMS 完成**——不以「评审完成」验收余额已变。

    **路径 C · 只建通知单**

    - **给定：** 现场只建了 Q1/Q2，未走检验评审。
    - **期望：** 通知/索赔事实可查；若组织要求 MRB 结论，仍缺评审 ATR——勿把 Q 单当成已闭环处置。

    **路径 D · 看错菜单**

    - **给定：** 在 WMS「质量评审/待评审物料」找检验评审申请。
    - **期望：** 找不到属正常；该入口是仓储待处置视角。回到本页检验评审，或用检验记录号反查。

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 走评审还是只出通知单 | 组织 MRB 流程 | 避免只建 Q 单无结论 |
| 让步后能否使用 | 使用决策与 WMS 状态 | 让步≠已自动放行 |
| 报废 | 是否已有报废出库 | 防重复或漏处置 |
| 看错菜单 | QMS 检验评审 vs WMS 质量评审 | 对象不同 |

### 关键字段业务角色

完整操作与字段见[维护与查询参考](质量评审-维护与查询参考.md)。本表只列主线关键项。

| 字段/配置点 | 在系统中的作用 | 关键行为要点 | 警惕什么 |

### 建议验证点

- 从来料或生产不合格记录可建立检验评审申请并完成任务；结论为让步/报废/返修之一。
- 任务完成后检验记录使用决策与映射预期一致（合格/报废/隔离）。
- 让步/报废样例在 WMS 侧核对是否已做放行/报废；不以 QMS 评审完成代替库存结果。
- Q1/Q2/Q3 可独立建单查询；与评审关系按组织说明，不强行编造自动链路。
- 培训区分 QMS 检验评审与 WMS 质量评审菜单；用检验记录号可反查评审单。
