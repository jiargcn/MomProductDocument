> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。

## 字段说明

下表服务排障与维护，不替代主文档主线。完整行为细则以测试环境为准。

| 字段 | 它决定什么 | 谁在何时维护 | 约束摘要 | 配错/漏配时常见现象 |
| --- | --- | --- | --- | --- |
| 待检触发条件 | 能否自动建申请 | 到货确认写入库存状态 | 数量>0 且待检；方案非免检 | 非待检或不满足数量 → 无申请 |
| 供应商 / 物料 / ASN | 追溯与回写键 | 多由上游带入，偶有手工 | 须与到货行一致 | 键错 → 回写失败或对不上账 |
| 方案 / 严格度 | 抽什么、多严 | 建单或任务带出/调整 | 来料类型方案；有效期内 | 无方案无法执行；免检则可能不建单 |
| 评估码 | 接收还是拒绝 | 记录发布前确认 | 二选一结论 | 拒收未闭环 → 下游处置悬空 |
| 使用决策 | 合格/不合格/报废/隔离**意图** | 记录 | 意图表达；**库存事务归 WMS** | 误把决策当成库存已改 |
| 数量拆分 | 回写包装合格/不合格/破坏/冻结 | 记录 | 与判定一致；有 ASN 且类型匹配才回写 | 合计对不齐 → 到货侧对账失败 |
| 申请/任务/记录状态 | 动作门禁 | 系统随动作迁移 | 申请九态 / 任务四态（见主文档） | 非预期状态强发/强删 |

### 状态与动作摘要

| 对象 | 允许动作（入口存在） | 前置检查 | 成功后影响 | 禁止或风险 |
| --- | --- | --- | --- | --- |
| 申请 | 提交/同意/驳回/处理/关闭 | 来源与方案 | 生成任务或结束 | 免检场景勿强建 |
| 任务 | 承接、执行、完成 | 已分配 | 形成记录 | — |
| 记录 | 录入、发布 | 结果完整 | 条件满足时回写到货 | 未发布勿当生效；勿手工改库存顶回写 |

## 页面与业务对象

| 菜单入口 | 用途 |
| --- | --- |
| 来料检验申请 | 建立/处理检验需求。 |

## 触发与联查

| 场景 | 建议操作 |
| --- | --- |
| 期望自动建申请 | 在 WMS 完成到货确认；明细库存状态为待检；方案非免检。 |
| 采购收货后无申请 | **正常**：收货记录已不再创建检验；回到到货确认链路核对。 |
| 核对方案 | QMS 检验方案（物料 + 采购收货检验类型）；免检则预期不建单。 |
| 核对回写 | 记录已发布 + 有 ASN；到货侧检验结果；库存后续仍查 WMS。 |

## 申请

| 应核对 | 提醒 |
| --- | --- |
| 供应商、物料、申请检验数量、包装 | 与到货一致。 |
| ASN、参考订单/行、回写请求号 | 追溯与幂等键。 |
| 方案、阶段、严格度、水平、AQL | 通常由方案带出，慎改。 |
| 自动提交/同意/执行 | 以配置为准。 |

### 常用动作

新增/修改/删除、提交、同意、驳回、处理、关闭。已生成任务后删除需谨慎。

## 任务与记录

| 应核对 | 提醒 |
| --- | --- |
| 任务状态、承接人、计划/完成时间 | 待处理→进行中→完成。 |
| 明细步骤：特性、方法、样本量、接收/拒收数 | 按模板执行。 |
| 包装：抽样/合格/不合格/破坏/冻结数量 | 发布回写用；合计须与判定一致。 |
| 定量值 / 定性选项与缺陷等级 | 选定集来源见配置。 |
| 评估码、使用决策 | 接收/拒绝；合格/不合格/报废/隔离（意图，非库存事务）。 |

### 常用动作

任务承接与完成；记录录入、保存、发布。发布后关注上游回写是否成功；库存隔离/放行到 WMS 核对。

## 发布与回写

| 步骤 | 关注点 |
| --- | --- |
| 发布前 | 包装数量合计与判定一致；ASN/类型是否具备回写条件。 |
| 发布后 | 到货检验结果是否收到；**库存后续动作在 WMS**。 |
| 失败 | 查回写异常与回调号，勿重复手工改库存顶替回写。 |

## 验收时建议核对

与主文档「建议验证点」同口径，便于现场勾选：

- [ ] 待检到货确认后出现来料检验申请（非免检物料）。
- [ ] 免检物料不建或跳过（与方案一致）。
- [ ] 申请处理生成任务；承接后可录入定量/定性。
- [ ] 发布后包装数量写回到货侧；QMS 记录可追溯 ASN/订单行。
- [ ] 采购收货完成**不再**新增检验申请（兼容入口仅日志/跳过）。
- [ ] 不合格样例可联查是否进入评审（按组织流程）。
- [ ] 使用决策已选但记录未发布时，不以库存已变为前提；库存事务归 WMS。
