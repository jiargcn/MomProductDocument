# 来料检验-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[来料检验](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 理解触发/判定/回写语义 | “字段说明” |
| 查为什么没自动建单 | “触发与联查” |

| 现场做检验 | “任务与记录” |
| 查结论是否回写仓库 | “发布与回写” |

## 字段说明
### 字段说明总表
| 字段 | 业务作用 | 谁在何时维护/选择 | 业务约束摘要 | 影响哪些功能 | 变更或错选风险 |
| --- | --- | --- | --- | --- | --- |
| 待检触发条件 | 自动建申请门槛 | 到货确认 | 数量>0 且待检；非免检 | 申请生成 | 非待检不建单 |
| 供应商/物料/ASN | 追溯键 | 上游带入/手工 | 与到货一致 | 回写、跟踪 | 键错 |
| 方案/严格度 | 执行依据 | 建单或任务 | 来料类型方案 | 抽样与判定 | 无方案 |
| 评估码 | 接收/拒绝 | 记录发布前 | 二选一结论 | 下游处置 | 拒收未闭环 |
| 使用决策 | 合格/不合格/报废/隔离 | 记录 | 意图；库存归 WMS | 放行/隔离预期 | 误当已改库存 |
| 数量拆分 | 回写包装数量 | 记录 | 合格/不合格/破坏/冻结 | 到货回写 | 对不齐 |
| ATR 状态 | 动作门禁 | 系统迁移 | 申请九态/任务四态 | 承接、发布 | 非法动作 |

### 状态-动作摘要（P9）

| 对象 | 允许动作（入口存在） | 前置检查 | 成功后影响 | 禁止或风险 |
| --- | --- | --- | --- | --- |
| 申请 | 提交/同意/驳回/处理/关闭 | 来源与方案 | 生成任务或结束 | 免检场景勿强建 |
| 任务 | 承接、执行、完成 | 已分配 | 形成记录 | — |
| 记录 | 录入、发布 | 结果完整 | 回写到货（条件满足时） | 未发布勿当生效 |

## 触发与联查

| 场景 | 建议操作 |
| --- | --- |
| 期望自动建申请 | 在 WMS 完成到货确认；明细库存状态为待检；方案非免检。 |
| 采购收货后无申请 | **正常**：收货记录已不再创建检验；回到到货确认链路。 |
| 核对方案 | QMS 检验方案（物料 + 采购收货检验类型）。 |
| 核对回写 | 记录已发布 + 有 ASN；到货侧检验结果。 |

## 申请

| 应核对 | 提醒 |
| --- | --- |
| 供应商、物料、申请检验数量、包装 | 与到货一致。 |
| ASN、参考订单/行、回写请求号 | 追溯与幂等键。 |
| 方案、阶段、严格度、水平、AQL | 通常由方案带出，慎改。 |
| 自动提交/同意/执行 | 以配置为准。 |

### 常用动作

新增/修改/删除、提交、同意、驳回、处理、关闭。已生成任务后删除需谨慎。

## 任务与记录

| 应核对 | 提醒 |
| --- | --- |
| 任务状态、承接人、计划/完成时间 | 待处理→进行中→完成。 |
| 明细步骤：特性、方法、样本量、接收/拒收数 | 按模板执行。 |
| 包装：抽样/合格/不合格/破坏/冻结数量 | 发布回写用。 |
| 定量值 / 定性选项与缺陷等级 | 选定集来源见配置。 |
| 评估码、使用决策 | 接收/拒绝；合格/不合格/报废/隔离。 |

### 常用动作

任务承接与完成；记录录入、保存、发布。发布后关注上游回写是否成功。

## 发布与回写

| 步骤 | 关注点 |
| --- | --- |
| 发布前 | 包装数量合计与判定一致。 |
| 发布后 | 到货检验结果是否收到；库存后续动作在 WMS。 |
| 失败 | 查回写异常与回调号，勿重复手工改库存。 |

## 最小验收清单

- [ ] 待检到货确认后出现来料检验申请（非免检物料）。
- [ ] 免检物料不建或跳过（与方案一致）。
- [ ] 申请处理生成任务；承接后可录入定量/定性。
- [ ] 发布后面包数量写回到货侧；QMS 记录可追溯 ASN/订单行。
- [ ] 采购收货完成**不再**新增检验申请（回归兼容入口仅日志）。
- [ ] 不合格样例可联查是否进入评审（按组织流程）。
