# 生产检验-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[生产检验](index.md)。面向测试、实施、运维的日常处理、查询与异常定位；四套菜单操作口径相同，仅入口与类型不同。主线、触发边界与建议验证点以主文档为准。

| 你要做什么 | 先看哪里 |

| 想先理解四套分工与主线 | 返回[生产检验](index.md) |

## 字段说明

下表服务排障与维护，不替代主文档主线。

### 字段说明总表

| 字段 | 它决定什么 | 约束摘要 | 配错/漏配时常见现象 |
| --- | --- | --- | --- |
| 检验类型/菜单 | 分流方案与统计 | 首件/末件/巡检/其他 | 选错菜单 → 方案/统计错 |
| 工单/产线/物料 | 生产上下文 | 建议齐全 | 无上下文难追溯 |
| WMS 制品收货触发 | 自动建单（已证实一类） | 类型码与巡检枚举名同码场景 | 与菜单名简单等同 |
| MES 报工 NG | **未证实**自动建单 | 仅线索（`GAP-071`） | 空等自动单 |
| 评估码/使用决策 | 判定与处置意图 | **库存事务归 WMS** | 决策≠库存已变 |
| ATR 状态 | 动作门禁 | 同来料九态/四态 | 非预期状态强发 |

## 申请

| 应核对 | 提醒 |
| --- | --- |
| 物料、数量、方案、检验类型 | 与菜单一致；勿串到来料方案。 |
| 参考订单/凭证、生产相关号 | 便于联查工单。 |
| 包装明细 | 有包装管理时必填。 |

动作：新增、修改、提交、同意、驳回、处理、关闭（以页面为准）。

## 任务与记录

| 应核对 | 提醒 |
| --- | --- |
| 承接人、状态 | 待处理→进行中→完成。 |
| 步骤：特性、方法、样本量、结果 | 定量/定性分开录。 |
| 评估码、使用决策 | 意图表达；决定是否评审；库存到 WMS。 |
| 发布 | 关注是否回写仓储（联调确认）；未发布勿当生效。 |

## 联查

| 场景 | 建议 |
| --- | --- |
| 工单质量问题 | 工单号 → 四套记录过滤；MES 报工记录（NG 线索）。 |
| 制品收货触发 | 收货记录号/回调号 → 检验申请。 |
| 方案不对 | 检验配置中该类型方案与工序码。 |
| 结论后库存未变 | 先确认记录已发布；再到 WMS 查隔离/放行/报废。 |

## 验收时建议核对

与主文档「建议验证点」同口径，便于现场勾选：

- [ ] 四套菜单均可手工建申请并生成任务、完成记录。
- [ ] 首件/末件类型与方案绑定正确（不串到料方案）。
- [ ] 制品收货在开关/规则允许时能建检验申请；回调号可联查。
- [ ] 报工 NG **不**假定自动出单；需手工或其它触发时有操作说明。
- [ ] 不合格样例可转到质量评审（按流程）。
- [ ] 使用决策已选但记录未发布时，不以库存已变为前提；库存事务归 WMS。
- [ ] 与 EAM 设备巡检菜单不混淆。
