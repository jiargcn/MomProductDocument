## 字段业务语义

本页模式归类：**P1 / P2 / P5 / P9 / P12**。四套菜单同结构。`GAP-071`：MES NG→自动建单未证实。

### 字段业务语义总表

| 字段（中文业务名） | 行为模式 | 业务作用 | 业务约束摘要 | 影响哪些功能 | 变更或错选风险 |
| --- | --- | --- | --- | --- | --- |
| 检验类型/菜单 | P1 / P6 | 分流 | 首件/末件/巡检/其他 | 方案与统计 | 选错菜单 |
| 工单/产线/物料 | P2 / P5 | 生产上下文 | 建议齐全 | 追溯 | 无上下文 |
| WMS 制品收货触发 | P12 | 自动建单（已证实一类） | 类型码与巡检枚举名同码场景 | 申请生成 | 与菜单名简单等同 |
| MES 报工 NG | P12 | **未证实**自动建单 | 仅线索 | — | 假自动建单 |
| 评估码/使用决策/ATR | P1 / P9 | 同来料 | 库存归 WMS | 闭环 | 决策≠库存 |

## 申请

| 应核对 | 提醒 |
| --- | --- |
| 物料、数量、方案、检验类型 | 与菜单一致。 |
| 参考订单/凭证、生产相关号 | 便于联查工单。 |
| 包装明细 | 有包装管理时必填。 |

动作：新增、修改、提交、同意、驳回、处理、关闭（以页面为准）。

## 任务与记录

| 应核对 | 提醒 |
| --- | --- |
| 承接人、状态 | 待处理→进行中→完成。 |
| 步骤：特性、方法、样本量、结果 | 定量/定性分开录。 |
| 评估码、使用决策 | 决定是否评审。 |
| 发布 | 关注是否回写仓储（联调确认）。 |

## 联查

| 场景 | 建议 |
| --- | --- |
| 工单质量问题 | 工单号 → 四套记录过滤；MES 报工记录（NG 线索）。 |
| 制品收货触发 | 收货记录号/回调号 → 检验申请。 |
| 方案不对 | 检验配置中该类型方案与工序码。 |

## 最小验收清单

- [ ] 四套菜单均可手工建申请并生成任务、完成记录。
- [ ] 首件/末件类型与方案绑定正确（不串到料方案）。
- [ ] 制品收货在开关/规则允许时能建检验申请；回调号可联查。
- [ ] 报工 NG **不**假定自动出单；需手工或其它触发时有操作说明。
- [ ] 不合格样例可转到质量评审（按流程）。
- [ ] 与 EAM 设备巡检菜单不混淆。
