# 生产检验-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[生产检验](index.md)。四套菜单操作口径相同，仅入口与类型不同。

## 快速定位

| 你要做什么 | 菜单 |
| --- | --- |
| 开线首件 | 首件检验申请/任务/记录 |
| 收尾确认 | 末件检验申请/任务/记录 |
| 过程巡回 | 巡检检验申请/任务/记录 |
| 其他过程场景 | 其他检验申请/任务/记录 |
| 查制品收货是否建单 | WMS 制品收货记录 + 本模块申请（回调号） |

## 申请

| 应核对 | 提醒 |
| --- | --- |
| 物料、数量、方案、检验类型 | 与菜单一致。 |
| 参考订单/凭证、生产相关号 | 便于联查工单。 |
| 包装明细 | 有包装管理时必填。 |

动作：新增、修改、提交、同意、驳回、处理、关闭（以页面为准）。

## 任务与记录

| 应核对 | 提醒 |
| --- | --- |
| 承接人、状态 | 待处理→进行中→完成。 |
| 步骤：特性、方法、样本量、结果 | 定量/定性分开录。 |
| 评估码、使用决策 | 决定是否评审。 |
| 发布 | 关注是否回写仓储（联调确认）。 |

## 联查

| 场景 | 建议 |
| --- | --- |
| 工单质量问题 | 工单号 → 四套记录过滤；MES 报工记录（NG 线索）。 |
| 制品收货触发 | 收货记录号/回调号 → 检验申请。 |
| 方案不对 | 检验配置中该类型方案与工序码。 |

## 最小验收清单

- [ ] 四套菜单均可手工建申请并生成任务、完成记录。
- [ ] 首件/末件类型与方案绑定正确（不串到料方案）。
- [ ] 制品收货在开关/规则允许时能建检验申请；回调号可联查。
- [ ] 报工 NG **不**假定自动出单；需手工或其它触发时有操作说明。
- [ ] 不合格样例可转到质量评审（按流程）。
- [ ] 与 EAM 设备巡检菜单不混淆。
