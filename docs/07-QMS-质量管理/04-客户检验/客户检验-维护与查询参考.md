## 字段业务语义

本页模式归类：**P1 / P2 / P5 / P9 / P12**。对象为**客户退回检验**，非出货 OQC。

| 字段（中文业务名） | 行为模式 | 业务作用 | 业务约束摘要 | 影响哪些功能 | 变更或错选风险 |
| --- | --- | --- | --- | --- | --- |
| 退货记录触发开关 | P12 / P9 | 自动建申请 | 创建后建检验 | 申请生成 | 关则漏建 |
| 客户/物料/数量 | P2 / P5 | 检验对象 | 退货带入 | 追溯 | 参考订单可能置空 |
| 类型=客户退货检验 | P1 | 与方案匹配 | 非发货检验菜单 | 方案选择 | 当 OQC 培训 |
| ATR/判定 | P9 / P1 | 同来料 | 库存归 WMS | 闭环 | — |

## 触发与联查

| 场景 | 操作提醒 |
| --- | --- |
| 自动建单 | 客户退货记录创建后，规则「退货记录创建后创建检验申请」开启；回调号形如「退货申请号:明细ID」防重。 |
| 手工建单 | 选客户退货检验类型方案；补包装数量。 |
| 联查仓储 | 退货记录号、包装号、批次。 |
| 旁路 | 寄售退货/发货调整若建单，用回调号联查，勿只在本菜单死等。 |

## 申请 / 任务 / 记录

口径同来料：状态推进、步骤录入、评估码与使用决策、发布。重点核对物料、数量、包装与退货一致。

## 最小验收清单

- [ ] 规则开启时，客户退货记录可生成客户退回检验申请且不重复。
- [ ] 规则关闭时不自动建单，手工申请仍可用。
- [ ] 任务可完成并形成记录结论。
- [ ] 培训材料不将本页称为出货 OQC。
- [ ] 不合格可联查质量评审/通知单（按组织流程）。
