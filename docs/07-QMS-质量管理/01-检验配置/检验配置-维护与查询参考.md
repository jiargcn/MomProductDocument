# 检验配置-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[检验配置](index.md)。面向测试、实施、运维的日常维护、查询与异常定位；配置主线、边界与建议验证点以主文档为准。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 弄清方案开关如何改行为（免检/自动建单/改库存意图） | 「字段说明」 |
| 建抽样与 AQL | 「抽样与 AQL」 |
| 建方法、特性、模板 | 「方法、特性与模板」 |

| 想先理解配置如何拼成方案 | 返回[检验配置](index.md) |

## 字段说明

下表服务排障与维护，不替代主文档主线。完整行为细则以测试环境为准。

### 字段说明总表

| 字段 | 它决定什么 | 谁在何时维护 | 约束摘要 | 配错/漏配时常见现象 |
| --- | --- | --- | --- | --- |
| 物料 | 方案绑定对象 | 建方案时选 | 可用物料 | 错料 → 执行选错方案 |
| 检验类型 | 来料/过程/首末件等分流 | 建方案时选 | 与执行入口对应 | 类型错对不上入口 |
| 检验模板 | 步骤集合 | 建方案时选 | 模板须可用 | 空模板无法执行 |
| 免检 | 是否跳过检验 | 方案维护 | 到货确认会查询 | 误免检漏检 |
| 自动创建申请 | 自动建单意图 | 方案维护 | 与上游开关叠加 | 期望自动却不建 |
| 改库存状态 | 配置意图 | 方案维护 | **库存事务归 WMS** | 误当库存已变 |
| AQL / 水平 / 拆批 | 抽样规模 | 方案维护 | 依赖字码与 AQL 行 | 抽错量 |
| 生效/失效 | 时间窗 | 方案维护 | 窗口外不可用 | 过期仍被手工误用 |
| MES 工序码 | 过程挂接线索 | 可选 | 映射 ❓（`GAP-071`） | 假自动建单 |
| 动态规则 / 计数器 | 严格度调整 | 质量工程师 | 物料+类型+供应商 | 加严不到位 |
| 选定集项目 | 定性选项 | 配置维护 | 缺陷等级 | 缺选项无法判定 |

### 选择器可选范围

| 选择字段 | 选择对象 | 可选范围 | 范围依赖 | 选中后带回 | 选不到时通常原因 |
| --- | --- | --- | --- | --- | --- |
| 物料 | 物料主数据 | 可用物料 | — | 名称、单位 | 停用 |
| 检验模板 | 检验模板 | 可用模板及版本 | 工厂/业务 ❓ | 过程集合 | 未建、停用 |
| 采样过程 / AQL | 抽样配置族 | 已维护且可用 | 水平枚举 | 样本量线索 | 字码/AQL 行缺失 |
| MES 工序码 | 工序 | 工艺/工序主数据中的工序（精确过滤 ❓） | 可选 | 工序名 | 未建模 |

### 取值影响矩阵（方案关键开关）

| 取值 | 业务含义 | 影响的功能或流程 | 系统行为变化 | 错选/漏选后果 |
| --- | --- | --- | --- | --- |
| 免检=是 | 该物料该类型可跳过检验 | 到货确认建申请 | 可能不建检验申请 | 漏检入可用 |
| 免检=否 | 需要检验 | 到货/执行链 | 待检行可建申请 | — |
| 自动创建=是 | 意图自动建申请 | 上游触发 | 在上游开关允许时建单 | 上游关则仍不建 |
| 自动创建=否 | 依赖手工/其它入口 | 申请生成 | 不自动 | 现场以为会自动 |
| 改库存状态=开 | 配置意图 | 培训/联调预期 | **不替代 WMS 事务** | 误判库存已变 |

## 抽样与 AQL

| 应维护/确认 | 关键限制 |
| --- | --- |
| 采样方案编码与描述、可用 | 被采样过程与方案引用。 |
| 样本字码：批量上下限与 S1–S4 / G1–G3 字码 | 与检验水平枚举对应。 |
| AQL：字码、样本量与各 AQL 档接收/拒收数 | 行数据密，导入须整行核对。 |
| 采样过程：样本类型、评估模式（手工/自动）、样本量、关联采样方案 | 挂到模板过程步骤。 |

### 常用动作

| 动作 | 提醒 |
| --- | --- |
| 新增/修改/启用停用 | 已被方案引用时勿随意改码。 |
| 导入/导出（若页面提供） | 不导入系统编号与审计列。 |

## 方法、特性与模板

| 应维护/确认 | 关键限制 |
| --- | --- |
| 检验方法：编码、描述、版本、状态、操作指导 | 版本变更不影响已生成任务快照（执行侧以任务明细为准）。 |
| 检验特性：定量目标/上下限或定性 | 结果录入方式区分定量/定性。 |
| 模板过程步骤：顺序、特性、方法、是否破坏性、岗位、采样过程 | 决定任务明细行。 |
| 检验模板：编码、描述、版本、工厂/业务、动态规则码、过程集合 | 供方案引用。 |

## 检验方案与物料配置

| 应维护/确认 | 关键限制 |
| --- | --- |
| 物料、检验类型、模板、生效/失效日期、版本 | 执行按类型选方案。 |
| AQL、检验水平、拆批规则、样本量/单位 | 影响抽检规模。 |
| 免检、是否改库存状态、是否自动创建申请、生成方式/定时 | 上游触发与免检查询依赖。 |
| 可选 MES 工序码 | 仅配置线索；自动触发未证实。 |
| 物料检验配置：物料与要求标识 | 与方案交叉复核。 |

### 常用动作

| 动作 | 提醒 |
| --- | --- |

| 停用/到期 | 确认无在途申请依赖。 |

## 验收时建议核对

与主文档「建议验证点」同口径，便于现场勾选：

- [ ] 新建特性（定量+定性）并挂入模板过程。
- [ ] 维护 AQL 与样本字码后，方案能选到对应水平；新任务抽样数与水平一致。
- [ ] 为测试物料配置来料类型方案：非免检；到货确认待检行可建申请（联查来料页）。
- [ ] 免检方案：到货侧查询免检为真时不建或跳过检验（以联调为准）。
- [ ] 计数器累计后，阶段/严格度切换符合阶段定义（抽样验证）。
- [ ] 停用或过期方案后，新申请不可再引用（或按页面校验提示）。
- [ ] 「改库存状态」开启时不以库存已变为验收前提；库存事务归 WMS。
- [ ] MES 工序码仅作线索；不把报工 NG 自动建单列为必过（`GAP-071`）。
