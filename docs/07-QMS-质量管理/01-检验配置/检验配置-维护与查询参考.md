## 抽样与 AQL

| 应维护/确认 | 关键限制 |
| --- | --- |
| 采样方案编码与描述、可用 | 被采样过程与方案引用。 |
| 样本字码：批量上下限与 S1–S4 / G1–G3 字码 | 与检验水平枚举对应。 |
| AQL：字码、样本量与各 AQL 档接收/拒收数 | 行数据密，导入须整行核对。 |
| 采样过程：样本类型、评估模式（手工/自动）、样本量、关联采样方案 | 挂到模板过程步骤。 |

### 常用动作

| 动作 | 提醒 |
| --- | --- |
| 新增/修改/启用停用 | 已被方案引用时勿随意改码。 |
| 导入/导出（若页面提供） | 不导入系统编号与审计列。 |

## 方法、特性与模板

| 应维护/确认 | 关键限制 |
| --- | --- |
| 检验方法：编码、描述、版本、状态、操作指导 | 版本变更不影响已生成任务快照（执行侧以任务明细为准）。 |
| 检验特性：定量目标/上下限或定性 | 结果录入方式区分定量/定性。 |
| 模板过程步骤：顺序、特性、方法、是否破坏性、岗位、采样过程 | 决定任务明细行。 |
| 检验模板：编码、描述、版本、工厂/业务、动态规则码、过程集合 | 供方案引用。 |

### 常用动作

| 动作 | 提醒 |
| --- | --- |
| 维护模板过程 | 先特性与方法，再排顺序。 |
| 复制/升版本（若有） | 旧方案仍指向旧模板时需显式切换。 |

## 检验方案与物料配置

| 应维护/确认 | 关键限制 |
| --- | --- |
| 物料、检验类型、模板、生效/失效日期、版本 | 执行按类型选方案。 |
| AQL、检验水平、拆批规则、样本量/单位 | 影响抽检规模。 |
| 免检、是否改库存状态、是否自动创建申请、生成方式/定时 | 上游触发与免检查询依赖。 |
| 可选 MES 工序码 | 仅配置线索；自动触发未证实。 |
| 物料检验配置：物料与要求标识 | 与方案交叉复核。 |

### 常用动作

| 动作 | 提醒 |
| --- | --- |
| 新增/修改方案 | 同物料同类型勿重叠有效期冲突。 |
| 停用/到期 | 确认无在途申请依赖。 |

## 阶段、动态规则与计数器

| 应维护/确认 | 提醒 |
| --- | --- |
| 阶段：起始阶段、跳检、严格度、连续合格/不合格切换 | 与计数器配合。 |
| 动态规则：类型级/批次级/特征级 | 层级选错会导致加严不到位。 |
| 计数器：物料+检验类型+供应商的合格/不合格次数与下一阶段 | 来料场景尤其关键。 |

## 选定集

| 应维护/确认 | 提醒 |
| --- | --- |
| 选定集分类与项目：字典值、评估码、缺陷等级 | 定性录入下拉来源。 |

## 最小验收清单

- [ ] 新建特性（定量+定性）并挂入模板过程。
- [ ] 维护 AQL 与样本字码后，方案能选到对应水平。
- [ ] 为测试物料配置来料类型方案：非免检；到货确认待检行可建申请（联查来料页）。
- [ ] 免检方案：到货侧查询免检为真时不建或跳过检验（以联调为准）。
- [ ] 计数器累计后，阶段/严格度切换符合阶段定义（抽样验证）。
- [ ] 停用方案后，新申请不可再引用（或按页面校验提示）。
