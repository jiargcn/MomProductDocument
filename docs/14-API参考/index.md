# API 参考

## 概述

MOM 系统提供统一的 RESTful API，供各模块之间以及外部系统调用。

## 接口取证模板

| 项目 | 需要记录的内容 |
|------|----------------|
| 接口名称 | 使用业务可理解名称，不只写方法名。 |
| 所属模块 | DBC、WMS、MES、QMS、EAM、ANDON、SCP、PS、System、Infra。 |
| 请求方法与路径 | `GET`、`POST`、`PUT`、`DELETE` 等方法和真实 URL。 |
| 认证与权限 | 登录态、Token、租户、菜单权限、按钮权限、接口权限、数据范围。 |
| 请求字段 | 字段技术名、类型、是否必填、长度、枚举、来源和校验规则。 |
| 响应字段 | 字段技术名、类型、含义、分页结构、状态码。 |
| 幂等与重试 | 是否支持幂等键、重复调用结果、失败重试和补偿机制。 |
| 业务影响 | 创建/更新哪些申请、任务、记录、库存事务、余额、消息或日志。 |
| 错误处理 | 错误码、错误提示、业务校验失败、系统异常和第三方异常。 |
| 审计与日志 | 是否记录操作者、请求摘要、响应摘要、关联单据和追踪号。 |

## 第三方接口挂接占位

| 接口类型 | 典型方向 | 待确认内容 |
|----------|----------|------------|
| ERP / SAP 主数据 | ERP/SAP → MOM 或 MOM → ERP/SAP | 物料、供应商、客户、组织、工厂、库存地点等同步规则。 |
| 采购与供应链 | ERP/SAP/SCP ↔ WMS/SCP | 采购订单、ASN、收货、退货、发货、对账和发票。 |
| 生产与排程 | ERP/MES/PS ↔ MOM | 生产计划、工单、排程结果、报工、完工和异常回写。 |
| 质量 | QMS ↔ WMS/MES/ERP | 检验任务、检验结果、放行、冻结、让步、退货和质量评审。 |
| 设备与数采 | EAM/ANDON/数采 ↔ MOM | 设备状态、报警、维修、停机、OEE 和异常响应。 |
| 报表与看板 | MOM → 报表平台/BI | 数据源、统计口径、权限过滤和刷新频率。 |
