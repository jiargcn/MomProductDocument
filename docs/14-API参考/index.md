# API 参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：实施、集成开发、需要查「怎么调 / 调失败怎么办」的支持人员。

## 这一组文档解决什么问题

API 参考说明 **如何调用与联查** MOM 接口：认证与权限边界、文档发现方式、按模块索引关键能力，以及跨系统集成场景。业务「为什么有这个动作」仍以各业务模块页为准；平台调用轨迹与重试见[数据交换与集成可靠性](../03-基础设施/09-数据交换与集成可靠性.md)。

本轮 **不** 把全站 Controller 逐条抄成 OpenAPI 手册（体量过大且易过期）。完整字段以环境 Knife4j/Swagger 与联调样例为准；此处沉淀已证实的组织方式、抽样路径与挂接矩阵。
