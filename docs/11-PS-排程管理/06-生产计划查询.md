# 生产计划查询

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> **资料状态：薄弱。** 不写未取证甘特交互、冲突高亮与导出能力。

## 业务目的

本分组在产品规划中，以可视化方式（常见为甘特图）展示**已发布或选定版本的生产计划**，帮助计划员识别产线负荷、空档与冲突。它是计划层视图，不是 MES 线边执行看板。
