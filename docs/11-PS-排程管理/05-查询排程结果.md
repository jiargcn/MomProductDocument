# 查询排程结果

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> **资料状态：薄弱。** 不写未取证明细字段与超产能告警规则。

## 业务目的与适用范围

本分组在产品规划中，提供排程执行后的**明细只读查询**：按版本、日期、产线、物料查看计划产量与产能占用，支持从汇总钻到明细。

读完本页，应能：把它与 MES 工单列表、WMS 生产计划、甘特视图区分开；在 API/表结构取证前不恢复旧稿明细字段表。

## 如何使用本页

| 你的目的 | 建议怎么做 |
| --- | --- |
| 了解排程明细查询的规划意图 | 本页「规划中应覆盖的能力」 |
| 查 MES 工单列表 | 去 [MES 计划管理](../06-MES-生产管理/03-计划管理/index.md) |
| 查仓储侧生产协同计划 | 以 [WMS 生产管理](../05-WMS-库房管理/08-生产管理/index.md) 为准 |
| 看甘特可视化（规划） | [生产计划查询](06-生产计划查询.md) |
