# 执行生产排程

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> **资料状态：薄弱。** 不写未取证算法参数、汇总指标公式与状态机。

## 业务目的与适用范围

本分组在产品规划中，是排程引擎的**运行入口**：选择场景与订单版本，加载产能/节拍等约束，生成排程结果版本，供后续对比与查询。

读完本页，应能：理解期望步骤；知道工单下发/开工不在本页；在引擎与接口取证前不把旧稿利用率公式当培训事实。

## 如何使用本页
