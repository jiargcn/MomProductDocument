## 维护入口

| 入口 | 作用 | 说明 |
| --- | --- | --- |
| 租户列表 | 维护租户状态、联系人、套餐、到期、账号数量等。 | 多租户运营主入口。 |
| 租户套餐 | 维护套餐可选菜单集合。 | 决定租户功能上限。 |
| 用户管理 | 维护租户内账号。 | 登录主体。 |
| 角色/菜单 | 租户内功能授权。 | 见 RBAC 维护参考。 |

## 配置项说明

| 配置项 | 业务含义 | 配置时注意 |
| --- | --- | --- |
| 租户名称 / 状态 | 识别与启停。 | 停用/到期会影响登录与使用。 |
| 套餐 | 功能菜单上限。 | 先定套餐再开租户更稳妥。 |
| 到期时间 | 租户有效期。 | 临期应有运维提醒机制（环境侧）。 |
| 账号数量 | 租户账号规模上限。 | 超限策略以环境实际行为为准。 |
| 域名 | 租户访问识别相关配置。 | 按部署方式核验是否启用。 |
| 登录方式 | 密码/短信/社交等。 | 仅培训环境已启用的方式。 |

## 建议配置顺序

1. 先维护租户套餐菜单范围。
2. 再创建租户并绑定套餐与有效期。
3. 确认初始管理员可登录。
4. 再在租户内维护部门、用户、角色。
5. 用业务角色账号验收菜单与数据范围。

## 查询与联查

| 场景 | 建议看什么 | 联查建议 |
| --- | --- | --- |
| 租户菜单缺失 | 套餐是否包含、角色是否授权、菜单是否启用。 | 套餐、RBAC。 |
| 无法登录 | 租户状态/到期、用户状态、认证方式。 | 租户列表、用户管理。 |
| 登录后权限标识不对 | 角色菜单与权限标识。 | [RBAC权限模型-维护与查询参考](../03-用户与权限/RBAC权限模型-维护与查询参考.md)。 |

## 验收检查清单

- [ ] 套餐菜单与租户可见功能一致。
- [ ] 停用或到期租户无法正常业务登录（按环境策略）。
- [ ] 登录、刷新、退出在验收账号下可用。
- [ ] 未把未启用的短信/社交登录写成必配步骤。

!!! example "写实示例：给定配置 → 期望可见范围"
    **给定：** 套餐仅勾选「系统管理 + 仓储收货」；租户已绑定该套餐；验收账号角色勾选了收货与排程菜单。
    **期望：** 可见收货入口；排程入口因套餐上限不可见。若收货仍不可见，先查角色勾选与菜单启用，再查是否未重新登录。
