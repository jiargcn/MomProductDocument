# 租户与认证

> 状态：占位页已建立。当前租户模型、登录方式和会话实现待 System 模块源码与测试环境核验。

## 页面定位

本分组说明“谁能够以何种身份进入系统”，不承担组织层级、数据范围或业务审批规则的解释。

## 待核验内容

| 主题 | 需要确认的内容 |
| --- | --- |
| 租户 | 租户对象、隔离维度、跨租户管理和初始化规则。 |
| 身份认证 | 账号密码、单点登录、第三方身份源、终端登录方式。 |
| 会话安全 | 登录有效期、刷新、退出、冻结、密码与失败锁定策略。 |
| 审计 | 登录、登出、认证失败、账号状态变更的记录与查询。 |
