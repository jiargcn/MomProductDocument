# 系统参数、字典与个性化配置

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：系统管理员、实施顾问，以及需要维护下拉选项与参数开关的配置人员。

## 业务目的与适用范围

本页说明跨模块复用的三类配置：

1. **数据字典**：为页面下拉、状态标签等提供统一选项；
2. **系统/基础设施参数**：键值型运行参数（可见性可控）；
3. **个性化**：用户级页面偏好（如按用户与页面保存的配置）。

业务含义强的策略、业务类型、单据开关仍归 DBC/业务页，不在本页当“字典”混写。

## 使用前准备

| 需要确认什么 | 为什么重要 |
| --- | --- |
| 字典编码是否已被业务引用 | 改值/删项可能导致页面选项异常或历史数据难读。 |
| 参数是否对前端可见 | 不可见参数通常仅后端使用，勿当页面开关乱改。 |
| 变更环境 | 生产改字典/参数需评审与回滚预案。 |
| 租户范围 | 多租户下确认是否改到目标租户。 |

## 数据字典

```mermaid
flowchart LR
    A["字典类型"] --> B["字典数据项"]
    B --> C["业务页面下拉/标签"]
```

| 对象 | 业务含义 | 维护注意 |
| --- | --- | --- |
| 字典类型 | 一组选项的编码与名称、启停状态。 | 类型编码稳定后不宜随意改名改码。 |
| 字典数据 | 标签、值、排序、颜色样式、启停。 | 值被持久化引用时，停用优于删除。 |

常见用途：状态、类型、业务细分等下拉。业务页应写清使用了哪个字典语义，而不是只写“下拉可选”。

## 系统参数

| 能力 | 当前结论 |
| --- | --- |
| 参数配置入口 | 基础设施“配置管理”维护分类、键、值、类型、是否可见、备注。 |
| 可见性 | 不可见参数一般不暴露给前端读取。 |
| 与业务策略区别 | 参数是通用键值；业务策略/单据开关有独立业务模型。 |

变更后若页面不生效，应检查缓存、重新登录与是否改错键名。

## 个性化与相关配置

| 能力 | 当前结论 |
| --- | --- |
| 用户页面配置 | 支持按用户与页面组件保存/查询页面级配置。 |
| 多语言配置 | 存在多语言配置管理菜单；启用范围与覆盖字段待环境确认。 |
| 统一用户视图中心 | **未证实**为全站共享默认视图平台（见查询页）；勿把旧规划直接写成现状。 |
| 个人中心 | 用户可维护个人资料类信息（密码/头像等以实际页面为准）。 |

## 其它相关配置入口

| 入口 | 用途 | 详见 |
| --- | --- | --- |
| 错误码管理 | 维护业务错误码与提示文案。 | 本页联查；变更需评估前端提示一致性。 |
| 敏感词管理 | 内容过滤类配置。 | 启用范围待业务确认。 |
| 短信/邮件/站内信模板 | 通知渠道配置。 | [消息、通知与待办](../../03-基础设施/06-消息、通知与待办.md)。 |

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 下拉缺项 | 字典项是否停用、类型是否正确、缓存。 | 先查字典再改业务代码预期。 |
| 参数改了无效 | 键名、可见性、租户、是否需重启/刷缓存。 | 避免重复改值。 |
| 个性化丢了 | 是否换浏览器/清缓存、是否换账号。 | 用户级配置不跨人共享（除非另有共享能力）。 |

## 查询与联查

| 场景 | 建议看什么 | 联查 |
| --- | --- | --- |
| 状态显示不对 | 字典标签与值、业务存的是值还是标签。 | 业务页字段说明。 |
| 参数争议 | 配置管理中的键值与备注、变更审计。 | 操作日志。 |
| 业务开关找不到 | 是否在单据开关/业务类型而非字典。 | DBC 策略设置。 |

## 常见问题与处理

| 情况 | 建议处理 |
| --- | --- |
| 删除仍被引用的字典值 | 改为停用；评估历史单据展示。 |
| 用字典模拟复杂策略 | 改走业务策略/规则配置。 |
| 把 Infra 参数当成租户业务参数随意改 | 先确认影响范围与备份。 |

## 当前限制与待确认事项

- 字典引用检查与导入规则的完整约束待抽样；
- 参数变更强制审计与审批流未证实为通用能力；
- 全站共享默认视图/字段权限配置台未证实；
- 多语言覆盖范围与运行时切换体验待测试环境确认。
