# 数据权限与决策权限

> 状态：占位结构已建立。  
> 目标：补足“能否进入页面”之外的两类权限：能看到哪些数据，以及能否作出业务决策。

## 待填写内容

| 类型 | 需要说明 |
| --- | --- |
| 数据权限 | 组织、部门、工厂、仓库、库位、货主、业务类型等过滤维度及优先级。 |
| 决策权限 | 提交、审批、驳回、接单、执行、关闭、撤销、冲销、补录等动作的授权依据。 |
| 审计要求 | 关键动作记录的操作者、时间、前后状态、原因、关联单据和附件。 |
| 异常授权 | 允许超数量、负库存、修改库位/批次、跳过检验等例外能力的授权与留痕。 |

> 业务页面只引用本页定义的权限类型，再补充本业务的具体动作，不重复定义权限模型。
