# 用户与权限

> 状态：BATCH-02 首轮占位页。当前页面用于说明 System 模块中用户、角色、权限、数据范围和决策权限的文档边界；具体字段、权限码、接口和角色配置仍需从 System 源码、菜单清单和测试环境逐项取证。

## 1. 页面定位

用户与权限用于回答三个问题：

1. 谁可以登录系统、属于哪个组织或岗位。
2. 登录后能看到哪些菜单、按钮和页面。
3. 在具体业务状态下，能否执行审批、驳回、关闭、撤销、人工干预等决策动作。

本分组不替代 WMS、MES、QMS、EAM 等业务页中的动作规则。业务页需要引用本分组的权限模型，但仍需说明该业务动作的状态前置条件、数据影响和审计要求。

## 2. 分页说明

| 页面 | 说明 | 后续取证重点 |

| 数据权限与决策权限 | 说明数据范围与高风险业务动作权限的差异。 | 组织、部门、工厂、仓库、货主等数据范围；审批/关闭/撤销等动作权限。 |
