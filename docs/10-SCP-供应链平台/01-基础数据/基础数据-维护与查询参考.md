# 基础数据-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[基础数据](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 配供应商与可供物料 | “供应商与物料” |
| 配价格 | “采购价格” |
| 配包装/标签 | “包装与标签” |
| 配仓/月台 | “仓储建模（SCP）” |
| 关业务入口 | “单据开关与策略” |
| 开门户账号 | “供应商用户” |
| 改企业权威主数据 | 转 DBC，勿默认只在 SCP 改 |

## 页面与业务对象

| 菜单入口（scp-ui 视图域） | 用途 |
| --- | --- |
| 供应商 / 供应商物料 / 采购价格 | 协同供应与计价。 |
| 物料基本信息 / 物料包装 / 包装规格 / 层级 / 客户物料 | 物料与包装。 |
| 仓库 / 库区 / 库位组 / 库位 / 月台 | 协同侧仓储定位。 |
| 单据开关 / 策略 / 供应商发货策略 / 供应商周期 | 入口与节奏。 |
| 供应商用户 | 门户绑定。 |
| 标签类型/规则/模板、包装相关标签、接口信息 | 标签与接口登记。 |
| ERP 成本中心 | 只读同步数据。 |

## 供应商与物料

| 应维护/确认 | 提醒 |
| --- | --- |
| 供应商编码、名称、联系、可用期 | 订单/计划按供应商过滤。 |
| 供应商+物料关系、包装单位/标准量 | 发货与下单引用。 |
| 与 DBC 编码是否一致 | 不一致时跨模块联查会失败；同步规则待确认。 |

### 常用动作

新增/修改/启停；避免删除已被订单、计划、发货引用的编码。

## 采购价格

| 应维护/确认 | 提醒 |
| --- | --- |
| 供应商、物料、币种、单价、生效/失效 | 过期价格勿再用于新单。 |

## 包装与标签

| 应维护/确认 | 提醒 |
| --- | --- |
| 包装规格与层级、物料包装 | 发货包装与标签依赖。 |
| 标签类型/规则/模板 | 打印与推送 WMS 条码前确认模板。 |

## 仓储建模（SCP）

| 应维护/确认 | 提醒 |
| --- | --- |
| 仓库→库区→库位组→库位、月台 | 计划到仓/到月台；**非** WMS 库存权威。 |

## 单据开关与策略

| 应维护/确认 | 提醒 |
| --- | --- |
| 开关编码与生效值 | 关闭后入口不可用。 |
| 策略 / 发货策略 / 供应商周期 | 影响计划与预测节奏；改前确认在用单据。 |

## 供应商用户

| 应维护/确认 | 提醒 |
| --- | --- |
| 用户与供应商绑定、可用 | 决定门户可见订单/计划范围。 |

## 最小验收清单

- [ ] 至少一家可用供应商 + 一条供应商物料可供下单。
- [ ] 至少一条有效采购价格。
- [ ] 包装规格可被发货明细引用。
- [ ] 单据开关打开后对应菜单可见。
- [ ] 供应商用户可登录并只见本供应商数据（按环境权限验证）。
- [ ] 企业级主数据变更需求能区分走 DBC 还是 SCP。
