# 采购跟踪-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[采购跟踪](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 按 ASN/PO 查收货 | “收货链” |
| 查退货 | “退货链” |
| 查短缺 | “短缺” |
| 仓库现场作业 | 转 WMS，不要只在 SCP 点完成 |

## 页面与业务对象

| 菜单入口（scp-ui） | 用途 |
| --- | --- |
| 采购收货申请/任务/记录 | 到货处理与结果。 |
| 采购退货申请/任务/记录 | 退货。 |
| 上架/检验相关页 | 迁移能力；权威在 WMS/QMS。 |

## 收货链

| 应核对 | 提醒 |
| --- | --- |
| 申请←发货 ASN / PO | 来源正确。 |
| 任务状态与拒收 | REFUSAL 需原因。 |
| 记录数量与订单行 receivedQty | 与累计一致。 |

### 常用动作

查询、联查发货记录；创建/执行类动作若按钮不可用或报依赖 WMS，改走 WMS。

## 退货链

| 应核对 | 提醒 |
| --- | --- |
| 来源收货记录、退货数量与原因 | 影响订单 returnedQty。 |
| SCP 创建退货记录的未实现警告 | 以 WMS 采购退货为准。 |

## 短缺

| 应核对 | 提醒 |
| --- | --- |
| 短缺明细与收货差异 | 用于差异追溯，不是库存调整入口。 |

!!! example "📷 截图占位"
    按 ASN 联查发货→收货记录。

## 最小验收清单

- [ ] 用 ASN 能从发货记录联查到收货申请或记录（按环境链路）。
- [ ] 收货完成后订单/计划已收数量增加。
- [ ] 库存余额变化在 WMS 可查。
- [ ] 退货样例在权威系统可闭环；SCP 仅作跟踪时不报错误导。
