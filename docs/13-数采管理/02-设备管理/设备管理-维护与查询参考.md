# 设备管理-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[设备管理（数采）](index.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 建采集设备 | “采集设备” |
| 查某参数历史值 | “采集日志” |
| 看客户端是否在线 | “MQTT 客户端监控” |

## 页面与业务对象

| 菜单入口 | 组件路径线索 | 用途 |
| --- | --- | --- |
| 设备管理 | `iot/assetcollectinfo` | 采集设备 CRUD / 导出 |
| 采集日志 | `iot/paramcollectlogs` | 参数采集时序查询 / 导出 |
| MQTT客户端监控 | `iot/mqttclientstatus` | 客户端在线监控 / 导出 |

## 采集设备

### 查询

| 要找什么 | 推荐条件 |
| --- | --- |
| 某编码设备 | 设备编码 |
| 仅启用设备 | 设备状态 = 启用 |

### 应核对字段

| 字段 | 提醒 |
| --- | --- |
| 设备编码 / 名称 | 参数配置与日志按编码关联 |
| 设备类别 | 字典 `iot_asset_type` |
| 设备状态 | 启用后方可期望采集 |

### 常用动作

新增、编辑、删除、导出。权限码形如 `iot:asset-collect-info:*`。

## 采集日志

### 查询

| 要找什么 | 推荐条件 |
| --- | --- |
| 某设备近期采集 | 创建时间范围 + 设备编码 |
| 某参数 | 参数编码 |
| 某采集点 | 采集点 |
| 某客户端上报 | 客户端 ID |

### 应核对字段

| 字段 | 提醒 |
| --- | --- |
| 设备编码 / 参数编码 / 采集点 | 与配置侧一致 |
| 采集值 / 参数类型 | 业务解读在消费模块 |
| 采集时间（及时区线索） | 排查时钟偏差 |
| 客户端 ID | 与 MQTT 监控对照 |

【截图占位：采集日志筛选条与结果行。】

### 常用动作

搜索、重置、导出。勿把日志编辑当成改业务单据的手段（除非现场明确授权）。

## MQTT 客户端监控

### 查询

| 要找什么 | 推荐条件 |
| --- | --- |
| 某客户端 | 客户端 ID |
| 仅在线/离线 | 在线状态（字典 `iot_client_link_staus`） |
| 某时段变更 | 创建时间范围 |

### 应核对字段

| 字段 | 提醒 |
| --- | --- |
| 客户端 ID | 与日志 clientId 对照 |
| 在线状态 | 链路健康第一眼 |
| 上线 / 下线时间 | 断线窗口 |
| 用户名 | 认证线索（若展示） |

页面可能隐藏「新增」而保留导出；以测试环境按钮为准。权限码形如 `iot:mqtt-client-status:*`。

## 最小验收清单

- [ ] 新建并启用一台采集设备。
- [ ] 为其配置至少一项「采集」参数后，边缘上报成功。
- [ ] 采集日志可按设备编码查到对应采集值。
- [ ] MQTT 监控中能看到对应客户端状态（若环境启用 MQTT）。
- [ ] 导出日志/监控（有权限时）可用。
