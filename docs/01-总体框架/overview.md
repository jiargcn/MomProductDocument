# 系统概述

> 状态：总体结构已建立；业务边界、实际部署与集成对象待以 `dev` 基线源码、菜单和测试环境核验。

## 产品定位

本产品文档将 MOM 作为制造运营业务的协同与执行中枢进行说明：统一管理主数据、供应链协同、仓储执行、生产执行、质量、设备、异常、排程与数据采集等能力。

旧版规划书提出“核心业务、支撑业务、公共业务”三层划分，可作为本文档梳理范围的参考；模块是否已在当前基线实现，必须另行取证。

## 核心能力

- 统一主数据与工厂业务建模
- 供应、仓储、生产、质量、设备与异常的协同闭环
- 单据、库存、批次/包装与业务记录追溯
- 报表、看板、标签打印、消息、接口、日志等公共能力
