# 模块边界与系统上下文

> 状态：占位页已建立。职责划分来自现有模块目录与旧版规划书，实际数据写入与调用关系待源码核验。

## 1. 模块职责地图

| 模块 | 业务职责 | 主要协作对象 |
| --- | --- | --- |
| DBC | 主数据、工厂建模、策略和业务基础配置 | 全部业务模块。 |
| WMS | 仓储作业、库存、包装/器具、库内管理 | SCP、QMS、MES、EAM。 |
| MES | 工艺、计划执行、投料、报工、追溯 | WMS、QMS、EAM、ANDON、数采。 |
| QMS | 检验、质量判定、不合格与质量闭环 | SCP、WMS、MES、ANDON。 |
| EAM | 设备、备件、工装及维护作业 | MES、ANDON、数采、WMS。 |
| SCP | 采购、预测、供应商/客户协同和结算 | DBC、WMS、QMS、财务/外部平台。 |
| ANDON | 异常触发、响应、升级、关闭和展示 | MES、EAM、QMS、数采、消息平台。 |
| 系统与基础设施 | 组织权限、公共能力、运行与集成基础 | 全部模块。 |
