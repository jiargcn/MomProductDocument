# ERP 成本中心

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。

## 当前用途

ERP 成本中心是科目账户和结算归属的选择来源。WMS 菜单提供该入口，但当前资料尚未确认成本中心是否允许在 WMS 内新增、编辑或仅供查询。

## 维护与查询说明

| 事项 | 当前结论 |
| --- | --- |
| 新增/编辑 | 【待确认】先以 ERP 或同步责任方为唯一维护来源。 |
| 选择/使用 | 选择前确认代码、名称、组织归属和有效状态。 |
| 查询 | 优先按代码、名称和有效状态定位，并回查科目账户配置。 |
