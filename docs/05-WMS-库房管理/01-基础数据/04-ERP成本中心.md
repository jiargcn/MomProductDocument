# ERP 成本中心

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。

## 当前用途

ERP 成本中心为科目账户、产品类成本中心字段及结算归集提供可选的成本中心主数据。菜单挂在 WMS 基础数据下，名称强调与 ERP/QAD 口径对齐。

## 当前可证实行为

| 事项 | 当前结论 |
| --- | --- |
| 维护入口 | WMS 页提供新增、编辑、删除、查询、导入/导出与启停类能力；创建时会校验成本中心 ID、代码不与已有记录重复。 |
| 业务角色 | 作为科目账户等配置的选择来源；导入科目账户时可校验默认成本中心是否仍可用。 |
| 编辑/删除保护 | 更新可写回；删除未见科目账户等下游引用拦截。 |
| 与“仅 ERP 同步”口径 | 源码已提供本地维护接口；是否仍应以 ERP 同步为唯一权威来源、本地改动是否会被同步覆盖，尚未闭合。 |

## 维护与查询说明

| 事项 | 建议做法 |
| --- | --- |
| 新增/编辑 | 在未确认同步策略前，优先与财务/ERP 责任方对齐编码与名称，避免只在 WMS 侧临时建码。 |
| 选择/使用 | 选择前确认代码、名称、类型和可用状态。 |
| 查询 | 优先按代码、名称和可用状态定位，并回查科目账户配置。 |

!!! warning "❓ 待确认"
    同步频率、失败告警、本地维护与 ERP 权威源冲突时以谁为准，以及历史成本中心停用后的业务处理（关联 `WMS-SA-002`）。
