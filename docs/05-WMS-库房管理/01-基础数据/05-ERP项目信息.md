# ERP 项目信息

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。

## 当前用途

ERP 项目信息为需要项目归集的科目账户提供选择来源。该入口的维护边界和同步机制尚未完成取证，因此本页只固化当前可安全使用的查询与选择原则。

## 维护与查询说明

| 事项 | 当前结论 |
| --- | --- |
| 新增/编辑 | 【待确认】默认由 ERP 或同步责任方维护。 |
| 选择/使用 | 先核对项目代码、名称、有效状态及适用组织。 |
| 查询 | 从项目代码或名称定位，再回查关联科目账户和业务结果。 |
