# 产品类

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。

## 这项资料解决什么问题

产品类为物料的差异、重估、报废和盘点等结算场景提供分类及账户归属基础。它是业务与财务口径之间的桥梁，而不是现场作业人员的临时选择项。

## 维护与查询说明

| 位置 | 应关注的信息 | 使用建议 |
| --- | --- | --- |
| 新增/编辑 | 产品类、所属域、差异/重估/报废/盘点的账户及成本中心、可用状态。 | 由熟悉财务口径的角色维护，先完成账户有效性核对。 |
| 列表 | 产品类、描述、主要账户、可用状态。 | 优先定位异常结算所使用的产品类。 |
| 详情 | 基本分类、各类结算归属、成本中心、变更信息。 | 变更前核对受影响物料和未结业务。 |

当前已确认支持维护、查询、导入导出与启停；唯一性、已被物料/业务引用后的编辑限制仍待专项验证。
