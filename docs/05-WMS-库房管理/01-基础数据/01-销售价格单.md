# 销售价格单

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。具体操作见[销售价格单-维护与查询参考](07-销售价格单-维护与查询参考.md)。

## 这项资料解决什么问题

销售价格单把特定客户、物料和币种下的价格及有效期集中维护，避免业务人员在订单或结算环节临时判断价格。

## 维护时先做的判断

| 判断 | 业务含义 |
| --- | --- |
| 适用客户与物料是否明确 | 防止价格被错误客户或物料使用。 |
| 币种、价格和有效期是否完整 | 防止价格无从解释或跨期使用。 |
| 是否已存在生效配置 | 同一适用范围的并存或切换规则尚待验证。 |

## 使用与风险

价格单支持新增、编辑、查询、导入导出与启停。已在用价格不宜直接覆盖历史口径；应采用新建并受控切换的方式处理调整。
