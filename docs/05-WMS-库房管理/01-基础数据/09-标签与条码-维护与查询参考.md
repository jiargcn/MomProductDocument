# 标签与条码-维护与查询参考

> 本页是[标签与条码](06-标签与条码.md)的操作参考。

## 配置、生成与打印的顺序

1. 先维护标签规则、类型和模板，并用脱敏样例验证解析与校验。
2. 再生成采购件、制造件、器具、库位或叫料标签。
3. 打印后以扫码验证识别结果；确需补打时记录原因并复核打印次数。

## 维护限制

| 对象 | 新增/编辑应关注 | 高风险限制 |
| --- | --- | --- |
| 标签规则 | 代码、顺序、长度、前缀、对应业务属性、是否启用。 | 在用规则调整可能改变解析结果，应新建版本并测试。 |
| 标签类型/模板 | 数据协议、解析/校验方式、模板文件、加密/压缩及启用状态。 | 必须同时验证打印和扫码，不能只看模板预览。 |
| 标签信息 | 标签号、类型、模板、状态、关联对象、条码内容和打印记录。 | 已生成标签不应通过普通编辑改变其业务关联；具体限制待确认。 |
