# 科目账户配置-维护与查询参考

> 本页是[科目账户配置](03-科目账户配置.md)的操作参考。

## 新增与编辑约束

| 信息组 | 应维护的信息 | 维护限制 |
| --- | --- | --- |
| 使用与账户 | 使用场景、会计账户、账户名称。 | 使用场景必须能被业务和财务共同解释。 |
| 归集维度 | 成本中心、项目、是否必填、默认策略。 | 选择值应来自有效 ERP 基础资料。 |
| 产品类关系 | 是否由产品类带出、对应产品类类型。 | 已由产品类承担的规则不重复设置。 |
| 生效控制 | 可用状态、备注。 | 影响在用业务前先评估切换范围。 |

## 导入、查询与联查

当前已确认支持模板下载、导入导出和启停；模板具体列、重复处理和已使用后的编辑保护待测试环境验证。

| 位置 | 建议优先显示 | 联查 |
| --- | --- | --- |
| 列表 | 使用场景、会计账户、成本中心、项目、产品类来源、可用状态。 | 回查 ERP 成本中心/项目与产品类。 |
| 详情 | 基本账户、归集维度、产品类规则、生效控制、变更信息。 | 查看受影响的结算或库存差异业务（待验证）。 |
