> 状态：首轮占位，待基于菜单、前端页面、后端服务、DDL 和测试环境继续核验。当前页为 WMS 系统设置分组入口，不直接作为单一业务实体页。

## 当前菜单与配置边界（BATCH-01 首轮取证）

> 取证依据：`reference/menu.csv`、Web 目录 `ui/src/views/wms/basicDataManage/systemSetting` 与 `documentSetting`。本页只确认当前入口和业务关联方向，具体字段、校验和状态仍待实体页取证。

| 配置组 | 当前菜单/页面 | 已确认业务关联 | 当前边界 |
| --- | --- | --- | --- |
| 系统日历 | 系统日历（`systemcalendar`） | 为 WMS 提供日历配置入口。 | 是否实际驱动工作日、计划日期或任务排程，尚未从服务调用与测试环境确认。 |
| 账期日历 | 账期日历（`accountcalendar`） | 调整服务已注入账期日历服务，盘点调整属于需要重点验证的使用场景。 | 关账、反过账、跨期间调整等规则尚不能按历史草稿下结论。 |
| 事务类型 | 事务类型（`transactiontype`） | 盘点调整等业务会从业务类型配置取入/出事务类型，再创建库存事务并影响余额。 | 事务类型的完整字段、可编辑边界、库存动作和所有业务映射需单独取证。 |
| 计划设置 | 计划设置（`plansetting`） | 当前是 WMS 单据设置下的独立入口。 | 与盘点计划、备料计划、生产计划或定时任务的实际关系尚未确认。 |

### 当前使用原则

- 将系统日历、账期日历、事务类型和计划设置视为**受控配置**，而不是普通业务数据。培训材料应先说明其影响范围与变更审批要求，具体可编辑字段和权限待实体取证后再固化。
- 页面存在不等于所有业务已接入。只有能从服务调用、配置引用或测试环境验证的关系，才可写为当前运行规则。
- 标签、条码、打印不属于本页的配置对象；它们当前有 WMS 菜单入口，但应在 [基础数据](../01-基础数据/index.md) 标注为“当前入口、目标 Infra 归属”的过渡能力。
