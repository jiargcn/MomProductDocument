# WMS 系统设置

## BATCH-01 标准占位

> 状态：首轮占位，待基于菜单、前端页面、后端服务、DDL 和测试环境继续核验。当前页为 WMS 系统设置分组入口，不直接作为单一业务实体页。

本页用于汇总 WMS 内影响业务运行的配置入口，包括日历、账期、单据设置、事务类型、计划设置、标签规则等。若配置实际属于 DBC、System 或 Infra，应在本页保留入口说明，并在对应模块页面维护字段、导入、权限和业务规则。

| 主题 | 当前占位 | 后续取证 |
| --- | --- | --- |
| 页面范围 | 待核验 WMS 系统设置菜单下实际包含哪些页面。 | `reference/menu.csv`、前端路由、WMS/DBC/Infra 后端模块。 |
| 归属边界 | 待区分 WMS 业务配置、DBC 策略配置、Infra 平台能力。 | 菜单、源码、业务确认。 |
| 业务影响 | 待确认各设置对申请/任务/记录、库存事务、标签打印、计划生成的影响。 | 后端服务调用、规则配置、测试环境。 |
| 字段真实性 | 不在分组页写字段事实；字段在具体配置页取证。 | DDL、DO、DTO、前端配置。 |
| 导入导出 | 待确认各设置是否支持导入、导出和变更日志。 | 前端按钮、Excel VO、导入导出服务。 |
| 权限与日志 | 按 [RBAC 与动作权限取证模板](../../15-版本路线图/RBAC与动作权限取证模板.md) 逐页补。 | 菜单权限、按钮权限、接口权限、操作日志。 |
