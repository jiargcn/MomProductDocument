> 取证依据：`reference/menu.csv`、Web 目录 `ui/src/views/wms/basicDataManage/systemSetting` 与 `documentSetting`。本页只确认当前入口和业务关联方向，具体字段、校验和状态仍待实体页取证。

| 配置组 | 当前菜单/页面 | 已确认业务关联 | 当前边界 |

### 当前使用原则

- 将系统日历、账期日历、事务类型和计划设置视为**受控配置**，而不是普通业务数据。培训材料应先说明其影响范围与变更审批要求，具体可编辑字段和权限待实体取证后再固化。
- 页面存在不等于所有业务已接入。只有能从服务调用、配置引用或测试环境验证的关系，才可写为当前运行规则。
- 标签、条码、打印不属于本页的配置对象；它们当前有 WMS 菜单入口，但应在 [基础数据](../01-基础数据/index.md) 标注为“当前入口、目标 Infra 归属”的过渡能力。
