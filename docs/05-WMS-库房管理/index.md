## 业务分组

| 分组 | 说明 |
|------|------|
| 01-基础数据 | 客户、销售价格单、科目、产品类、ERP成本中心、ERP项目 |
| 02-系统设置 | 系统日历、账期日历、单据设置、事务类型、计划设置、标签规则 |
| 03-采购收货 | 采购收货（申请+任务+记录）、采购拒收记录 |
| 04-采购退货 | 采购退货（申请+任务+记录） |
| 05-采购上架 | 上架（申请+任务+记录） |
| 06-发料管理 | 补料、发料、高低储配置、备料计划 |
| 07-生产收料 | 生产收料、生产退料、隔离退料 |
| 08-生产管理 | 生产计划、制品收货、完工撤销、隔离收货、制品拆解/返修/上架 |
| 09-库存管理 | 库存查询、预计出/入库存、余额、汇总、事务、转移日志、库位容量 |
| 10-销售出库 | 发货计划、客户预测、销售订单、成品发货、客户退货、备货、结算出库 |
| 11-库内作业 | 调拨、计划外、报废、转库、器具管理 |
| 12-盘点管理 | 盘点计划、盘点任务、盘点调整、差异库存 |
