## 字段业务语义
本页是 **WMS PDA 入口对照与执行检查**，不是独立主档或独立单据对象。字段语义以各 Web 业务维护参考为准；此处只归纳终端侧共性行为模式。

写法约定见[页面数据字典规范](../../02-业务模型/04-页面数据字典规范.md)；选择器通例见[通用选择器过滤惯例](../../02-业务模型/12-通用选择器过滤惯例.md)；库位通例见[库位与仓储级联惯例](../../02-业务模型/13-库位与仓储级联惯例.md)；粒度通例见[库存管理精度与唯一粒度](../../02-业务模型/08-库存管理精度与唯一粒度.md)。

**模式归类：** **P2（任务定位）**、**P3（包装/标签）**、**P4（仓→位校验）**、**P7（实绩数量）**、**P8（扫码主路径）**、**P9（承接/执行/关闭）**。

**本页无 P5 / P6 / P11 / P12：** 不在 PDA 页维护主数据编码规则、不维护默认库位策略、不配置业务类型；来源回填发生在对应 Web 申请/任务对象上。

### 字段业务语义总表（终端共性）
| 字段（中文业务名） | 行为模式 | 业务作用 | 谁在何时维护/选择 | 业务约束摘要 | 影响哪些功能 | 变更或错选风险 |
| --- | --- | --- | --- | --- | --- | --- |
| 任务号 / 单据号 | P2/P8 | 定位待执行任务 | 扫码或查询 | 必须属于当前业务对象族 | 打开正确执行页 | 进错菜单对象族（如生产收料 vs 制品收货） |
| 物料 / 来源信息 | 展示/P5 | 核对对象 | 任务带出 | 现场以任务为准 | 防错发错收 | 手工改任务关键信息（若被禁止） |
| 库位 | P2/P4/P8 | 执行地点 | 扫描或选择 | 须在任务允许仓库范围内 | 余额地点 | 扫错位、跨仓 |
| 包装 / 批次 / 托盘 | P3/P8 | 采集粒度 | 扫描或录入 | 与管理精度一致 | 追溯与余额键 | 漏采无法完成或串料 |
| 实绩数量 | P7 | 完成量 | 扫描累加或确认 | 受计划/可用量/任务配置约束 | 记录与库存事务 | 缺货假装完成 |
| 拒收/异常原因 | P1 | 未完成说明 | 异常时必填（场景相关） | PDA 常强制 | 协同与关闭 | 无原因无法闭环 |
| 任务状态 | P9 | 门禁 | 系统迁移 | 以对应 Web 状态机为准 | 承接、执行、关闭 | 跨对象族套用状态 |

### 选择器范围表（P2）— 终端定位
| 选择字段 | 选择对象 | 可选范围（必须写清） | 范围依赖的前置字段/状态/权限 | 选中后带回或锁定什么 | 选不到时通常原因 |
| --- | --- | --- | --- | --- | --- |
| 待办任务 | 对应业务任务 | 当前 PDA 入口所属对象族 + 可执行状态 + 执行人/权限 ❓ | 菜单入口、用户权限 | 物料、计划量、建议库位等 | 任务未生成、已关闭、进错入口、权限外 |
| 库位 / 标签 | 库位或包装标签 | 任务允许范围内合法对象 | 任务仓库/扫描开关 | 定位或追溯维度 | 范围外、停用、标签无效 |

### 条码与采集关系（P8）
| 条码/维度 | 何时必须采集 | 采集入口（Web/PDA） | 缺失时能否继续 | 对库存唯一粒度的影响 |
| --- | --- | --- | --- | --- |
| 任务号/单据号 | 定位任务时 | PDA 扫/查 | 无法进入正确任务 | — |
| 包装/批次/库位 | 业务要求扫描时 | PDA 主路径；Web 可作补录 | 要求扫描时通常禁止继续 | 余额业务键 |
| 实绩数量 | 完成确认前 | PDA/Web | 不得空完成 | 事务数量 |

### 状态-动作摘要（P9）— 终端侧
| 对象 | 允许动作（入口存在） | 前置检查 | 成功后的下游影响 | 禁止或风险说明 |
| --- | --- | --- | --- | --- |
| 各业务任务（PDA） | 承接、执行、异常/拒收、关闭（以各业务为准） | 任务状态与扫描校验 | 形成记录并驱动库存结果 | 隐藏“直接*”入口不作默认培训路径 |
| 库存结果核对 | 回 Web 查询 | 记录已生成 | 事务/余额/预计清理 | 仅看 PDA 成功不等于账实已闭环 |

## 执行检查清单

### 执行前

| 核对项 | 通过标准 |
| --- | --- |
| 业务入口 | 与 Web 任务类型一致。 |
| 任务可执行 | 状态允许承接/执行，执行人匹配。 |
| 扫描要求 | 已知是否必须扫包装、批次、库位。 |
| 库存条件 | 来源库存状态、冻结、放行情况已理解。 |

### 执行中

| 情况 | 处理 |
| --- | --- |
| 扫描匹配 | 按任务数量/包装继续。 |
| 扫描不匹配 | 停止，核对实物与任务，不绕过。 |
| 数量差异 | 按任务是否允许少做/多做处理；否则回 Web。 |
| 需要拒收/异常 | 填写原因，保留现场证据。 |

### 执行后

| 核对项 | 在哪里看 |
| --- | --- |
| 业务记录是否生成 | 对应 Web 记录页 |
| 库存是否变化 | 库存事务、库存余额 |
| 预计库存是否清理 | 预计入/预计出查询 |
| 是否有后续单 | 上架、质检、协同等联查 |

## 常见终端动作说明

| 动作 | 业务含义 | 注意 |
| --- | --- | --- |
| 定位任务 | 按任务号、单据号或扫描码找到待办。 | 找不到时先回 Web。 |
| 承接 | 将任务交给当前执行人。 | 部分页面进入详情即承接。 |
| 扫描确认 | 校验包装、批次、库位等。 | 以任务配置为准。 |
| 提交完成 | 形成记录并触发后端库存处理。 | 成功提示 ≠ 已完成全部后续协同。 |
| 拒收/异常 | 保留原因并中止正常完成。 | 不要用正常完成掩盖异常。 |

## 查询建议

终端列表适合找“我现在能做什么”；审计和追溯应回到 Web：

| 问题 | 推荐路径 |
| --- | --- |
| 刚才那笔有没有入账 | PDA 提交结果 → Web 记录 → 库存事务 → 余额 |
| 为什么 PDA 不让做 | Web 任务状态/配置 → 库存状态 → 权限 |
| 扫码规则是什么 | 对应 Web 业务维护参考中的任务限制说明 |
