> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[终端操作业务说明](index.md)使用。本页提供 WMS PDA 入口与 Web 业务页对照；业务规则仍以各 Web 页为准。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 找某业务的 PDA 入口 | “Web 与 PDA 对照” |

## 范围边界

| 范围 | 是否本页覆盖 | 说明 |
| --- | --- | --- |
| WMS PDA 作业入口 | 是 | 收货、发料、发运、库内、盘点、生产结果等。 |
| 线边客户端 | 否（仅边界） | 菜单主要指向 MES 前端，规则回 MES。 |
| EAM PDA | 否 | 巡检保养/维修等回 EAM 终端文档。 |
| PDA 模板配置 | 仅提示 | Web 基础设施配置，不定义仓储业务规则。 |

## 执行检查清单

### 执行前

| 核对项 | 通过标准 |
| --- | --- |
| 业务入口 | 与 Web 任务类型一致。 |
| 任务可执行 | 状态允许承接/执行，执行人匹配。 |
| 扫描要求 | 已知是否必须扫包装、批次、库位。 |
| 库存条件 | 来源库存状态、冻结、放行情况已理解。 |

### 执行中

| 情况 | 处理 |
| --- | --- |
| 扫描匹配 | 按任务数量/包装继续。 |
| 扫描不匹配 | 停止，核对实物与任务，不绕过。 |
| 数量差异 | 按任务是否允许少做/多做处理；否则回 Web。 |
| 需要拒收/异常 | 填写原因，保留现场证据。 |

### 执行后

| 核对项 | 在哪里看 |
| --- | --- |
| 业务记录是否生成 | 对应 Web 记录页 |
| 库存是否变化 | 库存事务、库存余额 |
| 预计库存是否清理 | 预计入/预计出查询 |
| 是否有后续单 | 上架、质检、协同等联查 |

## 常见终端动作说明

| 动作 | 业务含义 | 注意 |
| --- | --- | --- |
| 定位任务 | 按任务号、单据号或扫描码找到待办。 | 找不到时先回 Web。 |
| 承接 | 将任务交给当前执行人。 | 部分页面进入详情即承接。 |
| 扫描确认 | 校验包装、批次、库位等。 | 以任务配置为准。 |
| 提交完成 | 形成记录并触发后端库存处理。 | 成功提示 ≠ 已完成全部后续协同。 |
| 拒收/异常 | 保留原因并中止正常完成。 | 不要用正常完成掩盖异常。 |

## 查询建议

终端列表适合找“我现在能做什么”；审计和追溯应回到 Web：

| 问题 | 推荐路径 |
| --- | --- |
| 刚才那笔有没有入账 | PDA 提交结果 → Web 记录 → 库存事务 → 余额 |
| 为什么 PDA 不让做 | Web 任务状态/配置 → 库存状态 → 权限 |
| 扫码规则是什么 | 对应 Web 业务维护参考中的任务限制说明 |
