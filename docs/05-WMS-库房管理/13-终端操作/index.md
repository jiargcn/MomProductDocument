# 终端操作

> 状态：页面结构已建立，具体终端功能待按 WMS 业务链逐项取证。  
> 适用范围：PDA 与线边端中归属于 WMS 的操作入口。  
> 业务规则来源：以对应 WMS Web 页面、后端服务和数据库结构为准。

## BATCH-01 标准占位

> 状态：首轮占位增强。本页是 WMS 终端操作汇总入口，不替代各业务页中的终端执行规则。

终端操作不单独定义业务单据、任务、记录或库存过账规则。PDA/线边端只作为对应 WMS 业务页的执行入口补充；本页负责汇总终端入口、扫码对象、设备能力、离线/异常处理和与 Web 的差异。

## 定位

本页不重复定义采购收货、上架、发料、盘点等业务规则，而是补充终端操作与 Web 操作之间的差异：扫码与标签识别、设备输入、简化录入、离线处理、按钮权限和异常反馈。
