# 终端操作

> 状态：页面结构已建立，具体终端功能待按 WMS 业务链逐项取证。  
> 适用范围：PDA 与线边端中归属于 WMS 的操作入口。  
> 业务规则来源：以对应 WMS Web 页面、后端服务和数据库结构为准。

## 定位

本页不重复定义采购收货、上架、发料、盘点等业务规则，而是补充终端操作与 Web 操作之间的差异：扫码与标签识别、设备输入、简化录入、离线处理、按钮权限和异常反馈。

## 已确认的编写原则

| 内容 | 写入位置 |
| --- | --- |
| 单据、任务、执行记录、库存变化、状态与审批规则 | 对应 WMS 业务页面 |
| PDA/线边端入口、扫描方式、设备能力、终端字段与异常处理 | 本页及对应业务页的“终端操作”小节 |
| 终端页面的访问与操作权限 | 对应业务页及系统管理的 RBAC 说明 |

## 首轮范围

采购收货是首个终端取证场景。待完成 `pda`、线边端前端路由与 WMS 后端接口的对应关系后，本页将按“采购收货—采购上架—发料—盘点”的优先顺序补充终端事实卡。
