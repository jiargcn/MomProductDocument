# 生产管理-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[生产管理业务说明](index.md)使用。本阶段以制品收货为主线；完工撤销、上架、隔离收货、拆解、返修仅作分支说明。发料/生产收料/生产退料不在本页。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 接收生产产出 | “制品收货” |
| 回退已入库生产结果 | “完工撤销” |
| 收货后上架 | “制品上架” |
| 隔离类产出 | “隔离收货” |
| 查库存结果 | “库存影响与追溯” |
| 使用 PDA | “终端执行参考” |

## 页面与业务对象

| 菜单入口 | 当前对象链 | 使用说明 |
| --- | --- | --- |
| 制品收货 | 申请 → 任务 → 记录 | 本分组主链。 |
| 隔离收货 | 申请 → 任务 → 记录 | 复用制品收货对象，业务类型区分。 |
| 完工撤销 | 申请 → 任务 → 记录 | 回退链，不与收货混用。 |
| 制品上架 | 申请 → 任务 → 记录 | 收货后的地点/状态转移。 |
| 制品拆解 | 申请 → 记录 | 无独立任务菜单。 |
| 制品返修 | 申请 → 记录 | 无独立任务菜单；可见性待确认。 |

## 制品收货

| 阶段 | 应维护/确认 | 关键限制 |
| --- | --- | --- |
| 申请 | 生产来源、物料/数量、批次包装、目标状态/地点。 | 来源与产出不一致时不得处理。 |
| 任务 | 执行人、实收数量、库位、库存状态、扫描要求。 | 待检/隔离产出不得按合格库存完成。 |
| 记录 | 实收结果、执行时间与人员。 | 以记录、事务、余额核对是否入库。 |

申请可创建预计入；任务执行生成库存事务并清理预计入。是否自动创建上架、质检或外部接口，取决于规则/开关，不能默认每次都发生。

## 完工撤销

| 阶段 | 应维护/确认 | 关键限制 |
| --- | --- | --- |
| 申请/任务 | 原生产入库结果、撤销原因、可回退范围。 | 已有后续上架/接口时先评估影响。 |
| 记录 | 实际回退结果。 | 核对库存事务是否反向生效。 |

## 制品上架

| 阶段 | 应维护/确认 | 关键限制 |
| --- | --- | --- |
| 申请 | 来源收货结果、目标库位/状态、数量。 | 来源未接收完成前不应上架。 |
| 任务/记录 | 实际移入结果。 | 可形成预计库存并在完成后清理；以事务确认。 |

## 隔离收货、拆解与返修

| 分支 | 操作提醒 |
| --- | --- |
| 隔离收货 | 按隔离/非合格状态接收；不要与合格制品收货入口混用。 |
| 拆解 | 以申请和记录为主；执行前后库存变化需单独核对。 |
| 返修 | 以申请和记录为主；不得套用制品收货状态机。 |

## 列表、详情与快速跳转

| 位置 | 建议优先显示 | 快速跳转 |
| --- | --- | --- |
| 收货申请/任务 | 单号、生产来源、物料/数量、状态、地点。 | 记录、库存、PDA。 |
| 收货记录 | 记录号、实收数量、批次/包装、库位、状态、时间。 | 库存事务、上架、质量。 |
| 撤销/上架 | 来源单号、数量、状态、时间。 | 原收货、库存。 |
| 详情 | 生产来源、物料与数量、执行与状态、库存影响、条件后续、系统信息。 | 申请、任务、记录、库存。 |

## 库存影响与追溯

| 发生时点 | 系统业务结果 | 如何追溯 |
| --- | --- | --- |
| 制品收货申请/任务 | 形成预计入。 | 从任务号查预计入。 |
| 任务执行/记录完成 | 形成库存事务，清理预计入。 | 从记录查事务和余额。 |
| 条件后续 | 可能生成上架、质检或接口信息。 | 从收货记录联查后续单，不默认必然存在。 |

## 终端执行参考

| 场景 | PDA 当前入口 |
| --- | --- |
| 制品收货 | 成品/制品收货任务等入口已定位。 |
| 完工撤销 | 产品回收/完工撤销相关入口已定位。 |
| 制品上架 | 成品/隔离上架相关入口已定位。 |
