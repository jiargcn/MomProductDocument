# 报表管理

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施、运维（主）。
> **本页为配置/菜单入口的导航收敛页。** 平台报表能力的完整业务说明见[报表、看板与数据输出](02-报表、看板与数据输出.md)；请勿在本页维护第二套字段表。

## 本页解决什么问题

读完本页，应能从菜单/配置入口**找到**报表相关能力，并立刻转到平台能力页或模块专项说明；不在此重复模板字段与嵌入细节。

**何时用本页 / 不归本页**

| 何时进来 | 不归本页 |
| --- | --- |
| 只想从菜单找到设计器、报表列表、专项统计入口 | 模板如何维护、业务如何嵌入预览、输出形态 → [报表、看板与数据输出](02-报表、看板与数据输出.md) |
| 区分报表模板与列表 Excel 导出 | 在本页另维护第二套字段表或完整流程 |

## 概述

报表管理入口覆盖：积木报表（JimuReport）模板维护、业务页内嵌预览、报表菜单聚合，以及各模块自有统计页。列表 Excel 导出属于导入导出能力，不在「报表模板」概念内。

## 功能列表（入口）

| 能力 | 说明 | 文档 |
| --- | --- | --- |
| 积木报表设计器 | 维护模板与标识 | [报表、看板与数据输出](02-报表、看板与数据输出.md) |
| 业务内嵌预览 | 业务页带令牌打开模板 | 同上 |
| 模块专项报表 | MES/EAM/WMS 等统计页 | 各模块「报表」分组 |
| 列表导出 | Excel 导出 | [导入、导出与批量操作](04-导入、导出与批量操作.md) |
| 数据权限影响 | 报表可见数据范围 | [数据权限](../12-系统管理/03-用户与权限/02-数据权限与决策权限.md) |
