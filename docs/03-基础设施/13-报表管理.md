## 概述

报表管理入口覆盖：积木报表（JimuReport）模板维护、业务页内嵌预览、报表菜单聚合，以及各模块自有统计页。列表 Excel 导出属于导入导出能力，不在「报表模板」概念内。

## 功能列表（入口）

| 能力 | 说明 | 文档 |
| --- | --- | --- |
| 积木报表设计器 | 维护模板与标识 | [报表、看板与数据输出](02-报表、看板与数据输出.md) |
| 业务内嵌预览 | 业务页带令牌打开模板 | 同上 |
| 模块专项报表 | MES/EAM/WMS 等统计页 | 各模块「报表」分组 |
| 列表导出 | Excel 导出 | [导入、导出与批量操作](04-导入、导出与批量操作.md) |
| 数据权限影响 | 报表可见数据范围 | [数据权限](../12-系统管理/03-用户与权限/02-数据权限与决策权限.md) |
