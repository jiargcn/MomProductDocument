# 公共能力

> 状态：公共能力总入口已建立。具体实现状态待 `dev` 基线核验。

## 页面定位

公共能力提供跨模块复用的运行服务和统一治理规则；业务模块只说明“使用了什么能力、业务数据来自哪里、何时触发”，不重复定义平台规则。

## 能力地图

| 能力 | 说明页面 | 业务侧责任 |
| --- | --- | --- |
| 标签、条码与打印 | [标签、条码与打印](01-标签、条码与打印.md) | 指明标签业务对象、触发时点和业务变量。 |
| 报表、看板与输出 | [报表、看板与数据输出](02-报表、看板与数据输出.md) | 指明数据口径、参数和可钻取对象。 |
| 查询与用户视图 | [查询、筛选与用户视图](03-查询、筛选与用户视图.md) | 指明默认字段、筛选与详情跳转。 |
| 导入导出与批量操作 | [导入、导出与批量操作](04-导入、导出与批量操作.md) | 指明模板业务字段、校验和错误回执。 |
| 文件与附件 | [文件、附件与业务文档](05-文件、附件与业务文档.md) | 指明附件业务用途和保留要求。 |
| 消息与待办 | [消息、通知与待办](06-消息、通知与待办.md) | 指明事件、接收主体和关闭条件。 |

## 归属说明

标签、条码和打印在当前菜单中可能由 DBC 配置入口承载，但其长期合理归属为平台能力与基础设施：DBC 负责提供物料、包装、业务类型等业务数据；平台负责模板、编码、渲染、输出、补打、审计和设备适配。
