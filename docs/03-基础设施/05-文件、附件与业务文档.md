# 文件、附件与业务文档

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施、运维（主）。

## 本页解决什么问题

文件能力统一承接业务附件、导入文件、证据材料等二进制内容的存储、列表、预览、下载与删除。读完本页，应能配置/排查**平台文件服务**，并分清：业务何时强制上传、附件是否参与审批，仍以业务页为准。

**何时用本页 / 不归本页**

| 何时进来 | 不归本页 |
| --- | --- |
| 配存储后端、查文件列表、上传失败先查主配置 | 某单据是否必传附件、展示哪些附件 → 业务页 |
| 区分「文件记录在、内容丢了」 | 导入错误文件联查 → [导入导出](04-导入、导出与批量操作.md)；谁上传/下载 → [日志](07-日志、审计与运行监控.md) |
| 评估换存储/容量/备份 | 把文件列表当成全部业务文档库；MES SOP 等业务文件库细则 → 业务页 |

## 使用前准备

| 需要确认什么 | 为什么重要 |
| --- | --- |
| 主文件配置是否可用 | 上传依赖已启用的文件配置。 |
| 存储类型与容量 | 本地/库表/FTP/SFTP 等策略影响运维与备份。 |
| 业务是否要求必传附件 | 平台能存，不等于业务强制。 |
| 谁能删除文件 | 删除受权限控制，误删影响取证。 |

## 当前能力

```mermaid
flowchart LR
    A["业务页上传"] --> B["文件服务写入存储"]
    B --> C["登记文件元数据"]
    C --> D["预览 / 下载 / 列表 / 删除"]
```

| 能力 | 业务含义 |
| --- | --- |
| 文件配置 | 维护存储方式（如数据库、本地磁盘、FTP、SFTP），可指定主配置并测试连通。 |
| 文件列表 | 管理端分页查询已上传文件元数据。 |
| 上传 | 支持单文件/多文件上传；可携带业务表与业务主键类关联信息。 |
| 预览 / 下载 | 按配置与路径读取内容供预览或下载。 |
| 删除 | 管理端可按权限删除文件记录及存储内容（以实际服务行为为准）。 |
| 业务关联 | 上传时可关联业务对象标识，便于按单据回查附件。 |

## 分工边界

| 角色 | 负责什么 |
| --- | --- |
| 平台（本页） | 存储配置、通用上传下载预览、文件清单治理。 |
| 业务模块 | 在何界面上传、是否必填、展示哪些附件、删除是否允许。 |
| 运维 | 存储容量、备份、主配置切换与连通性。 |

业务用户通常在业务页上传；运维在基础设施维护文件配置与排查文件列表。

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- | --- |
| 上传失败 | 主配置、存储连通、文件大小/类型限制（环境侧）。 | 先修配置再怪业务。 |
| 预览打不开 | 路径/配置是否匹配、文件是否仍存在。 | 区分「记录在、文件丢」。 |
| 附件与单据对不上 | 上传时是否带上业务关联。 | 回业务页重新挂接。 |
| 安全外发 | 下载链路与权限是否满足现场安全要求。 | 敏感文件需额外管控评估。 |

## 建议验证点

- 主文件配置可测试连通；业务页上传后能在文件列表按名称/时间找到，并能预览或下载。
- 带业务关联上传一次，能从单据回查附件；删除权限用非授权账号验证一次拒绝。
- 不默认存在完整文档版本库；业务若另有版本字段，以业务页为准。

## 查询与联查

| 场景 | 建议看什么 | 联查 |
| --- | --- | --- |
| 找不到附件 | 业务页附件区 → 文件列表（按名称/时间）。 | 业务单据。 |
| 存储告警 | 文件配置主节点与存储类型。 | 运维。 |
| 导入错误文件 | 导入日志中的错误文件线索。 | 导入导出页。 |

## 常见问题与处理

| 情况 | 建议处理 |
| --- | --- |
| 换存储后旧文件打不开 | 勿随意切换主配置；迁移需专项方案。 |
| 业务要求版本管理 | 当前不能默认存在完整文档版本库；以业务是否另建版本字段为准。 |
| 以为文件列表等于全部业务文档库 | 许多附件只在业务上下文可见，列表是平台治理视图。 |

## 当前限制与待确认事项

- 病毒扫描、水印、外发审批未证实为通用能力；
- 保留期限与归档策略待运维规范；
- 下载/预览链路的安全加固与网关策略需环境评估；
- SOP 等业务文件库（如 MES）可能另有入口，细则回业务页。
