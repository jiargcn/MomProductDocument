## 编写边界

1. 本模块描述“能力如何提供、如何配置、如何治理”。
2. 具体业务模块描述“业务在何处使用该能力、产生什么数据”。
3. 涉及权限时，链接到[RBAC 权限模型](../12-系统管理/03-用户与权限/01-RBAC权限模型.md)；涉及认证时链接到[租户与认证](../12-系统管理/01-租户与认证/index.md)。
