# 平台能力与基础设施

> 定位：承载跨业务模块复用的平台能力与运行底座；不重复描述 WMS、MES、QMS 等模块的具体业务规则。

## 文档分组

| 分组 | 说明 | 当前优先任务 |
| --- | --- | --- |
| 运行与安全 | 部署、环境、安全、监控与运维。 | 明确环境基线、服务清单、认证、安全边界和运维责任。 |
| 公共能力 | 标签、条码、打印、报表、看板、查询、导入导出、文件和消息等面向业务用户的可复用能力。 | 建立模板、参数、数据来源、权限和业务引用关系。 |
| 基础服务 | 日志、审计、后台作业与数据交换可靠性。 | 形成统一治理规则，供业务页面引用。 |

## 编写边界

1. 本模块描述“能力如何提供、如何配置、如何治理”。
2. 具体业务模块描述“业务在何处使用该能力、产生什么数据”。
3. 涉及权限时，公共规则链接到系统管理的 RBAC 页面；涉及接口时，链接到业务模型的第三方接口挂接页面。

## 长期归属约定

标签、条码与打印属于平台能力与基础设施。即使当前菜单中的配置入口位于 DBC，也只表示业务配置入口暂在 DBC；模板、编码、渲染、输出、设备适配与审计按平台能力维护。
