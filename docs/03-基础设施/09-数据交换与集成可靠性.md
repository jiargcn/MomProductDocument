在[第三方接口与单据挂接](../02-业务模型/03-第三方接口与单据挂接.md)定义业务关联的基础上，本页定义平台级可靠性：身份认证、消息投递、幂等、重试、补偿、对账、告警和日志。

## 2. 参考设计

| 模式 | 需要治理的重点 |
| --- | --- |
| 同步 API | 超时、错误码、事务边界、重复提交和降级。 |
| 异步消息/交换存储 | 事务性写入、投递确认、消费幂等、失败重试、死信/人工补偿与对账。 |
