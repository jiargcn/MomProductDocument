# 监控运维

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 区分**应用内可观察性**（本产品文档已覆盖）与**基础设施监控**（待运维资料）。

## 监控体系

| 层级 | 当前结论 | 入口 |
| --- | --- | --- |
| 应用业务日志 | System / Infra / WMS 等分层查询 | [日志、审计与运行监控](07-日志、审计与运行监控.md) |
| 接口调用与失败 | 调用信息、异步成功/失败、重试 | [数据交换与集成可靠性](09-数据交换与集成可靠性.md) |
| 定时任务健康 | XXL-Job 控制台任务与执行日志 | [定时任务与后台作业](08-定时任务与后台作业.md) |
| MQTT / 采集链路 | 客户端在线与采集日志 | [数采-设备管理](../13-数采管理/02-设备管理/index.md) |

| 业务 KPI 看板 | 报表/模块专项报表，非统一监控平台 | [报表](02-报表、看板与数据输出.md) |

## 日志体系

| 用途 | 建议路径 |
| --- | --- |
| 谁登录、谁操作菜单功能 | System 操作/登录日志 |
| 接口报错、慢调用 | Infra API 访问/错误日志 |
| 导入导出批次 | 导入导出能力页 + 相关日志 |
| 业务单据为何变更 | 业务模块记录/状态字段优先 |

采集、集中检索（ELK 等）是否统一接入：**待运维确认**。

## 应急排查（文档化最小路径）

| 现象 | 先查什么 |
| --- | --- |
| 用户看不到菜单 | 角色菜单、租户、用户状态 → RBAC |
| 能进页不能点按钮 | 权限标识、业务状态前置 → 业务页 + RBAC |
| 接口失败 / 外部系统未入账 | 接口调用信息与重试 → 数据交换页 |
| 定时任务未跑 | XXL-Job 执行器与任务状态 |
| 采集无数据 | 数采配置、MQTT 在线、边缘进程 |
| 库存数量不对 | WMS 事务/余额，勿先改采集或报表 |

详细恢复手册（扩容、切主、回档）属运维 runbook，不在本页编造。

## 限制与待确认
