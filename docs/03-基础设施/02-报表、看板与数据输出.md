# 报表、看板与数据输出

> 状态：占位页已建立。现有报表模块、模板和看板实现待核验。

## 1. 设计目的

统一定义报表、打印输出和看板的口径、权限、刷新、钻取与留痕；报表页面不能只说明样式，必须可追溯到业务数据来源。

## 2. 参考设计线索

旧版规划书将报表与看板列为支撑业务，并提出用户自定义视图和数据可视化需求。当前是否具备模板配置、实时刷新和保存视图，待核验。

## 3. 待核验清单

| 主题 | 需要确认的内容 |
| --- | --- |
| 报表 | 目录、模板、参数、数据集、权限、导出和打印记录。 |
| 看板 | 指标定义、数据来源、刷新频率、延迟、角色权限和钻取。 |
| 输出 | 文件格式、水印、脱敏、批量导出限制和异步任务。 |
