# 计划管理-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[计划管理](index.md)。面向测试、实施、运维的建单/下发/状态动作与排障；订单→执行主线与建议验证点以主文档为准。线边报工见终端操作；库存落地见 WMS。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |

## 字段说明
### 字段说明总表
| 字段 | 业务作用 | 谁在何时维护/选择 | 业务约束摘要 | 影响哪些功能 | 变更或错选风险 |
| --- | --- | --- | --- | --- | --- |
| ERP 转换状态 | 能否转入 MES | 转换动作 | 未转换/已转换 | 生产订单生成 | 重复转换 |
| ERP 锁定 | 防误改 | 锁定动作 | 锁定后变更受控 | 订单维护 | 改已锁单 |
| 生产订单状态 | 指令头门禁 | 系统/维护 | 字典文案 ❓ | 拆单、关闭 | 错误终态 |
| 工单状态 | 生命周期门禁 | 下发/开工等动作 | 见主文档状态表 | 线边执行、WMS 协同 | 非法动作 |
| 工艺路线 | 下发快照来源 | 建单/改单时选 | 可引用路线 | 运行快照、转序 | 缺路线无法下发 |
| 工厂/车间/产线/班次 | 资源定位 | 建单时选 | 有效工厂建模 | 线边、班组 | 资源错位 |
| 任务模式 | 报工/追溯粒度 | 建单时选 | 单件/批量/普通批量 | 终端、追溯 | 粒度不符 |
| 并行组 | 组级动作 | 待下发时建组 | 成员待下发且未分组 | 统一下发/开工 | 半组执行 |
| 计划数量 | 计划产量 | 建单/拆分 | 拆分量可核对 | 完工对比 | 超量难对账 |
| 队列序 | 排产顺序 | 计划层调整 | 已下发可能不回写运行 | 现场顺序认知 | 误以为已改运行序 |

### 选择器可选范围
| 选择字段 | 选择对象 | 可选范围 | 范围依赖 | 选中后带回或锁定 | 选不到时通常原因 |
| --- | --- | --- | --- | --- | --- |
| 工艺路线 | 工艺路线 | 与物料匹配且状态可引用的路线（启停 ❓） | 物料、BOM | 路线名、版本 | 无路线、已停用 |
| 产线 / 工位资源 | 生产线等 | 当前工厂/车间下有效对象 | 工厂、车间 | 名称 | 未建模、停用 |
| 班次 | 班次主数据 | 可用班次 | 工厂策略 | 班次名 | 未维护 |
| 物料 | 物料主数据 | 可用；可制造等用途 ❓ | 订单/工厂 | 单位、名称 | 停用、用途不符 |

### 状态-动作表（P9：工单，摘要）

| 当前状态 | 允许动作 | 前置检查 | 成功后的状态/下游影响 | 禁止时的业务原因 |
| --- | --- | --- | --- | --- |
| 待下发 | 下发/批量下发、作废、建并行组 | 路线/资源齐全；组规则 | 待点检或待开工；写运行工单+快照 | 缺路线/资源 |
| 待点检 | 点检通过类动作 ❓ | 点检完成入口 ❓ | 待开工 | 强制点检未闭合（`MES-PLAN`） |
| 待开工 | 开工 | 并行组可整组 | 生产中 | 条件未齐 |
| 生产中 | 挂起、中止、完工 | 挂起时无执行中工位作业（可能） | 已挂起/已中止/已完成 | 有在制作业禁挂起 |
| 已挂起 | 恢复 | — | 生产中 | — |
| 已完成 | 关闭 | — | 已关闭；联查 WMS 入库 | 无库存勿当完成误解 |
| 已退料 | 查询联查为主 | 与发料回冲 ❓ | — | 勿套用发料状态结论 |

> 报工亦可触发自动完工/关闭；以状态流转日志操作为准。

### 联动 / 回填摘要

| 上游变化 | 下游行为 |
| --- | --- |
| ERP 执行转换 | 生成/关联生产订单 |
| 工单下发 | 写运行工单；复制工艺运行快照 |
| 更换计划路线（未下发） | 下次下发吃新版本 |
| 已下发后改计划队列序 | 可能不更新运行表 |
| 创建并行组 | 组级统一下发/开工等 |

## ERP 订单

| 应维护/确认 | 关键限制 |
| --- | --- |
| 订单号、物料、数量、计划起止、单位、订单类型。 | 与外部系统一致，避免手工改关键识别字段。 |
| 转换状态（未转换/已转换）。 | 已转换勿重复转换。 |
| 锁定状态（未锁定/锁定）。 | 锁定后变更受控。 |
| 确认/已交/废品/返工数量、下达与实际时间。 | 用于进度核对，不以本页替代报表。 |

## 生产工单（计划层）

| 应维护/确认 | 关键限制 |
| --- | --- |
| 工单号、主工单号、来源生产订单号。 | 拆分后保持主从可追溯。 |
| 工厂/车间/产线/班次、物料、数量、单位。 | 下发与线边定位依赖这些字段。 |
| 工艺路线、BOM/版本、任务模式。 | 下发时生成运行快照；模式影响追溯/报工。 |
| 计划起止、队列序、并行组。 | 已下发后改队列序可能不更新运行表。 |
| 完工入仓库/器具（若页面展示）。 | 影响完工入库协同，细则见 WMS。 |

| 动作 | 目的 | 操作提醒 |
| --- | --- |

| 导入/导出 | 批量建单。 | 模板字段以系统为准。 |

## 工单状态动作

| 动作 | 业务结果 | 操作提醒 |
| --- | --- | --- |
| 下发 / 批量下发 | 计划工单进入可执行链，生成运行工单并复制工艺运行快照。 | 仅对待下发等允许状态；缺路线/资源会失败。 |
| 开工 / 手动开工 | 进入生产中（或经待点检/待开工）。 | 并行组可用整组开工。 |
| 挂起 / 恢复 | 暂停或继续生产中工单。 | 存在执行中工位作业时可能禁止挂起。 |
| 中止 | 结束执行（已中止）。 | 评估在制与物料。 |
| 完工 | 标记已完成。 | 也可能由报工自动触发。 |
| 关闭 | 标记已关闭。 | 可能与达量自动关闭并存。 |
| 作废 | 标记已作废。 | 多用于未执行或需作废场景。 |

并行组另有：整组挂起/恢复/中止、统一下发；创建组要求成员待下发且未分组。

## 并行工单组

| 应确认什么 | 为什么重要 |
| --- | --- |
| 组内工单产品/路线是否允许并行 | 创建时有校验。 |
| 组内序号 | 仅待下发组可改序。 |
| 线边布局与扫码路由 | 多产品同组时决定扫到哪张工单。 |

| 动作 | 目的 |
| --- | --- |
| 查候选 / 创建组 / 取消组 | 建立或拆除并行关系。 |
| 更新组内序号 | 调整组内顺序。 |
| 按组 ID / 工单号查组视图 | 前端多窗口识别。 |
| 统一下发、整组开工等 | 组级生命周期动作。 |

## 运行层与联查

| 场景 | 建议查什么 |
| --- | --- |
| 改了计划路线，现场仍旧 | 运行工单绑定的路线快照版本。 |
| 下发后改队列序无效 | 是否已下发；接口会提示不更新运行表。 |
| 状态对不上 | 工单状态流转日志中的操作类型（下发/开工/挂起/恢复/中止/完工/关闭/作废及自动完工类）。 |
| 完工无库存 | 完工记录/完工状态页 → [WMS 生产收料](../../05-WMS-库房管理/07-生产收料/index.md) / [生产管理](../../05-WMS-库房管理/08-生产管理/index.md)。 |
| 任务模式异常 | 运行工单任务类型是否为单件/批量/普通批量。 |

| 能力 | 范围 | 提醒 |
| --- | --- | --- |
| ERP/生产订单导入 | 订单头批量。 | 转换与锁定仍须单独确认。 |

## 最小验收清单

- [ ] ERP 订单：验证转换→执行转换→生产订单可查到来源。
- [ ] 生产订单拆成 ≥2 张工单，数量可核对。
- [ ] 工单绑定路线后下发：出现运行工单；改主数据路线不影响已下发快照。
- [ ] 单工单：下发→开工→挂起→恢复→完工（或报工自动完工）路径可走通。
- [ ] 并行组：创建→统一下发→整组开工；取消组仅待下发可做。
- [ ] 已下发后改计划队列序：行为符合“不回写运行表”的提示口径。
- [ ] 状态流转日志能看到对应操作类型。
