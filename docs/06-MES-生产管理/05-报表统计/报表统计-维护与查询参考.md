# 报表统计-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[报表统计](index.md)。面向测试、实施、运维的查询、导出与联查；记录选型主线与建议验证点以主文档为准。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 想先知道该查哪张记录表 | 返回[报表统计](index.md) |
| 查某人/某工位报工 | “报工记录” |
| 查工单完工与包装 | “完工记录” |
| 查开工点检是否做 | “点检记录 / 上下工记录” |
| 查谁在岗 | “点检记录 / 上下工记录” |
| 完工有数但仓库无账 | “完工记录” → [WMS 生产收料](../../05-WMS-库房管理/07-生产收料/index.md) |
| 查建议验证点 | 返回[报表统计 · 建议验证点](index.md#建议验证点) |
| 从记录跳正逆向 | [追溯管理](../04-追溯管理/index.md) |

## 页面与入口

| 菜单（以环境为准） | 用途 |
| --- | --- |
| 报工记录 / 报工记录（woWsreportLog） | 过程报工流水。 |
| 完工记录 / 完工记录（woCompleteLog） | 完工事实。 |
| 点检记录 | MES 侧点检结果。 |
| 上下工记录 | 线边上下工。 |

若同时出现两套报工/完工菜单，先问实施启用哪套，再培训；不要混用筛选口径。

## 报工记录

| 常用筛选 | 用途 |
| --- | --- |
| 工单号、产品、追溯码、批次 | 定位过程事实。 |
| 工位、工序、报工结果/类型 | 绩效与质量分析。 |
| 任务模式 | 区分单件/批量等。 |

支持分页、高级搜索与导出（以页面为准）。联查：同条件开逆向追溯人员页签。

## 完工记录

| 常用筛选/字段关注 | 用途 |
| --- | --- |
| 工单、产品、批次、包装号 | 定位完工包。 |
| 完工结果/类型、数量 | 产量核对。 |
| 入库位置、库存/线边入库状态 | 判断是否已协同仓储。 |
| 追溯码、产品 SN、物料 SN | 跳转追溯。 |

完工有数但仓库无账：联查 [WMS 生产收料](../../05-WMS-库房管理/07-生产收料/index.md) / [生产管理](../../05-WMS-库房管理/08-生产管理/index.md) 与线边入库任务重试。

## 点检记录 / 上下工记录

| 类型 | 查询要点 |
| --- | --- |
| 点检记录 | 按时间、对象查是否完成；与 EAM 设备点检菜单区分。 |
| 上下工记录 | 按客户端/人员/时间查在岗；用于责任归属。 |

## 最小验收清单

- [ ] 样例工单在报工、完工记录中均可按工单号查到。
- [ ] 报工记录可导出且关键列齐全。
- [ ] 完工记录能看到包装/追溯码线索。
- [ ] 上下工记录能对应本班次上工人员。
- [ ] 培训材料不把积木报表设计器当作本页内容。
