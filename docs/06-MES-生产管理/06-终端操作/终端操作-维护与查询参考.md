# 终端操作-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[终端操作](index.md)。面向班组长培训与现场排障；配置类维护回[基础建模-维护与查询参考](../01-基础建模/基础建模-维护与查询参考.md)。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 登录不上 / 工位不对 | “登录与上下文” |
| 领不到或领错作业 | “领取与开始作业” |
| 报工报错 | “报工” |
| 转不了序 | “转序门闸” |
| 并行多产品 | “并行组现场” |
| 查历史 | “查询联查” |

## 页面与入口

| 入口 | 当前用途 |
| --- | --- |
| 线边客户端（菜单目录 `/mesprocess`） | MES 线边工作台主入口。 |
| 生产报工（Web） | 报工相关 Web 入口；与工位作业 API 并存，以现场实际使用为准。 |
| 线边库存 | 查询线边物料可视信息。 |
| 报工记录 / 上下工记录 | 事后查询。 |
| WMS `pages/...` PDA | **不是**本页范围。 |

## 登录与上下文

| 现场应确认 | 为什么重要 |
| --- | --- |
| 客户端编码是否对应本机 | 绑错机会带错产线/工位。 |
| 账号是否绑定该客户端 | 用户—客户端关系决定能否进入。 |
| 产线、工位集合、当前工序 | 作业池过滤条件。 |
| 是否已上工 | 未上工可能影响责任归属与权限（以实测为准）。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 登录线边端 | 进入执行上下文。 | 失败先查基础建模中的客户端配置。 |
| 上工 / 下工 | 登记在岗。 | 换人必须下工再上工。 |

## 领取与开始作业

| 现场应确认 | 为什么重要 |
| --- | --- |
| 工单是否生产中/可执行 | 计划层未开工则无活。 |
| 分派模式 | 抢单需认领；派单应已指定工位。 |
| 过滤方式（SN / 顺序） | 领取接口按组合定位一条作业。 |
| 作业状态 | 待分派、待执行、执行中等决定可做动作。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 统一领取/定位 | 按模式取一条作业。 | 领不到时核工位绑定与作业池。 |
| 抢单认领 | 待分派 → 待执行。 | 仅抢单模式。 |
| 开始作业 | 进入执行中。 | 开始前确认 SN/工序无误。 |
| 重置为待执行 / 待分派 | 异常回收作业。 | 待分派会清空工位，慎用。 |
| 首工序作业池补槽 | 保持可领作业数量。 | 班组长场景，操作工一般不碰。 |

## 报工

| 模式 | 应使用的报工 | 操作提醒 |
| --- | --- | --- |
| 单件 | 工位单件报工（含 payload / 设备入口）。 | 作业须为执行中；设备入口用设备+工位（可选校验工序）。 |
| 批量 | 工位批量报工。 | 受工艺转序策略影响（累计门槛或按任务立即）。 |
| 普通批量（BULK） | 工单级报工/装箱相关入口。 | 细则见执行侧 BULK 能力；尾箱、待装箱分页属装箱协同。 |

报工结果（OK/NG）影响门闸累计、不良协同与是否自动完工；具体自动完工规则见计划管理状态日志说明。

## 转序门闸

| 应查看 | 含义 |
| --- | --- |
| OK 累计 / 门槛 / 是否放流 | 当前工序能否转入下一工序。 |
| 工艺策略 | 工单累计达标 vs 按任务立即转序。 |

未放流时继续按模式报工，或回工艺配置检查最小合格数量是否过严。

## 并行组现场

| 动作/确认 | 说明 |
| --- | --- |
| 执行布局（多 Tab / 全局扫码） | 同组多工单同时展示。 |
| 追溯码命中工单 | 扫码路由到组内正确产品工单。 |
| 整组挂起/恢复等 | 计划层组动作；现场先处理执行中作业。 |

## 查询联查

| 场景 | 建议路径 |
| --- | --- |
| 报工是否成功 | 报工记录 → 工位作业状态 → 工单状态。 |
| 人员在岗 | 上下工记录。 |
| 线边缺料 | 线边库存 → WMS 发料任务/记录。 |
| 开工卡点检 | 工单状态（待点检）→ 基础建模开工点检模板。 |

## 最小验收清单

- [ ] 客户端登录后工位集合正确，上工成功。
- [ ] 抢单：待分派作业可认领并开始、单件报工成功。
- [ ] 派单：指定工位出现待执行作业并可报工。
- [ ] 批量：门闸累计达到前后转序行为符合工艺配置。
- [ ] 并行组：扫码命中正确工单 Tab。
- [ ] 报工记录与上下工记录可查到本班次样例。
- [ ] 确认未将 WMS PDA 页面当作 MES 终端培训内容。
