# 基础建模-维护与查询参考

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 用途：配合[基础建模](index.md)。面向测试、实施、运维的日常配置与排障；开产主线、写实示例与建议验证点以主文档为准。

## 快速定位

| 你要做什么 | 先看哪里 |
| --- | --- |
| 想先理解开产前要配什么 | 返回[基础建模](index.md) |
| 配线边电脑/终端、查登不上 | “线边客户端” |
| 上传并挂 SOP、查无指导书 | “SOP 文件与挂接” |
| 配工位技能门槛 | “工位技能要求” |
| 配开工前检查、卡在待点检 | “开工点检” |
| 配线边库/料口，或发料对不上线边 | “线边库与料口” |
| 维护工序用料、与工单 BOM 对不上 | “生产 BOM” |
| 查建议验证点 | 返回[基础建模 · 建议验证点](index.md#建议验证点) |

## 线边客户端

| 应维护/确认 | 关键限制 |
| --- | --- |
| 客户端编码、名称、IP/MAC。 | 编码唯一；网络标识用于现场识别。 |
| 产线、工位集合、工序、工人。 | 必须与现场物理布局一致。 |
| 关联系统用户、登录方式、密码（若启用）。 | 决定谁能在该终端登录。 |
| 状态、可用性。 | 停用后不应继续作为开产终端。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 新增/修改/导入 | 建立终端档案。 | 导入后核验工位集合。 |
| 按登录用户查询 | 排查当前账号绑了哪台客户端。 | 一人多终端时先确认现场机位。 |

## SOP 文件与挂接

| 应维护/确认 | 关键限制 |
| --- | --- |
| SOP 编码、名称、文件类型、版本、附件。 | 版本变更保留可追溯；注意生效/失效时间。 |
| 工序 SOP：工序、物料、SOP、文件版本。 | 同工序多物料时分别挂接。 |
| 工位 SOP：工位与 SOP 关系。 | 与工序挂接并存时，现场以实测展示规则为准。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 上传/启停 SOP 文件 | 管理指导书版本。 | 失效旧版前确认无在用挂接。 |
| 维护工序/工位挂接 | 决定现场打开哪份文件。 | 改挂接后到终端验证。 |

## 工位技能要求

| 应维护/确认 | 关键限制 |
| --- | --- |
| 工位编码/名称。 | 必须存在于工位主数据。 |
| 技能编码/名称/类型、等级编码/名称/顺序。 | 引用 DBC 技能与等级。 |
| 状态、可用性、有效期。 | 过期要求不应继续放行上岗。 |

## 开工点检

| 应维护/确认 | 关键限制 |
| --- | --- |
| 点检项内容与分类。 | 项要可执行、可判定。 |
| 模板及与路线/产品的关联。 | 未覆盖的路线可能卡在待点检。 |
| 设备点检项关联（若使用）。 | 与 EAM 点检执行区分职责。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 维护点检项/模板 | 定义开工前必须完成的检查。 | 用一张样例工单验证待点检→待开工。 |
| 关联路线/产品 | 控制模板适用范围。 | 新产品导入时同步加关联。 |

## 线边库与料口

| 应维护/确认 | 关键限制 |
| --- | --- |
| 线边库编码/名称/类型、状态。 | 线边侧识别码。 |
| 对应 WMS 仓库编码/名称。 | 映射错误会导致发料落错仓；库存事务仍以 WMS 为准，见[发料管理](../../05-WMS-库房管理/06-发料管理/index.md)。 |
| 料口编码/名称/类型、关联线边库。 | 投料口与库一致。 |
| 工位—线边库关系。 | 决定工位从哪取料。 |

## 生产 BOM

| 应维护/确认 | 关键限制 |
| --- | --- |
| 产品物料、组件物料、用量/分母、单位。 | 用量错误直接导致投料偏差。 |
| 工序、BOM 编码、版本、层。 | 与工单 BOM 版本对齐。 |
| 关键件、发料类型、项目等扩展。 | 按工厂口径使用，不随意清空。 |

### 常用动作

| 动作 | 目的 | 操作提醒 |
| --- | --- | --- |
| 新增/改版 | 维护工序用料。 | 优先升版，避免覆盖在用版本。 |
| 与工单核对 | 下发前确认版本一致。 | 联查计划工单 BOM 版本。 |

## 最小验收清单

- [ ] 新客户端绑定产线+工位后，终端可登录并看到对应工位作业。
- [ ] 同工序两物料挂不同 SOP，终端展示正确版本。
- [ ] 工位技能要求能挡住不合格上岗（或按项目开关验证）。
- [ ] 开工点检模板覆盖样例路线，工单可走过点检。
- [ ] 线边库 WMS 映射正确，发料/线边库存可联查。
- [ ] 生产 BOM 版本与工单一致，关键件标识符合预期。
