# 页面与数据字典规范

> 状态：占位结构已建立。  
> 目标：让每个页面以同一方式说明新增、编辑、导入、列表、详情和关联跳转，兼顾培训与业务查询。

## 1. 每页必须具备的内容

| 区块 | 必须说明 |
| --- | --- |
| 页面定位 | 解决的业务问题、使用角色、前置与后续影响。 |
| 新增约束 | 字段类型/长度、必填、默认值、选择器数据源与筛选条件。 |
| 编辑限制 | 可编辑字段、不可编辑字段、状态前置条件与并发控制。 |
| 导入规则 | 模板应含/不应含字段、字典转换、错误校验、处理模式。 |
| 数据列表 | 默认字段、显示顺序、查询字段、排序和导出范围。 |
| 详情页 | 分组、字段只读性、关联数据、快速跳转和 Tab。 |
| 证据与差距 | 基线、源码/DDL/测试证据、待验证项和产品差距编号。 |

## 2. 分派完成标准

页面负责人完成后，其他人员无需查看源码，即可判断“能否创建、能否修改、能否导入、会影响什么、如何追溯”。未经证实的内容不得伪装为当前规则。
