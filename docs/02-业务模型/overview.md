# 业务模型

## 领域模型概述

MOM 系统围绕制造运营的核心领域建模，涵盖从订单接收、物料准备、生产执行到成品交付的完整价值链。

## 核心领域

- **主数据域**：物料、BOM、工艺路线、工厂日历等基础数据
- **仓储域**：入库、出库、移库、盘点
- **生产域**：生产工单、工序流转、报工、批次追溯
- **质量域**：检验标准、检验任务、不合格品处理
- **设备域**：设备台账、维修保养、点巡检
- **异常域**：安灯呼叫、异常升级、闭环跟踪
- **供应链域**：采购协同、供应商管理、来料跟踪
- **排程域**：订单优先级、资源约束、排程优化
