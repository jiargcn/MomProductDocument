# 问题响应

> 状态：BATCH-03 首轮占位页。当前页面用于承接 ANDON 异常响应、处理、升级、关闭和复盘内容；具体动作、角色和 SLA 规则待取证。

## 1. 页面定位

问题响应关注异常被谁处理、如何升级、多久必须响应、处理结果如何回写业务对象。它需要与消息通知、看板、EAM 维修、QMS 质量处理和 MES 生产恢复形成闭环。

## 2. 后续取证范围

| 主题 | 需要确认的内容 |
| --- | --- |
| 响应主体 | 责任岗位、责任部门、值班人员、维修人员、质量人员或班组长来源。 |
| SLA 规则 | 响应时限、处理时限、升级规则、超时告警和看板展示。 |
| 处理动作 | 接单、转派、升级、暂停、完成、关闭、复盘、重新打开等动作。 |
| 结果回写 | 是否回写 EAM 维修记录、MES 生产状态、QMS 质量记录或通知消息。 |
| 权限与审计 | 谁能处理、关闭、升级，是否记录原因、附件和处理时长。 |
