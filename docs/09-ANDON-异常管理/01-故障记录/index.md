# 故障记录

> 状态：BATCH-03 首轮占位页。当前页面用于承接 ANDON 异常或故障的发起、记录、分派、响应和关闭入口；具体字段、状态和接口待 ANDON / MES / EAM 源码取证。

## 1. 页面定位

故障记录用于记录现场异常的发生事实，包括设备故障、质量异常、生产异常、物料异常、人员/安全异常等可能类型。它不替代 MES、QMS、EAM 的业务处理记录，而是作为异常发现和协同响应入口。

## 2. 后续取证范围

| 主题 | 需要确认的内容 |
| --- | --- |
| 异常来源 | 人工报障、设备报警、MES 生产异常、QMS 质量异常、EAM 设备异常等来源。 |
| 故障分类 | 异常类型、等级、影响范围、停线/停机标识和责任部门。 |
| 记录字段 | 发生时间、位置、设备、工位、工单、物料、图片附件、描述和上报人。 |
| 状态动作 | 上报、派工、响应、处理、升级、关闭、驳回、撤销等动作是否存在。 |
| 跨模块影响 | 是否触发 EAM 维修、MES 停线、QMS 评审、消息通知和看板展示。 |
