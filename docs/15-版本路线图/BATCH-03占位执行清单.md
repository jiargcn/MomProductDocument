# BATCH-03 占位执行清单

> 状态：已启动。
> 范围：MES + QMS + EAM + ANDON。
> 原则：只建立生产、质量、设备、异常相关业务页的结构和取证任务；不复制 WMS 具体字段、状态、库存规则或终端规则。

## 1. 执行边界

| 项目 | 本批处理方式 |
| --- | --- |
| MES | 保留现有较完整页面，补充终端操作、状态动作、追溯、报工和与 WMS/QMS/EAM/ANDON 的接口占位。 |
| QMS | 保留现有检验配置、来料、生产、客户、评审页面，后续补检验状态、判定、缺陷、评审和放行规则。 |
| EAM | 优先补齐空白分组页：基础数据、设备、备件、工装、巡检保养、终端操作。 |
| ANDON | 优先补齐故障记录、问题响应页面，建立异常协同闭环占位。 |
| 公共模型 | 涉及计划/任务/记录的业务先引用申请任务记录模型，具体差异逐页取证。 |
| 终端操作 | PDA、线边端、工位端作为所属业务页补充，不新建孤立模块。 |

## 3. 每页首轮占位最低输出

| 章节 | 首轮最低要求 |
| --- | --- |
| 页面定位 | 明确页面解决的业务问题、维护角色和被谁引用。 |
| 公共模型引用 | 计划/任务/记录类业务先引用公共模型，页面只写差异和例外。 |
| 字段真实性 | 标记 DDL、DO、DTO、VO、前端配置待校正。 |
| 新增/编辑/导入 | 保留约束、限制和导入模板占位。 |
| 状态与动作 | 保留按钮、状态前置、结果影响和审计表结构。 |
| 跨模块影响 | 标记 MES、QMS、EAM、ANDON、WMS、DBC、System/Infra 的引用关系。 |
| 终端操作 | 有 PDA/线边端/工位端的页面必须补终端入口占位。 |
| 图示与示例 | 保留状态图、流程图、追溯图、截图和业务样例占位。 |
