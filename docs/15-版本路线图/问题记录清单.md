# 问题记录清单

本文用于记录文档梳理过程中发现但暂不阻塞当前批次推进的问题。

## 处理原则

| 类型 | 处理方式 | 示例 |
| --- | --- | --- |
| 不影响继续补占位或结构梳理的问题 | 记录到本页，继续执行可完成部分 | 字段语义待确认、唯一索引缺失、页面字段顺序待前端取证 |
| 会影响当前页面结论但不影响其他页面推进的问题 | 在对应页面保留“待核验 / 待补充”标记，并同步记录到本页 | 状态流转来源冲突、前后端字段名不一致 |
| 会导致大范围返工或不可逆变更的问题 | 暂停并向负责人确认 | 批量改名、目录上移、跨模块归属重划分、删除页面 |
| 涉及生产数据、账号密钥、外部系统写入的问题 | 暂停并确认授权边界 | 连接生产库、读取密钥、修改第三方配置 |

默认策略：不中断优先。只要问题可以隔离记录，就继续推进当前批次中其他可执行工作。
