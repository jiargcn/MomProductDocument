# BATCH-01 占位执行清单

> 状态：已启动。
> 范围：DBC 主数据 + WMS 核心库存链路。
> 原则：只复制已验证的页面结构、取证方法和占位章节；具体字段、状态、权限、导入、终端和接口结论逐页取证。

## 1. 执行边界

| 项目 | 本批处理方式 |
| --- | --- |
| DBC 主数据 | 优先覆盖物料、供应商、客户、仓库、库区、库位、业务类型、单据设置等被 WMS 高频引用的主数据。 |
| WMS 核心库存链路 | 覆盖采购收货、采购退货、采购上架、发料、生产收料、生产管理、库存管理、销售出库、库内作业、盘点。 |
| 公共模型 | 申请、任务、记录类业务引用[申请、任务与记录模型](../02-业务模型/01-申请任务记录模型.md)，业务页只写差异和例外。 |
| 权限取证 | 引用 [RBAC 与动作权限取证模板](RBAC与动作权限取证模板.md)，不直接复制其它页面的权限结论。 |
| 图示与示例 | 先保留图示、截图、示例数据占位；真实截图和测试数据后续单独补。 |

## 3. DBC 后续铺开顺序

| 顺序 | 范围 | 原因 |
| --- | --- | --- |
| 1 | 物料管理下除物料基本信息外的页面 | 与库存、包装、BOM、采购、生产直接相关。 |
| 2 | 供应商、客户、货主、承运商 | 与采购、销售、SCP 和物流协同相关。 |
| 3 | 仓库、库区、库位、库位组、月台 | 与 WMS 库存粒度、上架、发料、出库、盘点直接相关。 |
| 4 | 业务类型、单据设置、单据开关、规则管理 | 影响申请/任务/记录生成、状态策略和库存挂接。 |
| 5 | 车间、产线、工位、工序、工艺路线 | 与 MES、QMS、WMS 生产相关链路相关。 |
| 6 | 设备、工装、模具、点位、AGV 点位 | 与 EAM、ANDON、AGV 和现场执行相关。 |

## 4. 每页首轮占位最低输出

| 章节 | 首轮最低要求 |
| --- | --- |
| 当前占位状态 | 明确本页属于 BATCH-01 首轮占位，旧字段和历史草稿未校正前不得作为当前事实。 |
| 公共模型引用 | 申请/任务/记录页链接公共模型；非单据页说明是否适用。 |
| 字段真实性 | 标记需从 DDL、DO、DTO、前端配置继续校正。 |
| 规则占位 | 新增约束、编辑限制、导入规则、列表字段、详情分组、快速跳转均保留任务。 |
| 动作与权限 | 引用 RBAC 模板，明确前端按钮、后端接口、状态前置条件、数据范围分开取证。 |
| 库存挂接 | 标记本页是否影响预计入、预计出、库存事务、库存余额。 |
| 终端操作 | 有 PDA/线边端可能的业务必须保留终端入口占位。 |
| 图示与示例 | 至少保留流程图、状态图、库存过账示例或截图占位之一。 |
