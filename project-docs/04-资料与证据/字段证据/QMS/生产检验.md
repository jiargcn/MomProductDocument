## 1. 菜单

| 分组 | 申请/任务/记录组件 |
| --- | --- |
| 首件 | `inspectionRequestFirst` / `inspectionJobFirst` / `inspectionRecordFirst` |

## 2. 类型码

`InspectionTypeEnum`：`11` 首件；`12` 末件；`4` 巡检；`3` 过程检验；另有性能等。记录主表含 `productionNumber` 等生产上下文字段。

## 3. 触发

| 来源 | 证据 |
| --- | --- |
| WMS 制品收货 | `ProductreceiptRecordMainService.createInspectRequest` → `inspectionType="4"`；`InspectionRequestApi.createInspectionMain` |
| MES | 模块内未见 `InspectionRequestApi` 调用；NG 映射见 `GAP-071` |

## 4. 回写

记录发布中制品收货回写分支曾注释；现状以联调为准，正式页不宣称固定回写成功。

## 5. 否定旧稿

虚构过程检验 ER、把 EAM 巡检写成 QMS 巡检、宣称 MES 必自动建单 → 废弃。

## 6. 待确认

类型码 `4` 在菜单「巡检」与制品收货触发间的展示过滤；过程检验码 `3` 与「其他」菜单对应关系。
