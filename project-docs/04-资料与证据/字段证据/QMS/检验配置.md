> 对应正式页面：[业务说明](../../../../docs/07-QMS-质量管理/01-检验配置/index.md)、[维护与查询参考](../../../../docs/07-QMS-质量管理/01-检验配置/检验配置-维护与查询参考.md)。

## 1. 菜单

`reference/menu.csv` QMS/`基础配置`：物料检验配置、采样方案、样本字码、AQL、采样过程、动态修改规则、计数器、选定集、检验方法、检验模板、检验方案。另有空节点「质检」。

## 2. 持久化对象

| 业务对象 | 表 | DO |

| AQL | `basic_aql` | `AqlDO` |

| 样本字码 | `basic_sample_code` | `SampleCodeDO` |
| 采样过程 | `basic_sampling_process` | `SamplingProcessDO` |

### 方案关键字段（`InspectionSchemeDO`）

`itemCode`、`inspectionType`、`programmeTemplateCode`、`effectiveDate`/`expirationDate`、`samplePieceSize`、`uom`、`version`、`splitRule`、`aql`、`inspectionLevel`、`dynamicUpdateCode`、`exemption`、`changeInventoryStatus`、`autoCreateRequest`、`generatedMode`、`crontab`、`quantity`、`mesProcessCode`。

## 3. 相关枚举

`InspectionLevelEnum`、`InspectionSeverityEnum`、`SamplingTypeEnum`、`EvaluationModeEnum`、`DynamicLevelModificationEnum`、`InspectionCharacteristicsEnum`（定量/定性）、`SplittingRuleEnum` 等。

## 4. 否定旧稿

旧稿虚构 `SamplingPlan`/`InspectionScheme` 英文字段 ER、臆造采样类型英文状态 → 废弃。
