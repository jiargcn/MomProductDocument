# System：系统参数、字典与个性化配置证据

> 对应正式页面：[系统参数、字典与个性化配置](../../../../docs/12-系统管理/07-系统参数、字典与个性化配置.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 字典（已证实）

| 对象 | 证据 | 结论 |
| --- | --- | --- |
| 字典类型 | `system_dict_type` / `DictTypeDO`：name、type、status、classes、remark | 类型维。 |
| 字典数据 | `system_dict_data` / `DictDataDO`：label、value、dictType、sort、status、colorType、cssClass | 选项维。 |
| 菜单 | `system/dict` 字典管理 | System 入口。 |
| API | `DictTypeController`、`DictDataController`；App 端字典数据接口 | 前后端可取。 |

## 2. 参数（已证实）

| 对象 | 证据 | 结论 |
| --- | --- | --- |
| 配置 | `infra_config` / `ConfigDO`：category、name、configKey、value、type、visible、remark | Infra 配置管理。 |
| 菜单 | `infra/config` | 见开发平台/本页。 |

## 3. 个性化（已证实/边界）

| 项 | 证据 | 结论 |
| --- | --- | --- |
| 用户页面配置 | `UserMenuConfigController` 按 userId+component 保存/查询 | 用户级页面配置存在。 |
| 多语言 | 菜单 `language-configuration` | 入口存在；覆盖范围待测。 |
| 全站共享视图中心 | — | **未证实**。 |
