# 承运商字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的输入与导入范围

| 信息分组 | 当前可维护/导入内容 |
| --- | --- |
| 基本识别 | 代码、名称、简称、类型。代码和简称必填。 |
| 联系与地址 | 地址、国家、城市、电话、传真、邮编、联系人。 |
| 结算信息 | 银行、币种、税率。 |
| 生效控制 | 生效/失效时间、备注、可用状态；可用状态必填。 |

## 当前证据结论

- 当前存在覆盖上述字段的导入模型；类型和币种采用受控口径转换。
- 导入对象注释误称“载具导入”，但类和字段内容属于承运商；正式页面按承运商语义说明。
- 未发现可据此确认的代码唯一性、类型/币种选择范围、税率合法性或已引用后的修改/停用保护。
