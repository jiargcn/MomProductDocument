# DBC：生产线物料关系字段证据

> 对应正式页面：[生产线物料关系管理](../../../../docs/04-DBC-主数据管理/01-物料管理/06-生产线物料关系管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/01-物料管理/10-生产线物料关系管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 新增 | 页面要求产线、物料和可用状态；保存校验产线、物料存在性，并按物料资料回填物料类型。 | 产线是否必须为可用状态；不同入口允许选择的完整物料范围。 |
| 关系保护 | 服务按产线+物料查重并排除自身；数据库层未确认有对应组合唯一保护。 | 并发新增和旁路写入的保护效果。 |
| 页面范围 | 主页面选择器过滤可用的 CCP/BCP 物料；另有原材料、产成品的分类查询/导出入口。 | 原材料入口的新增/编辑界面、权限和筛选条件。 |
| 编辑与状态 | 页面通常锁定产线、物料；支持启用、禁用，状态操作有单独权限控制。 | 删除前的 MES/WMS 引用检查，及旁路修改关联字段的影响。 |
| 导入 | 模板含产线、物料、物料类型、可用状态、有效期、备注；产线、物料、可用状态为必填。 | 仅新增/仅更新模式的错误回执完整性，以及物料类型是否应从用户模板移除。 |
| 下游线索 | 可按生产线查询关联物料与 BOM 版本，支持车间到产线的筛选。 | 与工单创建、发料、线边库存的自动挂接是否存在。 |

## 待核验项

| 编号 | 问题 | 影响 |
| --- | --- | --- |
| DBC-PLI-001 | 产线+物料的组合唯一性主要由服务层实现，数据库并发保护未确认。 | 重复关系可能影响按产线查询和导入结果。 |
| DBC-PLI-002 | 完工收货库位、原料库位虽存在资料线索，但主页面相关配置未启用。 | 不应在培训或模板中当作当前可维护规则。 |
| DBC-PLI-003 | 原材料/产成品视图与主页面的物料筛选范围、维护入口尚未完成测试验证。 | 不同岗位可能维护或查询到不一致范围。 |
| DBC-PLI-004 | 页面尚未确认有详情分组、物料/产线/BOM/工单快捷联查；删除影响未验证。 | 查询效率与变更风险控制不足。 |
