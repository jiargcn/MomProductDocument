# 设备台账字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的维护/导入范围

| 信息分组 | 当前可维护或导入内容 |
| --- | --- |
| 身份分类 | 设备编号、名称、型号、出厂编号、类型、产权。 |
| 现场与责任 | 存放位置、使用部门、责任人、工厂/车间/产线/工序/工位。 |
| 采购制造 | 供应商、生产商、采购/出厂/验收日期、价格、寿命。 |
| 状态与运行参考 | 状态、运行/保养时长、检验、故障参考指标、可用状态。 |

导入模型标记名称和车间必填；主输入模型的必填标识与导入模型不完全一致，需以实际页面/接口验证为准。
