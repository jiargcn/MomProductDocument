# DBC：月台字段证据

> 对应正式页面：[月台管理](../../../../docs/04-DBC-主数据管理/04-工厂建模/05-月台管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/04-工厂建模/18-月台管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 维护字段 | 编码、名称、城市、地址、类型、仓库、默认库位、联系人、状态和有效期。 | 默认库位是否必须属于所选仓库。 |
| 导入 | 模板要求编码、名称、类型、仓库、默认库位、可用状态，并支持联系信息。 | 导入模式、跨仓关系拦截和错误回执。 |
| 下游 | 月台与收货、发运、车辆交接等业务有关。 | 是否自动带入收发任务、预约和车辆信息。 |
