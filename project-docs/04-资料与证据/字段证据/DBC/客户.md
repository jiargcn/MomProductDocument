# DBC：客户字段证据

> 对应正式页面：[客户](../../../../docs/04-DBC-主数据管理/03-客户管理/01-客户.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 新增 | 页面要求代码、名称、简称；服务创建校验代码、名称、简称和可用状态。 | 更新时能否修改代码、名称、简称。 |
| 重复 | 创建服务分别检查代码、名称和简称是否重复。 | 数据库唯一约束及并发建档保护。 |
| 导入 | 代码、简称、可用状态不可为空；名称、简称重复会产生错误。 | 模板完整列、导入模式和回执体验。 |
| 列表 | 页面支持代码、名称、类型等查询，展示地址、联系信息、业务属性、状态和有效期。 | 默认排序和详情分组。 |
| 引用 | 客户作为客户物料、客户月台及交付业务的主体。 | 停用/删除后对订单、发运和选择器的实际影响。 |
