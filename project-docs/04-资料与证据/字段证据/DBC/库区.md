# DBC：库区字段证据

> 对应正式页面：[库区管理](../../../../docs/04-DBC-主数据管理/04-工厂建模/02-库区管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/04-工厂建模/15-库区管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 维护字段 | 编码、名称、仓库、类型、状态、有效期；另有管理精度、部门等资料。 | 前端实际展示/必填范围与服务校验的一致性。 |
| 导入 | 模板要求编码、仓库、类型、可用状态，支持部门和有效期。 | 仓库存在性、部门有效性及导入模式的完整校验。 |
| 业务关系 | 库区归属仓库，并为库位、库存和作业提供地点层级。 | 库区类型是否强制限制库存状态、物料或任务策略。 |

## 待核验项

| 编号 | 问题 | 影响 |
| --- | --- | --- |
| DBC-AREA-001 | 库区类型、管理精度与实际作业策略的自动关联未确认。 | 不应仅凭库区名称推断强制隔离或上架限制。 |
| DBC-AREA-002 | 变更/删除前对库位、库存和任务的引用保护待测试。 | 仓储布局调整存在操作风险。 |
