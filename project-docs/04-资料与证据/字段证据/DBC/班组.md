# 班组字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的输入与导入范围

| 业务信息 | 新增/编辑要求 | 导入模板情况 |
| --- | --- | --- |
| 班组代码 | 必填。 | 必填。 |
| 名称、描述、成员 | 名称、描述、成员可维护。 | 名称必填；成员和描述当前不在模板中。 |
| 车间、生产线 | 可维护。 | 两者均必填。 |
| 班组负责人编码、姓名 | 可维护。 | 两者均必填。 |
| 班组类别 | 可维护。 | 必填。 |
| 生效/失效时间、备注、是否可用 | 可维护。 | 均可导入，其中可用状态必填。 |

## 当前证据结论

- 班组关联车间、生产线和负责人，并保留成员、类别与有效期信息。
- 导入模板刻意不包含成员；成员的后续维护入口与批量维护方式未确认。
- 当前未发现可据此确认的负责人/成员是否必须存在、车间与生产线从属关系校验，以及停用引用保护。
