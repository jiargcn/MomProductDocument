# WMS：盘点管理字段证据

> 对应正式页面：[业务说明](../../../../docs/05-WMS-库房管理/12-盘点管理/index.md)、[维护与查询参考](../../../../docs/05-WMS-库房管理/12-盘点管理/盘点管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 对象关系

| 业务对象 | 数据/服务对象 | 证据结论 |
| --- | --- | --- |
| 盘点计划 | `plan_count_*`，`CountPlanMainService` | 计划层。 |
| 盘点申请/任务/记录 | `request/job/record_count_*`，`Count*Service` | ATR 齐全；读余额作账面基准。 |
| 盘点对比 | `count_compare_detail` 等，`CountCompare*` | 账面 vs 实盘。 |
| 盘点调整 | `request/record_countadjust_*`，`Countadjust*Service` | 申请+记录；无 Job 菜单。 |
| 差异库存 | `Difference*` / `*_difference_*` | 旁路对象；与现行主链关系见 GAP-043。 |

## 2. 已证实库存挂接

| 环节 | 结论 |
| --- | --- |
| 盘点任务/记录 | 未见直接 `createTransaction` / `createExpect*`。 |
| 盘点调整 | `transactionService.createTransaction`（正负数量入/出）。 |
| DIFF 差异库清账 | 相关事务代码整段注释；不能写成现行标准。 |

## 3. 菜单与终端

- Web：`wms/countManage/count/*`、`countadjust/*`。
- PDA：`pages/count/job/countJob`。
