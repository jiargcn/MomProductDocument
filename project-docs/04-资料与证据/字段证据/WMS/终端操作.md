## 1. 范围证据

| 来源 | 结论 |
| --- | --- |
| `menu.csv` PDA 根菜单 | path 形如 `pages/<biz>/...`，前端落在 `sourcecode/frontend/pda/src/pages`。 |
| 线边客户端根菜单 | component 多为 `/mes/*`，属 MES 线边端。 |
| PDA 模板配置 | Web `/infra/pdaConfig`，属基础设施配置。 |

## 2. 已定位 PDA 目录（WMS 相关）

`purchaseReceipt`、`purchaseReturn`、`putaway`、`issue`、`productionReceipt`、`productionReturn`、`productReceipt`、`productPutaway`、`productRecycle`、`deliver`、`stockUp`、`customerReturn`、`inventoryMove`、`transfer`、`unPlanned`、`scrap`、`count`、`container`、`fg` 等。

## 3. 行为边界

- 终端不另建库存对象族；库存结果以对应 Web/后端服务为准。
- “直接*”类入口多存在可见性限制，不能默认培训。
- 采购收货 PDA 已有较完整取证；其余业务多为菜单定位级证据。
