# WMS 采购退货：字段与行为证据

> 内部证据页；适用基线：测试环境 / `dev` 分支 / 2026-07-15。不得从正式业务文档直接链接本页。

## 已定位证据

- 申请、任务、记录及各自明细控制器齐备；另有撤销记录主控制器。
- 前端存在申请、任务、记录和撤销记录四个独立入口。
- 申请存在常规、跨月、备件、制造订单类型等导入对象；任务明细存在专用执行请求对象。

## 已确认事实

| 业务能力 | 当前事实 |
| --- | --- |
| 业务链 | 采购退货使用申请、任务、记录三层对象，并保留撤销记录。 |
| 批量资料 | 申请支持多类导入对象；具体模板与前端启用情况待验证。 |
| 现场执行 | 任务层存在执行请求对象，需继续核实状态前置条件和终端入口。 |

## 待确认问题

- WMS-RET-001：申请提交/审批、任务执行、记录生成及撤销的真实状态与动作前置条件。
- WMS-RET-002：退货可退数量、已上架/检验后的退货控制及库存扣减时点。
- WMS-RET-003：采购订单、供应商协同和外部系统回写范围。
