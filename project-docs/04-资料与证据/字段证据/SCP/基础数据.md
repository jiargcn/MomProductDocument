# 字段证据：SCP / 基础数据

> 对应正式页面：[业务说明](../../../../docs/10-SCP-供应链平台/01-基础数据/index.md)、[维护与查询参考](../../../../docs/10-SCP-供应链平台/01-基础数据/基础数据-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单与前端

- 主菜单不在 MOM 主站 `reference/menu.csv` 的 WMS 树下完整展开；SCP 使用独立前端 `sourcecode/frontend/scp-ui`，视图根 `src/views/scp/basicDataManage/`。
- 典型路径：`supplierManage/supplier`、`supplieritem`、`purchaseprice`、`itemManage/*`、`factoryModeling/*`、`documentSetting/switch`、`strategySetting`、`supplierCycle`、`systemSetting/supplierUser`、`labelManage/*`。
- 菜单迁移脚本：`scp-ui/sql/fix_system_menu_scp_permissions_and_claim.sql`（`wms/`→`scp/` component、`wms:`→`scp:` permission）。

## 2. 关键对象（表名仅证据层）

| 业务对象 | 表 | 关键字段（业务含义） |
| --- | --- | --- |
| 供应商 | `scp_basic_supplier` | 编码/名称、联系、可用期 |
| 供应商物料 | `scp_basic_supplieritem` | 供应商+物料、包装相关 |
| 采购价格 | `scp_basic_purchaseprice` | 供应商、物料、币种、价格、生效期 |
| 物料 | `scp_basic_itembasic` | 物料主数据副本 |
| 物料包装 | `scp_basic_itempackage` | 包装 |
| 包装规格/层级 | `scp_basic_packageunit` / `scp_packages_hierarchy` | 规格 |
| 仓库/库区/库位组/库位/月台 | `scp_basic_*` / `scp_basic_dock` | 协同侧仓储 |
| 单据开关 | `scp_document_switch` | code、effectiveSetValue |
| 策略 | `scp_strategy_strategy` | 策略配置 |
| 供应商周期 | `scp_basic_supplier_cycle` | 预测周期 |
| 供应商用户 | `scp_basic_supplier_user` | 用户-供应商绑定 |
| 标签/条码/模板 | `scp_label_*` | 标签能力 |
| 接口信息 | `scp_basic_interface_info` | SCP-WMS 等登记 |
| 成本中心 | QAD 成本中心 DO | 注释：只读，来源 WMS |

## 3. 边界

- 与 DBC `basic_supplier` / `basic_supplieritem` 等**并存**；同步方向未在本轮闭合。
- 标签/包装可经 `ScpWmsInterfaceSyncService` 推送 WMS。
