# 字段证据：SCP / 发票结算

> 对应正式页面：[业务说明](../../../../docs/10-SCP-供应链平台/07-发票结算/index.md)、[维护与查询参考](../../../../docs/10-SCP-供应链平台/07-发票结算/发票结算-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单与前端

- `supplierManage/supplierinvoice/*`、`purchaseclaim/*`、`supplierApbalanceMain`、`supplierApbalanceCalendar`、`invoicingcalendar`、已开票排程/离散视图。
- 索赔菜单迁移：`fix_system_menu_scp_permissions_and_claim.sql`（claim 挂到 SCP 根 id=4792）。

## 2. 关键对象

| 业务对象 | 表 | 关键字段 |
| --- | --- | --- |
| 发票申请 | `scp_request_supplierinvoice_*` | status、autoCommit/Agree/Execute 等 |
| 发票记录 | `scp_record_supplierinvoice_*` | 结果明细 |
| 已开票 | `scp_qad_supplierinvoice_invoiced` | 外部已开票 |
| 索赔申请/记录 | `scp_request/record_purchaseclaim_*` | poNumber/poLine、reason、关联发票申请号等 |
| 对账主 | `scp_export_supplier_apbalance_main` | 对账导出/主数据 |
| 对账日历 / 开票日历 | `scp_basic_supplier_apbalance_calendar` / `scp_basic_invoicingcalendar` | 周期 |

## 3. 枚举

- `SupplierinvoiceRequestStatusEnum`：NEW、REVIEWING、AGREED、REFUSED、CLOSED、HANDLING、PARTIAL、COMPLETED、ABORT。
- 通用 `RequestStatusEnum` 另含 INVOICING、ABROGATE 等（跨单据复用，具体按钮以页面为准）。

## 4. 边界

- 付款/总账不在 SCP；对账推送 ERP **未证实**。
- System 证据：SCP 对 `/admin-api/bpm/**` 返回模块禁用——不能写全站审批中心必开。

## 5. 待确认

- 发票与索赔轧差；财务回写协议与失败补偿。
