# 字段证据：SCP / 采购订单

> 对应正式页面：[业务说明](../../../../docs/10-SCP-供应链平台/02-采购订单/index.md)、[维护与查询参考](../../../../docs/10-SCP-供应链平台/02-采购订单/采购订单-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单与前端

- `scp-ui`：`purchasereceiptManage/supplierdeliver/purchaseMain`；离散订单 `supplierManage/purchaseDiscreteOrder/purchaseDiscreteOrderMain`。
- 控制器：`PurchaseMainController` / `PurchaseDetailController`。
- 首页统计：`IndexController` `/scp/index/indexSupplier` → `openPurchaseCount` / `allPurchaseCount`。

## 2. 关键对象

| 业务对象 | 表 | 关键字段 |
| --- | --- | --- |
| 订单头 | `scp_order_purchase_main` | number、supplierCode、type、status、orderDate、dueDate、taxRate、isConsignment、businessType、dataSource、erpOrderType |
| 订单行 | `scp_order_purchase_detail` | lineNumber、itemCode、orderQty、shippedQty、receivedQty、returnedQty、putawayQty、singlePrice、amount、overReceivingPercent、detailStatus |

## 3. 枚举

- `OrderTypeEnum`：SCHEDULE、DISCRETE。
- `OrderStatusEnum`：READY、PUBLISHED、CLOSED、COMPLETED。
- 接口类型：`ORDER_PURCHASE_MAIN` / `ORDER_PURCHASE_DETAIL`。

## 4. 边界

- WMS 菜单亦有「采购订单」`purchaseMainWms`；与 SCP 表是否同实例未证实。
- 可向 WMS 推送订单主/明细（接口类型枚举）。

- 发布后变更再确认规则；ERP 回写协议。
