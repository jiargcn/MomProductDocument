# 字段证据：EAM / 备件管理

> 对应正式页面：[业务说明](../../../../docs/08-EAM-设备管理/03-备件管理/index.md)、[维护与查询参考](../../../../docs/08-EAM-设备管理/03-备件管理/备件管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单（摘要）

备件台账、领用申请、备件出库/入库及记录、盘点计划/工单/差异/调整、物品入出库与换位、备件事务；以及工单侧备件申领/出库/归还/入库。PDA 备件申领/出库/归还/入库。

## 2. 库存模型

- EAM 备件台账 `basic_eam_item_accounts`：物料号、库区/库位、数量、单价、安全库存、科目等——**EAM 侧台账/结存视图**。
- 出入库申请/记录主从表：`request_spare_parts_*` / `record_spare_parts_*`。
- **与 WMS**：出库同步 `UnplannedissueServiceApi.sparePartUnplannedissue`；入库同步 `UnplannedreceiptServiceApi.sparePartUnplannedreceipt`。明细同步状态枚举：明细 / 已同步WMS / 同步失败。

## 3. 边界结论

EAM 发起备件业务动作并维护模块内台账；**仓库库存事务与余额权威在 WMS**（经非计划发料/收货接口回写单号）。不得写成「EAM 独立库存可替代 WMS」。

## 4. 待确认

- 所有备件路径是否强制同步 WMS，或仅工单备件路径同步。
- EAM 库位与 WMS 库位编码映射规则。
- 盘点调整是否同样回写 WMS。
