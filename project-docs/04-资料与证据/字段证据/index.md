## 共用证据来源

| 类型 | 位置 | 用途 |
| --- | --- | --- |
| DDL | `reference/dbddl/base.sql`、`reference/dbddl/wms.sql` | 表、列、类型、长度和注释。 |
| 后端对象 | `sourcecode/backend/win-module-dbc`、`sourcecode/backend/win-module-wms` | DO、DTO/VO、服务规则和导入模型。 |
| 前端页面 | `sourcecode/frontend/ui`、`sourcecode/frontend/pda` | 列表、表单、选择器、按钮、导入和终端行为。 |
| 测试环境 | 2026-07-15 对应测试基线 | 验证状态、权限、截图和端到端行为。 |

## 维护边界

- 正式 `docs/` 只写可培训、可查询的业务结论；不直接链接本目录。
- 字段技术名必须由本目录或同等证据链确认，不能由中文标签反推。
- 未验证的状态机、权限边界和跨模块闭环，在证据页明确标注并进入问题记录。
