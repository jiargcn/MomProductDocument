# Infra：文件、附件与业务文档证据

> 对应正式页面：[文件、附件与业务文档](../../../../docs/03-基础设施/05-文件、附件与业务文档.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 对象与菜单

| 项 | 证据 | 结论 |
| --- | --- | --- |
| 文件元数据 | `infra_file` / `FileDO`：`configId`、`name`、`path`、`url`、`type`、`size`、`tableName`、`tableId` | 可关联业务表+主键。 |
| 文件配置 | `FileConfigController`；菜单 `infra/fileConfig` | 存储配置、主配置、测试。 |
| 文件列表 | `FileController` `/page`；菜单 `infra/file` | 管理端查询。 |

## 2. 能力

| 能力 | 证据 |
| --- | --- |
| 上传 | `/upload`、`/uploadLine`；`WinFileController` 多文件；App 端上传 |
| 删除 | `/delete` + `infra:file:delete` |
| 下载/预览 | `/{configId}/get/**`、`/view/**`（`@PermitAll`） |
| 存储类型 | `FileStorageEnum`：DB / LOCAL / FTP / SFTP |

## 3. 安全注意

下载/预览接口标注允许匿名访问设计，正式页仅提示需结合网关与路径保密做环境评估，不展开利用细节。
