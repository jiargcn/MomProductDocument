# Infra：定时任务与后台作业证据

> 对应正式页面：[定时任务与后台作业](../../../../docs/03-基础设施/08-定时任务与后台作业.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 调度形态（已证实）

| 项 | 证据 | 结论 |
| --- | --- | --- |
| 应用内 Job API | `JobController.getJobPage` 返回错误提示改用 XXL-Job | 不以应用内嵌任务 CRUD 为主。 |
| 菜单 | `定时任务` 外链至 `u-job-admin/toLogin`（环境 IP 可变） | 跳转独立调度控制台。 |
| 仓库内调度组件 | `sourcecode/backend/xxljob` | 工程含 XXL-Job 相关包。 |
| Handler 注册 | 大量 `@XxlJob("...")`：WMS（补料/发料/盘点/清理/回冲等）、Infra（访问/错误日志清理、异步失败重试等） | 作业在代码中注册，由控制台调度。 |
