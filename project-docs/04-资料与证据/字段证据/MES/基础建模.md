## 1. 菜单证据（节选）

| 菜单 | 组件/路径 | 结论 |
| --- | --- | --- |
| 线边端配置 | `mes/lineclientinfo/index` | MES 客户端主配置 |
| SOP 文件 | `mes/basicsopinfo/index` | SOP 主文件 |
| 工序 SOP 配置 | `mes/processsopcfg/index` | 工序挂接 |
| 工位 SOP 配置 | `mes/workstationSopCfg/index.vue` | 工位挂接 |
| 工位技能要求 | `mes/workstationSkillCfg/index` | MES 消费技能 |

## 2. 关键 DO

| 业务 | 表 / DO | 关键字段 |
| --- | --- | --- |
| 线边客户端 | `mes_line_client_info` / `LineClientInfoDO` | `clientCode/Name`、`clientIp/Mac`、`lineCode/Name`、`workStations`、`processCode`、`workerCode`、`sysUserId`、`loginType`、`clientPwd`、`status`、`available` |
| SOP 文件 | `mes_basic_sop_info` / `BasicSopInfoDO` | `sopCode/Name`、`fileType`、`fileVersion`、`attachPath/Name`、`effectTime/invalidTime`、`status`、`groupType`、`attatchMd5` |
| 工序 SOP | `mes_process_sop_cfg` / `ProcessSopCfgDO` | `processCode/Name/Type`、`materialCode/Name`、`sopCode/Name`、`fileVersion/Type`、`attachPath` |
| 工位技能 | `mes_workstation_skill_cfg` / `WorkstationSkillCfgDO` | `workstationCode/Name`、`skillCode/Name/Type`、`levelCode/Name`、`levelOrders`、`status`、`available` |
| 线边库 | `mes_store_info` / `StoreInfoDO` | `storeCode/Name/Type`、`wmsStoreCode/Name`、`status` |
| 料口 | `mes_feedport_info` / `FeedPortInfoDO` | `portCode/Name/Type`、`lineStoreCode`、`toLineStoreCode/Name`、`status` |
| 线边库存信息 | `mes_store_line_info` / `StoreLineInfoDO` | 库/物料/批次/包装/数量等可视字段 |
| 生产 BOM | `mes_item_bom_work` / `ItemBomWorkDO` | `productItemCode`、`componentItemCode`、`componentQty`、`processCode`、`bomCode`、`version`、`isKeyMaterial`、`issueType` 等 |

开工点检：`checkprojectitem`、`checkprojecttemplate`、`startworkcheck*` 等包存在；模板可关联路线（`startworkchecktemplatetoprocroute`）。
