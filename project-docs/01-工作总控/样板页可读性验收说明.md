# 样板页可读性验收说明

> 状态：待产品负责人评审。
> 范围：第一阶段第一个目标——业务化表达标准、物料基本信息样板、采购收货样板。

## 1. 本次验收结论

两个样板页已按“正式页讲业务，内部页留证据”的标准重构，可作为 DBC/WMS 第一阶段的文风和站位样板。它们不再以 DDL、对象映射或源码调用链开篇，而是优先回答业务目的、角色、操作流程、状态动作、影响、查询与异常。

本说明不代表第一阶段全部完成；它只验证第一阶段的第一个目标。未经产品负责人确认，不得据此开始 DBC/WMS 全量大纲铺设或第二阶段扩写。

## 3. 建议评审方式

请以以下三个问题评审，而不是以字段数量评审：

1. 一个未接触过系统的人，能否在三分钟内说出“物料基本信息”和“采购收货”各自解决什么问题？
2. 培训人员能否只依赖流程图、表格和示例站位，讲清一笔业务如何从开始走到结果？
3. 需要查规则时，是否能快速找到新增/修改、现场执行、异常、查询和后续影响，而不必先阅读技术实现？
