## 3. 准入闸门处理结果

### GATE-001：隔离历史技术正文（已完成）

**原问题：** DBC/WMS 的大量叶页虽已新增业务大纲，但原有 DDL、DO、DTO、VO、Controller、Service、事实卡、BATCH 等技术正文仍在正式 `docs/` 页面中。读者继续向下阅读仍会进入研发底稿式内容，违背“培训材料 + 业务字典”的目标。

**处理结果：** 已将正式页统一重写为业务目的、典型流程、关键维护/执行、查询联查、常见问题、已知限制和可分派素材任务；技术字段名、源码路径、表名和证据冲突不再作为正式正文出现。

**完成核验：**

1. DBC 36 个非 P0 叶页、WMS 12 个非 P0 分组首页均已完成业务化重写；
2. P0 物料与采购收货样板未被机械改写，继续作为正式页质量基准；
3. 技术标记扫描结果：DBC 47/47 页、WMS 15/15 页均为 0 个命中；
4. 本批提交前仍须执行 `git diff --check` 和 `mkdocs build`；
5. 后续新增或回填页面不得把技术事实卡和 BATCH 取证段落带回正式页。
