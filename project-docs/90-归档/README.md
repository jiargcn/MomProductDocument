# 归档（过程记录）

> **【归档区】** 默认不必日常关注。仅复盘、追「当时为什么这么做」、或核对历史证据时查阅。  
> 每个归档文件文首均带有「归档文件」标记。

日常入口：[`../00-需关注/`](../00-需关注/README.md)

## 使用规则

1. 不要从归档文件复制字段表、状态机或权限结论到正式 `docs/`。
2. 需要现行规则时，回到 `00-需关注` 的模板与问题总账。
3. 新增「一次性过程稿」默认放入本归档区对应子目录，并加归档头。
