# 要货计划

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。  
> 阅读对象：**测试、实施（主）**；计划员、供应商（顺带）。操作见[要货计划-维护与查询参考](要货计划-维护与查询参考.md)。售前停在[模块首页](../index.md)。

## 这一组解决什么问题

要货计划（采购计划）把可执行供应窗口落到供应商可接受的计划单：含仓库/月台、时间窗、交期、关联采购订单号等。状态含 NEW、PUBLISHED、ACCEPTED、REJECTED、CLOSED、COMPLETED；支持自动发布/自动接受开关。客户长期计划作为另一类计划输入对象。计划主/明细可通过 SCP↔WMS 接口推送。

## 功能范围

| --- | --- |

| 客户长期计划对象 | 库内收货（WMS）；MES 消费计划（未证实） |

## 主对象

| 对象 | 业务含义 |
| --- | --- |
| 采购计划头 | 单号、供应商、时间窗、仓/月台、状态、交期、PO、联系人、自动发布/接受等。 |
| 采购计划行 | 物料、计划量、接受量、订购量、已发/未收/已收、行状态、交期排序等。 |
| 客户长期计划 | 消息号、视界起止、供应商/买方卖方编码、状态 NEW/CLOSED 等。 |

### 关键字段业务角色

| 字段/配置点 | 在系统中的作用 | 关键行为要点 | 警惕什么 |
| --- | --- | --- | --- |
| 计划状态 | 发布/接受/关闭门禁 | NEW→PUBLISHED→ACCEPTED→CLOSED/COMPLETED；可 REJECTED | 未接受不可当可发 |
| 自动发布/自动接受 | 跳过人工环节 | 头开关 | 误开导致未审即协同 |
| 供应商 / 仓库 / 月台 | 到货安排 | 有效主数据；仓→月台级联 | 月台错影响到货 |
| 时间窗 / 交期 | 供应窗口 | 窗口外计划需重排 | SLA 失真 |
| 关联采购订单号 | 与订单未交衔接 | 可选关联 | 无 PO 时发货约束不同 |
| 行计划量 / 已发量 | 可发上限 | 发货回写 shippedQty | 超发失败 |
| 推送 WMS | 计划同步 | 主/明细可推送；同实例 ❓（`GAP-017`） | 勿当库存已变 |

完整表见[维护与查询参考](要货计划-维护与查询参考.md)。

### 选择器范围（骨架）

通例见[通用选择器过滤惯例](../../02-业务模型/12-通用选择器过滤惯例.md)。下表只写本页差异；发货侧「可发计划」精确状态集见 `FSEM-006`。

| 选择字段 | 选择对象 | 可选范围（当前可写） | 范围依赖 | 选不到时通常原因 |
| --- | --- | --- | --- | --- |
| 供应商 | SCP 供应商 | 可用；门户绑定另论 ❓（`GAP-017`） | — | 停用、未绑定 |
| 仓库 / 月台 | SCP 仓储建模 | 有效仓；仓→月台级联；非 WMS 库存权威 | 上游仓库 | 未建模、月台错仓 |
| 关联采购订单 | 采购订单 | 可选关联；宜为可协同订单；精确可关联状态 ❓（`FSEM-006`） | 供应商、订单状态 | 未发布、供应商不符 |
| 行物料 / 计划量 | 物料与数量 | 可用物料；用途/关系过滤 ❓（`FSEM-001` / `GAP-045`） | 供应商物料、关联 PO | 停用、无关系 |
| 发货侧选计划/行 | 已接受计划行 | 宜已接受（或可发）且有剩余；精确状态集 ❓（`FSEM-006`） | 头状态、已发量、供应商 | 未接受、已发完、已关闭 |
| 客户长期计划 | 长期视界输入 | 新建可纳入；已关闭勿再汇总；强制汇总规则 ❓（`GAP-017`） | 视界、供应商账号 | 已关闭、视界外 |
| 来源预测 | 要货预测 | 已发布可汇总；自动/手工规则 ❓（`GAP-017`） | 预测版本/活动版 | 未发布、非活动版 |

## 与发货 / WMS / MES 边界

| 协同方 | 本页负责 | 不在本页展开 |

| 发货协同 | 提供可发计划号 ppNumber 与可发量 | ASN 物流细节 |
