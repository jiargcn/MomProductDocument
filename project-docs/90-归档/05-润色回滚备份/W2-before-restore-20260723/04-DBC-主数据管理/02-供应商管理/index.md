# 供应商管理

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施（主）；采购协同、收货核对人员（顺带）。

## 这一组解决什么问题 / 功能范围

供应商管理维护采购协同中的供应方及其与物料的对应口径，确认「由谁供应、供应什么、业务中如何识别」。

**范围外：** 采购订单、到货收货与退货执行归 SCP/WMS；物料自身单位/包装归[物料管理](../01-物料管理/index.md)。

## 如何使用本组文档（测试 / 实施）

| 你的目的 | 建议阅读 |
| --- | --- |
