# 客户管理

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施（主）；销售协同、发运人员（顺带）。

## 这一组解决什么问题 / 功能范围

客户管理维护交付对象、交付地点及客户侧物料口径，使发货、客户退货等业务能以一致客户信息识别与追溯。

**范围外：** 销售出库/发运执行归 WMS/SCP；物料自身识别归[物料管理](../01-物料管理/index.md)。

## 如何使用本组文档（测试 / 实施）

| 你的目的 | 建议阅读 |
| --- | --- |
| 验证客户/月台/客户物料可选与匹配 | 各叶页**主文档** |
| 新增/导入/字段细节 | 同对象**维护与查询参考**（如有） |
| 从发运/退货问题回查客户口径 | WMS 销售出库等 → 本组 |

售前介绍请停在 [DBC 模块首页](../index.md)。

| 依赖 | 影响 | 在哪确认 |
| --- | --- | --- |
