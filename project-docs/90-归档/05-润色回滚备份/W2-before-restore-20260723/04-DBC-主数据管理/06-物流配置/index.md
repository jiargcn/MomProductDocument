# 物流配置

> 适用基线：测试环境 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施（主）；仓储/运输协同人员（顺带）。

## 这一组解决什么问题 / 功能范围

物流配置维护库存归属与运输协同中的关键伙伴，区分「货物属于谁、由谁承运」。

**范围外：** 供应商/客户采购销售伙伴见对应分组；承运过程跟踪与结算细则不在本组展开。

## 如何使用本组文档（测试 / 实施）

| 你的目的 | 建议阅读 |
| --- | --- |
| 验证货主/承运商可选与引用保护 | 各叶页**主文档** |
| 新增/导入/字段细节 | 叶页内维护说明（本组暂无统一拆页时以主文档为准） |
| 从收发货归属/承运字段回查 | WMS 收发业务 → 本组 |

售前介绍请停在 [DBC 模块首页](../index.md)。

## 常见问题与相关分组

分不清货主 vs 供应商/客户时，先区分「归属」与「采销伙伴」；承运商是否进某类单据，在对应收发业务中核对。
