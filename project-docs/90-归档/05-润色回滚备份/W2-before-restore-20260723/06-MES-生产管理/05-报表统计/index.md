# 报表统计

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。
> 阅读对象：测试、实施（主）；生产统计、质量、班组长（顺带）。

## 这一组解决什么问题 / 功能范围

报表统计提供 MES 过程与结果类 **记录查询**：报工、完工、点检、上下工等，支撑追溯联查、绩效与审计。读完应能说清有哪些记录入口、与追溯/WMS 如何联查。

**范围外：** 不是积木报表设计器（见基础设施报表说明）；不替代[追溯管理](../04-追溯管理/index.md) 正逆向专题页。

## 如何使用本组文档（测试 / 实施）

| 你的目的 | 建议阅读 |
| --- | --- |

| 按工单/人员过滤、导出字段细节 | [报表统计-维护与查询参考](报表统计-维护与查询参考.md) |
| 从记录跳到正逆向 | [追溯管理](../04-追溯管理/index.md) |

## 本组学习顺序

1. 报工记录 → 2. 完工记录 → 3. 点检 / 上下工记录 → 4. 需要时联查追溯或 WMS。

## 配置依赖概览

| 依赖 | 影响 | 在哪确认 |
| --- | --- | --- |
| 终端报工 / 完工 / 上下工动作 | 有无记录可查 | [终端操作](../06-终端操作/index.md) |
| 开工点检执行 | 点检记录来源 | [基础建模](../01-基础建模/index.md)（与 EAM 点检区分） |
| 菜单多入口并存 | 查哪套列表 | 以环境菜单为准（`MES-RPT`） |

本组多为查询页，**无独立业务类型配置**；受上游采集与权限约束。

## 记录类型

| 类型 | 业务含义 | 典型用途 |
| --- | --- | --- |
| 报工记录 | 工位/过程报工流水（工单、产品、追溯码、批次、工位、工序、结果、类型、任务模式等）。 | 计件、过程审计、追溯人员页签来源。 |
| 完工记录 | 工单完工事实（数量/包装、完工结果、库位/线边入库状态、追溯码与产品 SN 等）。 | 结案、入库协同、标签联查。 |
| 点检记录 | 点检执行结果查询（开工/相关点检；与 EAM 设备点检记录区分）。 | 开产合规。 |
| 上下工记录 | 线边客户端人员上/下工流水。 | 工时与在岗责任。 |
| 打印相关 | 若环境提供打印历史入口，用于补打与标签审计（以菜单为准）。 | 标签追溯。 |

## 关键判断

| 判断点 | 应先确认什么 | 影响 |
| --- | --- |
| 报工有、追溯无 | 查询条件粒度；是否用了正确码字段。 | 先对齐码再开追溯单。 |
| 完工有、库存无 | 线边入库状态 / WMS 收货。 | 回 WMS 生产管理。 |
| 点检记录找不到 | 是 MES 开工点检还是 EAM 设备点检。 | 选对模块菜单。 |
