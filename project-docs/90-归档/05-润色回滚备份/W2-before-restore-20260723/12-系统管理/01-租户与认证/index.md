# 租户与认证

> 适用基线：测试环境目标 / `dev` 分支 / 2026-07-15。  
> 阅读对象：**测试、实施（主）**；系统管理员、安全协同（顺带）。售前只了解「多租户与登录边界」，细节见本页；维护步骤见维护参考。

## 这一组解决什么问题

本分组说明两件事：

1. **租户**：系统如何把不同客户/运营单元的数据与可用功能隔开；  
2. **认证**：用户如何登录、保持会话、退出，以及登录后如何拿到菜单与权限标识。

组织部门树、角色菜单、数据范围见同模块其它页；本页不解释具体业务单据规则。

## 配置依赖概览

| 依赖 | 影响 |
| --- | --- |
| 是否启用多租户、套餐菜单集 | 租户功能上限 |
| 租户状态/到期、初始管理员 | 能否登录验收 |
| 环境安全策略（验证码/短信登录等） | 登录路径是否开放 |

## 使用前准备

| 需要确认什么 | 为什么重要 |
| --- | --- |
| 是否启用多租户运营 | 决定是否需要维护租户列表与套餐。 |
| 租户套餐菜单范围 | 套餐会限制该租户可用的功能上限。 |
| 管理员账号与联系人 | 新建租户通常伴随初始管理能力。 |
| 验收是否用正确租户账号 | 避免跨租户数据与菜单结论串扰。 |

## 租户如何生效

```mermaid
flowchart LR
    A["维护租户套餐（可选菜单集）"] --> B["创建/维护租户"]
    B --> C["租户绑定套餐与有效期等"]
    C --> D["租户内用户登录"]
    D --> E["仅在租户与套餐边界内使用系统"]
```

当前已证实：

| 能力 | 业务含义 |
| --- | --- |
| 租户主数据 | 名称、联系人、状态、域名、套餐、到期时间、账号数量等可维护。 |
| 租户套餐 | 通过可选菜单集合限定租户功能上限；变更套餐会影响租户内角色菜单边界。 |
| 新建租户初始化 | 创建租户时可按套餐初始化管理角色等基础授权（实施时按环境验证具体步骤）。 |
| 数据隔离 | 大量业务与系统对象带租户维度，默认在当前租户上下文中读写。 |

系统级特殊套餐标识用于不走普通套餐限制的系统租户场景；实施时不要把“系统租户”与普通业务租户混用验收。

## 认证与会话

| 能力 | 当前可确认结论 |
| --- | --- |
| 账号密码登录 | 提供管理端登录入口。 |
| 无验证码登录入口 | 源码存在对应入口；是否在测试/生产开放需按环境与安全策略确认，培训默认按受控环境说明。 |
| 短信登录 / 社交登录 | 存在相关接口能力；是否启用取决于环境配置与对接，不能写成默认必开。 |
| 刷新令牌 | 支持刷新会话令牌。 |
| 退出 | 支持主动退出并失效当前令牌。 |
| 登录后权限信息 | 登录用户可获取角色、菜单与权限标识集合，供前端路由与按钮显隐使用。 |

## 关键判断

| 判断点 | 应先确认什么 | 判断后的影响 |
| --- | --- | --- |
| 登不上 | 账号状态、租户状态/到期、密码、验证码/网络。 | 区分认证问题与授权问题。 |
| 能登录但菜单很少 | 租户套餐、角色菜单、菜单启用状态。 | 先查套餐上限再查角色。 |
| 能进菜单无数据 | 数据范围/岗位/业务状态，而非登录本身。 | 转到用户与权限或业务页。 |
| 是否跨租户可见 | 当前账号租户上下文。 | 禁止用跨租户假设做培训结论。 |

## 查询与联查

| 想解决的问题 | 推荐定位方式 | 建议联查 |
| --- | --- | --- |
| 租户功能缺失 | 套餐菜单是否包含目标功能。 | 租户套餐、角色菜单。 |
| 租户到期/停用 | 租户状态与到期时间。 | 租户列表。 |
| 登录后权限不对 | 用户角色、菜单、权限标识。 | RBAC 页。 |
| 会话经常失效 | 刷新令牌使用方式、超时策略（环境配置）。 | 运维/安全配置（待专项）。 |

## 常见问题与处理

| 情况 | 建议处理 |
| --- | --- |
| 新建租户后管理员进不去 | 检查租户状态、套餐、初始账号与密码策略。 |
| 改了套餐部分角色菜单消失 | 预期可能收缩到套餐交集；复核角色授权。 |
| 把部门问题当成租户问题 | 先确认是否同租户内组织/权限问题。 |
| 生产使用无验证码登录 | 按安全要求关闭或限制，不写入常规操作手册为推荐路径。 |

## 当前限制与待确认事项

- SSO/第三方身份源在目标环境的实际启用清单待核验；
- 密码复杂度、失败锁定、会话超时的环境参数待运维页专项取证；
- 登录日志/审计查询入口与字段待运维或本页续作补充；
- MES 等模块若另有“租户连接配置”类入口，属业务侧扩展，不并入本页主链。
