## 与产品站点的边界

- 面向培训/实施的稳定结论写入 `docs/`，由 `mkdocs.yml` 导航。
- 本目录只保存「如何形成结论」的过程与证据；正式页不得直接链接此处未审定内容。
- 已确认且需对使用者说明的限制，提炼后写入业务页或 `docs/15-版本路线图/`。
