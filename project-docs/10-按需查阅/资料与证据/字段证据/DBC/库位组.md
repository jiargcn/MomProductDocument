# DBC：库位组字段证据

> 对应正式页面：[库位组管理](../../../../docs/04-DBC-主数据管理/04-工厂建模/04-库位组管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/04-工厂建模/17-库位组管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 维护字段 | 编码、名称、描述、仓库、库区、可用状态、有效期和备注。 | 编码唯一性、前端选择器与服务校验。 |
| 导入 | 模板要求编码、名称、仓库、库区、可用状态。 | 重复记录、错误回执和部分成功处理。 |
| 关系 | 库位组由库位资料引用，按仓库/库区组织地点。 | 是否直接参与任务推荐、盘点范围和数据权限。 |

## 待核验项

| 编号 | 问题 | 影响 |
| --- | --- | --- |
| DBC-LG-001 | 库位组与成员库位、盘点和任务推荐的实际挂接待测试。 | 分组用途不能写成既成的自动业务规则。 |
| DBC-LG-002 | 删除和跨库区调整的引用保护待确认。 | 可能影响地点层级完整性。 |
