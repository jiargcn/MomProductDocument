# 生产商字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-22 第二轮。

## 已核证的维护/模板范围

| 信息分组 | 当前可维护或模板覆盖内容 |
| --- | --- |
| 基本识别 | 制造商编号、名称、简称。编号在创建路径校验非空；名称在导入路径强制非空。 |
| 联系资料 | 地址、国家、城市、电话、传真、邮编、联系人。 |
| 组织与状态 | 部门、备注、可用状态。 |

## 当前证据结论

- `EquipmentManufacturerServiceImpl.beforeCreate`：仅 `validateNumberEmpty`。
- `beforeUpdate`：直接返回，无编号锁定或重码校验。
- `beforeDelete`：无引用检查。
- 导入：`validImport` 额外校验名称非空；`getExistWhenImport` → `selectByEntry`。
- 模板未显示与设备/工装之间的关系信息。
