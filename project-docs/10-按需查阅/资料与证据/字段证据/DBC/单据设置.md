# 单据设置字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的输入与导入范围

| 业务信息 | 新增/编辑要求 | 导入模板情况 |
| --- | --- | --- |
| 代码、名称、描述 | 均必填。 | 均必填。 |
| 单据类型、业务类型 | 均必填。 | 均必填。 |
| 编号组成 | 前缀、日期格式、流水长度、重置周期必填；分隔符可选。 | 对应字段均可导入。 |
| 生效控制 | 生效/失效时间、备注、可用状态可维护。 | 均可导入；可用状态在输入模型中必填、导入模型未标记必填。 |

## 当前证据结论

- 单据设置同时关联单据类型、业务类型与编号组成。
- 导入模型与新增/编辑模型对“是否可用”的必填要求存在不一致，需测试实际导入行为。
- 当前未发现可据此确认的唯一性、编号预览、已生成单据后的编辑锁定、流水重置和并发生成逻辑。
