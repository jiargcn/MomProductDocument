# 工艺路线字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 归属结论

- DBC 目录中存在工艺路线说明页，但当前可查的实际前端页面和后端维护对象主要位于 MES。
- 路线主信息包含编码、名称、类别、版本、物料/BOM关联、状态、前置校验、校验规则、图形与配置内容。
- 路线明细包含工序顺序、前后关系、节拍、节点数据，以及并行、放行和分派相关配置。
