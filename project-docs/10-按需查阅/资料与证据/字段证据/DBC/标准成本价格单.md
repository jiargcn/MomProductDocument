# DBC：标准成本价格单字段证据

> 对应正式页面：[标准成本价格单管理](../../../../docs/04-DBC-主数据管理/01-物料管理/07-标准成本价格单管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/01-物料管理/11-标准成本价格单管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 页面维护 | 页面要求物料、币种、价格和可用状态；物料选择器过滤可用物料，价格页面限制非负且六位小数。 | 接口和导入是否同样拒绝负数；物料编辑是否锁定。 |
| 供应商 | 后端资料保留供应商位置，但当前 Web 表单、导入模板和服务校验没有形成完整维护链路。 | 供应商维度是否为未来能力、是否应从当前对象移除。 |
| 关系保护 | 服务按物料检查既有记录；数据库层未确认物料唯一约束。 | 多币种、多期间、多组织价格的业务键与并发策略。 |
| 有效期 | 页面可维护生效/失效时间。 | 时间先后校验、自动失效、按有效期取价规则。 |
| 状态与删除 | 支持启用/禁用和删除，并记录变更轨迹。 | 启停/删除对下游调用的影响，接口权限与前端权限不一致。 |
| 导入 | 模板包含物料、币种、价格、可用状态、有效期和备注；导入检查必填和物料存在。 | 仅新增/仅更新的错误回执、部分成功时回滚行为。 |
| 下游线索 | 存在按物料查询可用价格的能力；生产线物料关系查询会读取可用价格作展示补充。 | 是否用于库存计价、工单成本、结算或报表的实际核算。 |

## 待核验项

| 编号 | 问题 | 影响 |
| --- | --- | --- |
| DBC-SCP-001 | 当前仅按物料处理既有价格，且未确认数据库唯一保护。 | 多币种、多期间或并发维护可能覆盖或误取价格。 |
| DBC-SCP-002 | 供应商字段存在但当前没有完整前端、导入、校验链路。 | 文档和模板不能把供应商作为当前维护条件。 |
| DBC-SCP-003 | 有效期未确认自动校验或自动取价；查无可用价格可能回传零值。 | 成本分析可能把“未维护”误解为零成本。 |
| DBC-SCP-004 | 状态接口权限配置、删除引用保护、导入模式回执未完成测试验证。 | 权限、变更风险和批量维护的可控性不足。 |
