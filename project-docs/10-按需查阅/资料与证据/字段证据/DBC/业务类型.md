# 业务类型字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的配置范围

| 配置分组 | 当前可维护内容 |
| --- | --- |
| 基本识别 | 代码、名称、分组、生效/失效时间、备注、可用状态。 |
| 物料与质量范围 | 可用物料类型、物料状态、物品种类、质量等级。 |
| 库存范围与事务 | 入/出库库区类型与范围、入/出库库存状态、入/出库及撤销事务类型、在途库使用与在途库位。 |
| 自动处理 | 自动提交、自动通过、自动执行、直接生成记录。 |
| 操作限制 | 库位、数量、批次、包装修改；部分完成、超量完成、重复扫描、目标库位扫描。 |
| 单据与打印 | 申请/任务/记录的编号设置及打印模板关联。 |

代码、名称、分组、物料类型范围、物料状态范围、是否使用在途库和可用状态被标记为必填；其余项目需结合具体业务场景配置。

## 导入模型结论

- 当前存在覆盖上述大部分范围的业务类型导入/导出模型。
- 模板中混入创建时间、创建者、更新者等审计信息；这些信息不应作为人工导入内容，需以实际下载模板为准进一步核验。
- 申请、任务、记录的编号与打印关联以内部标识或编码表达，批量导入前必须确认来源与有效性。
