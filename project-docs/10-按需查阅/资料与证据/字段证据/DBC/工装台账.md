# 工装台账字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的维护/模板范围

| 信息分组 | 当前可维护或模板覆盖内容 |
| --- | --- |
| 身份分类 | 编号、名称、型号、类型、产权。 |
| 客户与现场 | 客户资质、客户、存放位置、使用部门、责任人、工厂/车间/产线/工序/工位。 |
| 采购制造 | 供应商、生产商、采购/出厂/验收日期、价格、寿命。 |
| 使用维护参考 | 状态、运行/保养时长、使用次数、检验、故障参考指标、可用状态。 |

模板中编号、名称、类型、客户资质和状态标记必填；主输入模型仅明确名称具备非空校验，其余页面实际必填仍需验证。
