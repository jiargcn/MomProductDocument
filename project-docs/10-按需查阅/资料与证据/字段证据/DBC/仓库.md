# DBC：仓库字段证据

> 对应正式页面：[仓库管理](../../../../docs/04-DBC-主数据管理/04-工厂建模/01-仓库管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/04-工厂建模/14-仓库管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 维护字段 | 维护编码、名称、描述、类型、可用状态、有效期和备注。 | 编码长度、重复保护和编辑锁定。 |
| 导入 | 模板包含编码、名称、类型、描述、可用状态、有效期、备注；编码和可用状态必填。 | 导入模式、部分成功和错误回执。 |
| 查询 | 页面支持按编码、名称、类型、状态等查询。 | 详情分组、库存和任务快捷联查。 |
| 下游 | 仓库被库区、库位、月台和物料仓库默认配置引用。 | 停用/删除对库存和未完成任务的拦截。 |
