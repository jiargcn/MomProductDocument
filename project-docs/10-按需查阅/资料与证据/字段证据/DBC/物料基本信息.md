## 1. 对象与证据来源

| 项目 | 当前证据 |
| --- | --- |
| 业务对象 | 物料基本信息。 |

| 页面规则 | 物料新增/编辑、列表、导入、标签入口。 |

## 2. 关键字段映射

| 中文名称 | 后端属性 | 数据库列 | 类型/长度 | 结论 |
| --- | --- | --- | --- | --- |
| 物料号 | `code` | `code` | `varchar(64)`，非空 | 技术字段应写 `code`，不写历史草稿中的 `materialCode`。 |
| 名称 | `name` | `name` | `varchar(64)` | 页面名称长度限制与数据库长度需按页面/导入分别说明。 |
| 状态 | `status` | `status` | `varchar(64)`，非空 | 字典来源待继续核验。 |
| 基本单位 | `uom` | `uom` | `varchar(64)`，非空 | 不写历史草稿中的 `unit`。 |
| 替代单位 | `altUom` | `alt_uom` | `varchar(64)` | 属性和列为驼峰/下划线映射。 |
| 可采购 | `enableBuy` | `enable_buy` | `varchar(64)`，非空 | 取值字典口径待确认。 |
| 可制造 | `enableMake` | `enable_make` | `varchar(64)`，非空 | 取值字典口径待确认。 |
| 可委外加工 | `enableOutsourcing` | `enable_outsourcing` | `varchar(64)`，非空 | 取值字典口径待确认。 |
| 回收件 | `isRecycled` | `is_recycled` | `varchar(64)`，非空 | 已确认正确业务语义为“回收件”；前端/导入“标准件”是错误。 |
| 虚物料 | `isPhantom` | `is_phantom` | `varchar(64)`，非空 | 不使用未经证实的英文别名。 |
| ABC 类 | `abcClass` | `abc_class` | `varchar(64)`，非空 | 字典来源待确认。 |
| 物料类型 | `type` | `type` | `varchar(64)`，非空 | 不写历史草稿中的 `materialType`。 |
| 种类/分组 | `category` / `itemGroup` | `category` / `item_group` | `varchar(64)` | 字典来源待确认。 |
| 结算方式/质量等级 | `settlementType` / `eqLevel` | `settlement_type` / `eq_level` | `varchar(64)` | 旧文档技术名须校正。 |
| 有效天数 | `validityDays` | `validity_days` | `numeric(18,2)`，默认 0 | DO 为整数，存在类型转换待核验项。 |
| 是否可用 | `available` | `available` | `boolean`，默认 true | 系统可用性字段。 |
| 是否脱离 ERP | `isOutErp` | `is_out_erp` | `varchar(64)`，默认 `FALSE` | 业务影响待继续核验。 |

## 3. 页面、接口与导入证据

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 新增/编辑 | 接口层显式必填包括物料号、基本单位、采购/制造/委外、回收件、虚物料、ABC 类、类型、有效天数、可用性。 | 名称、状态在不同层的必填表达不完全一致。 |
| 页面校验 | 物料号非空、最长 30 字符并做格式校验；名称非空、最长 50 字符。 | 页面格式校验与导入是否完全一致。 |
| 编辑边界 | Web 编辑页面禁用物料号。 | 服务层仍保留直接改码兼容分支，跨业务保护未证实。 |
| 列表/查询 | 列表前序字段为物料号、名称、描述、类型、基本单位、审计信息、状态；查询包括物料号、名称、类型、单位、状态、是否可用。 | 详情实际布局和扩展属性展示。 |
| 字典/动态限制 | 类型、单位、状态由字典构造；类型字典备注可提供有效天数上限。 | 字典值和备注配置规范。 |
| 导入 | 导入模板与保存模型分层处理，支持更新、追加、覆盖和错误文件。 | `altUom`、`isOutErp` 在导入与新增规则中必填不一致。 |
