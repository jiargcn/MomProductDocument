# 班次字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-15。

## 已核证的输入与导入范围

| 业务信息 | 新增/编辑要求 | 导入模板情况 |
| --- | --- | --- |
| 班次代码 | 必填。 | 必填。 |
| 班次名称、描述 | 名称可维护，描述可选。 | 名称可导入；描述不在当前导入模板中。 |
| 开始与结束时间 | 均必填。 | 均必填。 |
| 生效/失效时间、备注 | 可维护。 | 均可导入。 |
| 是否跨天、是否可用 | 均必填。 | 均必填。 |
| 休息事项、休息起止时间 | 休息对象独立存在。 | 模板包含休息事项及起止时间，两个时间标记必填。 |

## 当前证据结论

- 班次主对象包含代码、名称、描述、起止时间、生效/失效时间、备注、跨天标识和可用状态。
- 批量导入将班次与一项休息安排放在同一行；多项休息安排的行组织方式尚未验证。
- 当前未发现可据此确认的唯一性、跨天时间先后校验、停用引用保护或既有业务重算逻辑。
