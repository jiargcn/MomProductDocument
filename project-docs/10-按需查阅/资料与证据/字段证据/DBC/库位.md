# DBC：库位字段证据

> 对应正式页面：[库位管理](../../../../docs/04-DBC-主数据管理/04-工厂建模/03-库位管理.md)、[维护与查询参考](../../../../docs/04-DBC-主数据管理/04-工厂建模/16-库位管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 维护字段 | 编码、名称、仓库、库区、库位组、外部系统库位口径、备货优先级、容量、状态和有效期。 | 各外部系统字段的真实含义和前端绑定关系。 |
| 导入 | 模板要求编码、名称、仓库、库区、库位组、外部系统库位信息、备货优先级、可用状态。 | 数值容量字段的单位、格式和上下限校验。 |
| 下游 | 库位被库存、上架、拣选、调拨、盘点和标签等场景引用。 | 停用、删除、改码时对库存/任务/标签的保护。 |

## 待核验项

| 编号 | 问题 | 影响 |
| --- | --- | --- |
| DBC-LOC-001 | 库位容量、混放、物料限制和备货优先级是否进入推荐规则待确认。 | 不能将地点属性直接当作已生效的作业限制。 |
| DBC-LOC-002 | 外部系统库位字段存在命名差异，映射与同步结果未验证。 | 导入和跨系统地点识别可能错误。 |
| DBC-LOC-003 | 删除/停用对实际库存和在途任务的拦截未验证。 | 现场地点调整风险高。 |
