# 承运商字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-22 第二轮。

## 已核证的输入与导入范围

| 信息分组 | 当前可维护/导入内容 |
| --- | --- |
| 基本识别 | 代码、名称、简称、类型。代码和简称必填。 |
| 联系与地址 | 地址、国家、城市、电话、传真、邮编、联系人。 |
| 结算信息 | 银行、币种、税率。 |
| 生效控制 | 生效/失效时间、备注、可用状态；可用状态必填。 |

## 当前证据结论

- 继承 `WinBaseService`；`beforeCreate/beforeUpdate` 调用代码查重与必填校验。
- 更新仍接收代码；未见“锁定原代码”回填。
- 导入：`validImport` 强制代码/简称/可用；`getExistWhenImport` 按代码匹配。
- 导入对象注释误称“载具导入”，但类和字段内容属于承运商；正式页面按承运商语义说明。
- 删除未见运输单据引用检查；DDL 未见业务唯一约束证据。
