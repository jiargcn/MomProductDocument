# DBC：物料包装信息字段证据

> 对应正式页面：[物料包装信息](../../../../docs/04-DBC-主数据管理/01-物料管理/03-物料包装信息.md)。
> 基线：测试环境 / `dev` 分支。本波字段技术名核对：2026-07-22（`base.sql` / `ItempackageDO` / `ItempackageBaseVO` / `ItempackageImportExcelVO` / `itempackage.data.ts`）。

## 1. 对象与证据来源

| 项目 | 当前证据 |
| --- | --- |
| 业务对象 | 物料包装信息。 |
| 数据对象 | `basic_itempackage`、`ItempackageDO`。 |
| 页面规则 | `itempackage.data.ts`；导入 `ItempackageImportExcelVO`。 |

## 2. 持久化字段映射（已核实）

| 业务中文名 | 前端/API 属性 | DO 属性 | 数据库列 | 类型/备注 | 结论 |
| --- | --- | --- | --- | --- | --- |
| 物料代码 | `itemCode` | `itemCode` | `item_code` | `varchar(64)` 非空 | 已确认；禁止写 `materialCode`。 |
| 计量单位 | `uom` | `uom` | `uom` | `varchar(64)` 非空 | 已确认；页面表单常禁用并由物料带入；导入 VO **不含** `uom`。 |
| 包装规格 | `packageUnit` | `packageUnit` | `package_unit` | `varchar(64)` 非空 | 已确认；禁止写 `packageCode`/`packUnit`。 |
| 包装数量 | `packageStdQty` | `packageStdQty` | `package_std_qty` | `numeric(18,6)` 非空 | 已确认。 |
| 默认包装规格 | `defaultPackageunit` | `defaultPackageunit` | `default_packageunit` | `varchar(64)` | 已确认（拼写为 `Packageunit` 连写，勿改成 `defaultPackageUnit`）。 |
| 是否可用 | `available` | `available` | `available` | `boolean` | 已确认。 |
| 生效/失效时间 | `activeTime` / `expireTime` | 同左 | `active_time` / `expire_time` | timestamp | 已确认。 |
| 备注 | `remark` | `remark` | `remark` | text | DO/DDL 有；当前导入模板注释掉备注列。 |

## 3. 仅展示/响应层字段（非持久化）

| 业务中文名 | VO/前端键 | DDL/DO | 结论 |
| --- | --- | --- | --- |
| 包装名称 | `packName` | 无 | 列表展示字段，来自包装规格补充；**不得**写入字段字典为可维护/可导入列。 |
| 包装类型 | `packType` | 无 | 同上。 |
| 替代包装单位/量 1–4 | `altPackageUnit1..4` / `altPackageStdQty1..4` | 无 | `ItempackageBaseVO` 仍声明；导入 VO 中对应列已注释；DDL/DO 不存储。 |

## 4. 页面、导入证据

| 事项 | 已确认内容 | 待核验项 |
| --- | --- | --- |
| 新增 | 页面要求物料、单位、包装规格、包装数量和可用状态；物料、包装规格选择器过滤可用对象。 | 新增后的默认状态及有效期处理；`uom` 服务端回填规则。 |
| 关系保护 | 服务创建/导入会校验物料与包装规格的对应关系；默认包装有专用校验。 | 同物料默认包装的唯一性和并发保护（见 GAP-025）。 |
| 导入 | 模板列：物料代码、包装规格、包装数量、是否可用、生效/失效、默认包装规格。 | 模板列、导入模式和错误回执体验；导入是否自动补 `uom`。 |
| 查询 | 支持按物料筛选；列表可见包装名称/类型（展示字段）。 | 默认排序、库存/标签快速跳转。 |

## 5. 字段技术名校正提醒

禁止把展示字段或历史反译写成持久化技术名：`packageName`、`packageType`、`materialCode`、`packUnit`、`altPackUnit1`（历史注释名）等。以 `itemCode` + `packageUnit` + `packageStdQty` + `defaultPackageunit` 为存储权威。
