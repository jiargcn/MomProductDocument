# 货主字段证据

> 内部证据页，不对产品站点发布。基线：测试环境 / `dev` 分支 / 2026-07-22 第二轮。

## 已核证的输入与导入范围

| 信息分组 | 当前可维护/导入内容 |
| --- | --- |
| 基本识别 | 代码、名称、简称、类型。代码和简称必填。 |
| 联系与地址 | 地址、国家、城市、电话、传真、邮编、联系人。 |
| 结算信息 | 银行、币种、税率。 |
| 生效控制 | 生效/失效时间、备注、可用状态；可用状态必填。 |

## 当前证据结论

- 普通新增/更新：`OwnerServiceImpl.validateCodeExists` 按代码查重（排除自身）。
- 更新路径：未见强制保留原代码；接口可写回代码等关键字段。
- 导入：`validateOwnerImport` 仅强制代码/简称/可用；代码唯一校验调用被注释；按 `selectByCode` 匹配后按 mode 写入。
- 删除：无下游引用检查。
- DDL：`basic_owner` 未见业务唯一约束证据。
