## 2. 对象与前端字段（摘要）

### 2.1 采集点 `CollectPointInfoVO`

`collectPointCode`、`collectPointName`、`targetIp`、`targetParamAddr`、`targetType`、`targetPort`、`targetExt1`、`targetExt2`、`id`

字典：`iot_target_type`
权限前缀：`iot:collect-point-info:*`
API：`/iotgw/collect-point-info/{page,get,create,update,delete,export-excel}`

### 2.2 设备参数配置 `AssetParamConfigVO`

`assetCode`、`assetName`、`paramName`、`paramType`、`paramCode`、`refrensValue`、`rwType`、`isCollect`、`collectPointCode`

字典：`data_type`、`iot_param_rw`
页面标签「分区编码」绑定 `collectPointCode`
`isCollect`：`1` 采集 / `0` 不采集
API：`/iotgw/asset-param-config/...`

### 2.3 采集设备 `AssetCollectInfoVO`

`assetCode`、`assetName`、`assetType`、`status`、`id`

字典：`iot_asset_type`
状态开关：启用 `0` / 不启用 `1`
API：`/iotgw/asset-collect-info/...`

### 2.4 参数采集日志 `ParamCollectLogsVO`

`assetCode`、`paramCode`、`collectTime`、`collectValue`、`collectPoint`、`paramType`、`clientId`、`collectTimetz`

API：`/iotgw/param-collect-logs/...`（页面以查询/导出为主）

### 2.5 MQTT 客户端状态 `MqttClientStatusVO`

`clientId`、`onlineDate`、`offlineDate`、`status`、`userName`

字典：`iot_client_link_staus`（注意字典类型拼写与代码一致）
API：`/iotgw/mqtt-client-status/...`；列表页新增按钮注释
