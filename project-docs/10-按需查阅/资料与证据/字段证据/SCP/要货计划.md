# 字段证据：SCP / 要货计划

> 对应正式页面：[业务说明](../../../../docs/10-SCP-供应链平台/04-要货计划/index.md)、[维护与查询参考](../../../../docs/10-SCP-供应链平台/04-要货计划/要货计划-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单与前端

- `purchasePlanMain`、`customerLongtermPlanMain`。
- 控制器：`PurchasePlanMainController` / `PurchasePlanDetailController`、`CustomerLongtermPlanMainController`。
- 首页：`openPurchasePlanCount` / `allPurchasePlanCount`。

## 2. 关键对象

| 业务对象 | 表 | 关键字段 |
| --- | --- | --- |
| 采购计划头 | `scp_plan_purchase_main` | number、supplierCode、timeWindow、warehouseCode、dockCode、status、deliveryDate、poNumber、autoPublish、autoAccept、publish/acceptTime |
| 采购计划行 | `scp_plan_purchase_detail` | itemCode、planQty、acceptQty、orderQty、shippedQty、notReceiveQty、receivedQty、detailStatus、poNumber/poLine |
| 客户长期计划 | `scp_plan_customer_longterm_main` | messageNumber、horizonStart/End、supplierAccountNumber、buyer/sellerCode、status |

## 3. 枚举

- `PurchasePlanStatusEnum`：NEW、CLOSED、PUBLISHED、ACCEPTED、REJECTED、COMPLETED。
- `CustomerLongtermPlanStatusEnum`：NEW、CLOSED。
- 接口：`PLAN_PURCHASE_MAIN` / `PLAN_PURCHASE_DETAIL`；推送时 `interfaceType=SCP-WMS`。

## 4. 边界

- 发货服务会 `syncPurchasePlanShippedQty` 回写已发量。
- MES 是否消费计划：未证实。
