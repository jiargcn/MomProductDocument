# 字段证据：SCP / 采购跟踪

> 对应正式页面：[业务说明](../../../../docs/10-SCP-供应链平台/06-采购跟踪/index.md)、[维护与查询参考](../../../../docs/10-SCP-供应链平台/06-采购跟踪/采购跟踪-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单与前端

- `purchasereceiptManage/purchasereceipt/*`、`purchasereturn/*`、`putaway/*`、`inspect/*`。
- 控制器：`Purchasereceipt*Controller`、`Purchasereturn*Controller`、`PurchaseshortageDetailController` 等。

## 2. 关键对象

| 业务对象 | 表 |
| --- | --- |
| 收货申请/任务/记录 | `scp_request_purchasereceipt_*`、`scp_job_purchasereceipt_*`、`scp_record_purchasereceipt_*` |
| 短缺 | `scp_record_purchaseshortage_detail` |
| 退货申请/任务/记录 | `scp_request/job/record_purchasereturn_*` |

任务状态枚举 `JobStatusEnum`：PENDING、PROCESSING、COMPLETED、CLOSED、REFUSAL。
库存状态枚举 `InventoryStatusEnum`：INSP、OK、NOK、HOLD、RETURN、SCRAP（SCP 侧出现，权威事务仍归 WMS）。

## 3. 实现缺口（源码注释）

`PurchasereceiptRecordMainServiceImpl` 等标明：

- 依赖 WMS `BusinesstypeService` / `DocumentsettingService` 未迁完；
- `createInspectRequest*`、`createPutawayRequestByPC`、`createPurchasereturnRecord` 等暂未在 SCP 实现。

## 4. 边界

- **库存事务与余额以 WMS 为准**。
- SCP 页用于供应商/采购跟踪与迁移中的协同视图，不能替代 WMS 仓内执行文档。

## 5. 待确认

- SCP/WMS 收货是否同库或接口镜像；未收货独立菜单形态。
