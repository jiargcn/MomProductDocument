# Infra：查询、筛选与用户视图证据

> 对应正式页面：[查询、筛选与用户视图](../../../../docs/03-基础设施/03-查询、筛选与用户视图.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 已证实

| 项 | 证据 | 结论 |
| --- | --- | --- |
| 分页列表 | 各业务 `page` / `PageReqVO` 模式 | 通用。 |
| 高级查询 | 大量业务使用 `CustomConditions` 条件组合 | 页面级能力，非独立模块菜单。 |
| 常用筛选 | 各业务页头筛选项 | 字段因页而异。 |

## 2. 未证实

| 项 | 结论 |
| --- | --- |
| 全站统一用户视图保存/共享中心 | **未证实** |
| 统一字段显示权限配置台 | **未证实**（若单页有列设置，属页面级） |

## 3. 与数据权限

列表结果仍可能叠加角色部门范围、岗位库位等，见 System 数据权限/岗位证据页。

## 问题登记

- 高级查询字段白名单与性能上限抽样。
- 用户视图需求与现状差异可登记产品项，不阻塞本页口径。
