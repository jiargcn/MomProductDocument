# Infra：报表、看板与数据输出证据

> 对应正式页面：[报表、看板与数据输出](../../../../docs/03-基础设施/02-报表、看板与数据输出.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 入口（已证实）

| 入口 | 证据 | 结论 |
| --- | --- | --- |
| 报表设计器 | 菜单 `report/jmreport`；前端 `views/report/jmreport/index.vue` 打开积木报表 list | JimuReport 设计入口。 |
| 报表列表/业务报表 | 菜单 `/report`、`/reportList`；`wms/reportList` 等 | 业务报表挂载。 |
| 业务内嵌 | 多页 `jmreport/view/{id}?token=...`（部分带 `tenant_id`） | 模板预览嵌入业务。 |
| 基地址 | `VITE_JMREPORT_BASE_URL` / `getJmreportBaseUrl()` | 环境可配置。 |

## 2. 边界

- 列表 Excel 导出 ≠ 报表模板（见导入导出证据）。
- MES OEE 等专项报表页可能独立，不全部等于设计器清单。
- 统一实时看板/指标语义层：未证实。
