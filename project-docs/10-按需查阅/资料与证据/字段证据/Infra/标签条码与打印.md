# Infra：标签、条码与打印边界证据

> 对应正式页面：[标签、条码与打印](../../../../docs/03-基础设施/01-标签、条码与打印.md)。
> 业务操作细则证据见：[WMS/标签与条码](../WMS/标签与条码.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 当前主入口（已证实）

| 能力 | 位置 | 结论 |
| --- | --- | --- |
| 标签模板 | WMS `LabelTemplateController` | 规则/模板维护在 WMS。 |
| 标签信息/补打 | WMS `BarbasicController`（含 `reprint`、库位标签打印等） | 标签实例与补打在 WMS。 |
| 打印 | WMS `PrintController`（`/wms/print`：打印机列表、模板打印） | 打印通道当前挂在 WMS。 |
| 正式业务说明 | `docs/05-WMS-库房管理/01-基础数据/06-标签与条码.md` | 已业务化。 |
