# Infra：导入、导出与批量操作证据

> 对应正式页面：[导入、导出与批量操作](../../../../docs/03-基础设施/04-导入、导出与批量操作.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 入口形态（已证实）

- 大量业务 Controller 提供 `importExcel`（常见 `mode` / `updatePart` 等参数）。
- 导出多为各业务 `export*Excel`；Infra 日志页亦有导出。
- **无**已证实的单一“全站导入中心”替代业务页入口。

## 2. 导入/导出日志（WMS）

| 对象 | 证据 | 结论 |
| --- | --- | --- |
| 导入日志 | `import_log` / `ImportLogController`；菜单 `wms/importLog` | 含业务名、状态、进度、错误文件 URL、文件名、异常信息等。 |
| 导出日志 | `ExportLogController`；菜单 `wms/exportLog` | 导出过程查询入口。 |
| 菜单分组 | `导入导出日志` | WMS 侧聚合入口。 |

## 3. 平台页职责

正式页只写通用口径与日志联查；模板字段/校验仍在各业务页。已知模板与保存不一致项见问题总账（如 GAP-018 等）。
