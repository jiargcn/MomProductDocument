# 字段证据：MES / 终端操作

> 对应正式页面：[业务说明](../../../../docs/06-MES-生产管理/06-终端操作/index.md)、[维护与查询参考](../../../../docs/06-MES-生产管理/06-终端操作/终端操作-维护与查询参考.md)。  
> 基线：测试环境 / `dev` / 2026-07-15。内部证据页。

## 1. 菜单与边界

| 证据 | 结论 |
| --- | --- |
| 目录「线边客户端」`/mesprocess` | MES 线边主入口 |
| 生产报工 `/mes/reportForWork`；报工记录；上下工记录 `lineclientworkerlogflow`；线边库存 `lineEdgeInventory` | MES 侧查询/辅助入口 |
| 大量 `pages/issue|deliver|putaway|...` | **WMS PDA**，正式终端页明确排除 |

## 2. 工位作业 API（核心）

控制器：`WoWorkstationJobController` → `/mes/wo-workstation-job`

| 能力（summary） | 业务含义 |
| --- | --- |
| 统一领取/定位（抢单/派单 × SN/顺序） | 现场领活主入口 |
| 抢单认领 FORASSIGN → WAIT | 抢单模式 |
| 开始作业 | 进入执行中 |
| 单件报工 / payload / 设备入口报工 | 仅单件任务模式 |
| 批量报工 | 仅批量；结合 WO_MIN_OK / BATCH_PER_TASK 门闸 |
| 工单工序 OK 门闸快照 | 累计/门槛/是否放流 |
| 重置待执行 / 待分派 | 异常回收 |
| 首工序作业池补槽 | 池维护 |
| 按 snCode 查作业等 | 扫码定位 |

## 3. 状态枚举

| 枚举 | 取值（code → 业务名口径） |
| --- | --- |
| `OrderTaskStatusEnum` | `forassign` 待分派、`wait` 待执行、`running` 执行中、`pause` 暂停、`locked` 锁定、`complete` 已完成、`abort` 异常中止、`packaged` 已装箱 |
| 工单状态 | 见计划管理证据 `WorkorderStatusEnum` |
| 分派 | `JobAssignModeEnum`：GRAB / DISPATCH |

## 4. 相关对象

| 对象 | 说明 |
| --- | --- |
| 运行工单 | `mes_workorder_info_run`，计划下发生成 |
| 报工日志 | `wowsreportlog`、`wobulkreportlog` 等包存在 |
| 上下工 | `lineclientworkerlog` / `lineclientworkerlogflow` |
| 扫码唯一码日志 | `linescanuniquecodelog` |
| BULK | `WorkOrderBulkController` `/mes/work-order/bulk`：报工、人工尾箱、待装箱 |
| 并行组布局/扫码路由 | `WorkOrderController` parallel-group execution-layout / resolve-by-trace |
