# WMS：生产管理字段证据

> 对应正式页面：[业务说明](../../../../docs/05-WMS-库房管理/08-生产管理/index.md)、[维护与查询参考](../../../../docs/05-WMS-库房管理/08-生产管理/生产管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。范围：WMS `productionManage`，不含 MES，不含发料侧生产收料/退料。

## 1. 对象关系

| 业务对象 | 数据/服务对象 | 证据结论 |
| --- | --- | --- |
| 制品收货 | `request/job/record_productreceipt_*`，`Productreceipt*Service` | ATR 齐全；主链。 |
| 隔离收货 | 菜单独立；复用 productreceipt + 业务类型 | 不另建表族。 |
| 完工撤销 | `*_productredress_*`，`Productredress*Service` | ATR 齐全。 |
| 制品上架 | 共用 putaway 表族 + `Productputaway*Service` | ATR 齐全。 |
| 制品拆解 | `*_productdismantle_*` | 申请+记录；无 Job。 |
| 制品返修 | `*_productrepair_*` | 申请+记录；无 Job。 |

## 2. 已证实库存挂接

| 对象 | 预计 | 事务 |
| --- | --- | --- |
| 制品收货 | 申请 `createExpectin`；任务执行清理预计 | 任务/记录 `createTransaction` |
| 完工撤销 | 可清理预计入 | 记录可 `createTransaction` |
| 制品上架 | 申请可建预计入/出 | 任务/记录事务并清理预计 |
| 拆解/返修 | 预计未完全证实 | 申请/记录可见事务调用 |

条件动作（回冲、外部接口、上架规则、QMS）受开关控制，见 GAP-069。

## 3. 菜单与终端

- Web：`wms/productionManage/*`。
- PDA：`pages/productReceipt`、`productRecycle`、`productPutaway`、`fg` 等。

`GAP-069`：对象族与条件后续动作未形成可执行矩阵。生产收料对象映射另见 `WMS-PROD`。
