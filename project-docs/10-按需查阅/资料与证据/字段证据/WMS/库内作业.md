# WMS：库内作业字段证据

> 对应正式页面：[业务说明](../../../../docs/05-WMS-库房管理/11-库内作业/index.md)、[维护与查询参考](../../../../docs/05-WMS-库房管理/11-库内作业/库内作业-维护与查询参考.md)。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。内部证据页。

## 1. 对象关系

| 业务对象 | 数据/服务对象 | 证据结论 |
| --- | --- | --- |
| 库内转移/状态转移/调拨 | `request/job/record_inventorymove_*`，`Inventorymove*Service` | 申请—任务—记录齐全；调拨菜单独立但 API/业务类型走 inventorymove。 |
| 计划外出库 | `*_unplannedissue_*`，`Unplannedissue*Service` | ATR 齐全；申请可建预计出。 |
| 计划外入库 | `*_unplannedreceipt_*`，`Unplannedreceipt*Service` | ATR 齐全；申请可建预计入。 |
| 报废 | `*_scrap_*`，`Scrap*Service` | ATR 齐全；申请可建预计出。 |
| 物料拆解 | `item_dismantle_*` / 记录表 | 非标准 ATR。 |

## 2. 已证实库存挂接

| 子域 | 预计库存 | 事务 |
| --- | --- | --- |
| inventorymove | 任务路径可建预计出；预计入存在分支不一致（GAP-042） | 记录完成 `createTransaction`，并清理预计 |
| 计划外出库 | 申请 `createExpectout` | 任务/记录可建事务 |
| 计划外入库 | 申请 `createExpectin` | 任务/记录可建事务 |
| 报废 | 申请 `createExpectout` | 任务/记录/申请路径可建事务 |

## 3. 菜单与终端

- Web：`inventoryjobManage`、`moveManage/inventorymove` 及状态转移入口。
- PDA：`pages/inventoryMove`、`transfer`、`unPlanned`、`scrap`。
