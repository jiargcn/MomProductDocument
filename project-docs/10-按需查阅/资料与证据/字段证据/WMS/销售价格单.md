## 已定位证据

## 已确认事实

| 业务信息 | 当前事实 | 文档使用方式 |
| --- | --- | --- |
| 适用范围 | 客户、物料、币种、价格、可用状态必填；价格不可为负。 | 正式页写必填与非负。 |
| 客户物料前置 | `validateCodeExists` 调客户物料 API，无关联则拒绝。 | 正式页写“先建客户物料”。 |
| 编辑边界 | Web 禁用客户/物料；服务更新强制保留原 `itemCode`，未见同等保留 `customerCode`。 | 正式页写 Web vs 旁路差异。 |
| 导入匹配 | `importSalepriceList` 用 `selectByItemCode` 定位既有行，不以客户+物料。 | 正式页标 ❓/`WMS-SP-001`。 |
| 业务取价 | `selectItemPackagingExist` / `getSalepriceList` 按客户+物料且要求可用。 | 与导入匹配键差异写入正式页。 |
| 有效期 | 字段可存；未见重叠校验或按期过滤。 | `WMS-SP-001/002`。 |
| 删除 | 无下游引用检查。 | `WMS-SP-003`。 |
