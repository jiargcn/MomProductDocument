## 1. PDA 菜单（在「PDA菜单管理」下）

设备/工装：**保养**（派发/执行/确认）、**巡检**（执行/确认/工单）、**点检**（执行/确认）、**开拉**（执行/确认）、**维修**（派发/执行/确认）、**报修**（新建/列表/审批/详情/记录）、**备件**（申领/出库/归还/入库）、工艺维修（部分菜单）。

路径形如 `pages/eam/...`，component 前缀 `eamPda*` / `eamInspection-*` / `eamRepair-*` 等。

## 2. 边界

- EAM PDA ≠ WMS PDA ≠ MES 线边客户端。
- 业务状态机与 Web 共用后端工单/报修服务；本页只补终端入口差异。
