# 字段证据：EAM / 工装管理

> 对应正式页面：[业务说明](../../../../docs/08-EAM-设备管理/04-工装管理/index.md)、[维护与查询参考](../../../../docs/08-EAM-设备管理/04-工装管理/工装管理-维护与查询参考.md)。
> 基线：测试环境 / `dev` / 2026-07-15。

## 1. 菜单

| 菜单 | component | 归属 |
| --- | --- | --- |
| 工装台账管理 | **`dbc/toolAccounts/index`** | DBC |
| 工装出库/入库记录 | `toolEquipmentOut` / `In` | EAM |
| 工装到货验收 | `toolSigning` | EAM |
| 工装变更记录 | `toolChangedRecord` | EAM |
| 工装改制 | `toolMod` | EAM |

巡检/保养/点检/维修 PDA 均有工装分支。

## 2. 边界

与设备相同：**台账主数据在 DBC**；EAM 管验收、出入库记录、变更、改制。保养经验等服务同时调 `ToolAccountsApi`。

改制表 `relation_tool_mod`：改制编码、工装编码、类型、部门等。

## 3. vs MES

MES 工艺/工单可引用工装编码；寿命扣减与工单绑定细节待联调，不在未证实前写死。
