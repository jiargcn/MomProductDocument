# P0 字段证据索引

> 原文件名“P0 字段证据底表”保留为兼容入口。
> 基线：测试环境 / `dev` 分支 / 2026-07-15。

P0 字段证据已按业务对象拆分，避免把物料、采购收货和库存对象混在一份大型底稿中。正式产品页面不直接链接这些技术证据；它们只用于内部字段校正、规则核验和差距追溯。

| 业务对象 | 证据页 | 覆盖范围 |
| --- | --- | --- |
| 物料基本信息 | [DBC / 物料基本信息](字段证据/DBC/物料基本信息.md) | 字段映射、页面/导入规则、回收件语义、待核验项。 |
| 采购收货 | [WMS / 采购收货](字段证据/WMS/采购收货.md) | 申请、任务、记录主明细、来源单据、导入、终端和动作证据。 |
| 库存对象与库存粒度 | [WMS / 库存对象与库存粒度](字段证据/WMS/库存对象与库存粒度.md) | 预计入、库存事务、库存余额、唯一粒度和库存页面证据。 |

## 使用规则

1. 先阅读正式业务页面，只有需要核验字段、导入、接口或实现差异时才进入本目录。
2. 证据页按“业务含义 → 技术字段/来源 → 待核验项”组织，不按单一技术层拆分。
3. 跨多个业务复用的库存粒度、预计入和库存事务统一在库存对象页维护，避免在采购收货、发料、出库页面重复复制。
4. 发现新冲突时，写入对应业务对象的证据页和问题总账；不要重新向本索引堆放字段明细。
